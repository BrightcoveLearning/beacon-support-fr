introduction

Avec Brightcove Beacon, vous recevrez une application de périphérique générée qui peut être ajoutée au Panasonic Store.

  • Détails du magasin : https://developer.vieraconnect.com/
  • Délai suggéré pour l'approbation : Trois semaines

Informations requises pour Panasonic :

  • Application
    • URL de l'application et URL de transfert de l'application
  • La description
    • Maximum 1024 caractères
    • 26 lignes maximum
    • Chaque ligne composée d'un maximum de 85 caractères (ou 44 caractères asiatiques)
    • Veuillez insérer des sauts de ligne le cas échéant
  • Icônes
    • Quatre images d'icônes sont nécessaires :
      • 160x112 (PNG)
      • Icône de style bannière 256x60 (PNG)
      • 360 x 252 (PNG)
      • Icône cercle rond diamètre 336x336 (PNG)
  • Capture d'écran
    • Il en faut au moins un
    • 480 l x 270 h PNG
    • Ils sont affichés sur la page de détail de l'application dans le Marché

Commencer

Pour accéder aux pages des développeurs Panasonic, procédez comme suit :

  1. Dans un navigateur, accédez au Connexion développeur Panasonic page.

  2. Connectez-vous avec votre compte ou créez un nouveau compte.

  3. Lorsque vous vous connectez, vous devriez voir le panneau de configuration.

  4. Lisez la note qui se trouve à droite du Soumettre une nouvelle candidature bouton. Sélectionnez le page d'accueil relier.

  5. Passez en revue les exigences de l'application. Ne vous inquiétez pas, Brightcove créera l'application avec la technologie nécessaire.

Informations sur l'application

Les Informations sur l'application La section est l'endroit où vous entrerez les informations de base sur l'application.

  1. Depuis le panneau de configuration, sélectionnez le Soumettre une nouvelle candidature bouton.

  2. Vous devriez voir un formulaire de saisie comme celui-ci :

  3. Dans le Informations sur l'application page, saisissez des valeurs pour les champs requis :

    • Type de projet - Dans le menu déroulant, sélectionnez la dernière version HTML disponible.
    • URL de candidature - Créez deux référentiels pour héberger vos builds d'applications depuis Brightcove : Un pour la production et un pour la mise en scène. L'URL de l'application est l'URL de production.
    • URL intermédiaire - Saisissez l'URL de préparation de votre build d'application.
    • Numéro de version - Entrez la version de votre application.

  4. Sélectionnez le Ajouter enlever bouton. Choisissez les langues que vous voulez disponibles pour votre application. Sélectionnez le Confirmer bouton.

  5. Saisissez des valeurs pour les champs avec la langue sélectionnée. Dans cet exemple, Anglais (Amérique du Nord) est sélectionné.

    • Langage par défaut - Dans le menu déroulant, choisissez la langue initiale de votre application. Si votre application prend en charge l'anglais, Panasonic vous recommande de choisir l'anglais ((Amérique du Nord) ou l'anglais (modèle britannique)).
    • Nom de l'application - Saisissez le nom que vous souhaitez utiliser pour votre application.
    • Nom court - Saisissez le nom abrégé que vous souhaitez utiliser pour votre application. Cela peut être le même que le nom de l'application.
    • Mots clés - Saisissez les mots-clés à utiliser par la fonction de recherche du marché pour trouver votre application. Séparez les mots-clés par une virgule (ex : tennis de table, sports). 128 caractères maximum.
    • La description - Saisissez une description pour votre application. Cela sera affiché sur la page de détail de l'application dans le marché. Cliquez à l'intérieur de la zone de saisie pour plus de détails.
    • Contrat de licence de l'utilisateur final - Optionnel. Saisissez le contrat de licence utilisateur final.

  6. Sélectionnez la langue suivante dans votre liste de langues choisies et saisissez les valeurs des champs décrits à l'étape précédente.

  7. Sélectionnez le Prochain bouton.

Détails supplémentaires

Les Détails supplémentaires La section vous permet d'ajouter des détails sur le support et le contenu.

  1. Après avoir sélectionné le Prochain bouton de la section précédente, vous devriez voir un formulaire de saisie comme celui-ci :

  2. Dans le Détails supplémentaires page, saisissez des valeurs pour les champs requis :

    • Courriel d'assistance - Fournissez l'adresse e-mail que les utilisateurs doivent contacter lorsqu'ils rencontrent des problèmes avec votre application.
    • Évaluation parentale - Développez le menu déroulant pour voir les définitions de chaque évaluation parentale. Sélectionnez la note appropriée pour votre application.
    • Évaluation du contenu au Brésil - Ceci est requis si vous prévoyez de sortir au Brésil. Développez le menu déroulant pour voir les définitions de chaque note du Brésil. Sélectionnez la note appropriée.
    • Email du contact - Entrez votre adresse e-mail de contact.
    • Catégorie - Développez le menu déroulant pour voir la liste des catégories. Sélectionnez la catégorie appropriée.
    • Prix - Normalement, les applications sur Panasonic sont téléchargeables gratuitement.
    • Achat dans l'application - Sélectionnez si votre application aura des achats intégrés.
    • Place du marché - Clique sur le Sélectionnez la place du marché bouton pour voir une liste de pays. Soit choisir Tous les pays ou sélectionnez les pays dans lesquels vous souhaitez que votre application soit disponible. Lorsque vous avez terminé, sélectionnez le Terminé bouton.
    • Instructions pour l'assurance qualité - Incluez toutes les instructions qui peuvent être utiles, y compris que vous avez ajouté un bouton, modifié un lien, etc. Si nécessaire, incluez des comptes de test pour votre application.

  3. Sélectionnez le Prochain bouton.

Télécharger des éléments

Les Télécharger des éléments La section vous permet d'ajouter des icônes et des images.

  1. Après avoir sélectionné le Prochain bouton de la section précédente, vous devriez voir un formulaire de saisie comme celui-ci :

  2. Dans le Télécharger des éléments page, saisissez des valeurs pour les champs requis :

    • Description de l'application Doc - Créez un document qui détaille le fonctionnement de votre application. Il peut s'agir du même document que vous avez créé pour LG. Sélectionnez le Parcourir bouton pour localiser votre fichier. Ensuite, sélectionnez le Télécharger bouton.

      Notez que seulement doc , docx ou pdf les fichiers sont autorisés pour le téléchargement. La taille maximale du fichier est de 8 Mo.

    • Icône 1 - Téléchargez une icône de marché, qui sera affichée sur le page de liste et page de détails de la demande sur le marché.

      160 l x 112 h PNG

    • Icône 2 (forme circulaire) - Téléchargez une icône Firefox OS, qui sera affichée sur le Page de documentation des applications sur le marché.

      Forme de cercle de 336 pixels de diamètre PNG

    • Exemple d'image 1 - Ce sont des captures d'écran de votre application, qui seront affichées sur le page de détails de la demande sur le marché. Ajoutez jusqu'à 5 exemples d'images.

      480 l x 270 h PNG

  3. Sélectionnez le Prochain bouton.

Accepter les CGU

Les Accepter les CGU La section vous permet d'examiner toutes les informations que vous avez ajoutées pour votre application avant de la soumettre au magasin. Il y a aussi une section avec les termes et conditions de Panasonic.

  1. Après avoir sélectionné le Prochain bouton de la section précédente, vous devriez voir un formulaire de saisie comme celui-ci :

  2. Passez en revue les informations et les images que vous avez ajoutées pour votre application.

  3. Lisez les conditions générales de Panasonic TV. Cochez la case indiquant que vous avez lu et accepté.

  4. Sélectionnez le Soumettre bouton.

  5. Sur le Listes d'applications page, vous devriez voir votre application enregistrée en tant que brouillon.

  6. Votre application passera au statut QA avant d'être mise en ligne.
