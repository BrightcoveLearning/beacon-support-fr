Information requise

Pour pouvoir installer l'application Brightcove Beacon sur votre appareil Apple, vous aurez besoin de :

  • Un lien public ou un e-mail d'invitation fourni par Brightcove.
  • Application TestFlight installée sur votre appareil Apple.

Plateformes prises en charge par Apple :

  • Applications iOS : iPhone, iPad ou iPod touch sous iOS 8 ou version ultérieure.
  • Applications tvOS : Apple TV exécutant tvOS 12 ou version ultérieure.

Une vue de haut niveau du processus est :

  • Installez l'application TestFlight sur votre appareil.
  • Ouvrez le lien public ou l'e-mail d'invitation qui vous a été envoyé par Brightcove.
  • Utilisez le code d'installation de l'application Brightcove Beacon dans l'application TestFlight.
  • Exécutez l'application.

Installation de l'application Brightcove Beacon iOS

Dans cette section, vous apprendrez à télécharger l'application Brightcove Beacon iOS pour la tester sur votre iPhone, iPad ou iPod touch.

  1. Installer Vol d'essai sur l'appareil iOS que vous utiliserez pour les tests.
  2. Ouvrez votre e-mail d'invitation ou appuyez sur le lien public sur votre appareil iOS.
  3. Vous recevrez un code d'échange dans l'e-mail d'invitation. Copiez le code.
  4. Ouvrez le Vol d'essai app sur votre appareil Apple.
  5. Clique le Racheter bouton dans le coin supérieur droit.
  6. Collez le code d'échange Brightcove.
  7. Clique le Racheter bouton.

Installation de l'application Brightcove Beacon tvOS

Dans cette section, vous apprendrez comment télécharger l'application Brightcove Beacon tvOS pour la tester sur votre Apple TV.

  1. Installer Vol d'essai sur votre Apple TV.
  2. Ouvrez votre e-mail d'invitation sur un appareil mobile ou un ordinateur.
  3. Vous recevrez un code d'échange dans l'e-mail d'invitation. Assurez-vous d'avoir le code visible afin de pouvoir le saisir plus tard.
  4. Ouvrez le Vol d'essai app sur votre Apple TV.
  5. Cliquez dans le Racheter onglet et entrez le code de remboursement Brightcove.
  6. L'application Brightcove Beacon sera installée sur votre appareil pour être testée.
