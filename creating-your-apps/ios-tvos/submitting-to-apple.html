Apple TV et iOS

  • Détails du magasin : https://itunesconnect.apple.com/login (nécessite Apple Developer Program - Adhésion, 99 $/an)
  • Présentation du programme pour développeurs :
    • Introduction: https://developer.apple.com/programs/
    • Pour s'inscrire : https://developer.apple.com/programs/enroll/
    • Infos sur l'inscription : https://developer.apple.com/support/enrollment/
  • Délai suggéré pour l'approbation : Une semaine
Notez que cela peut prendre jusqu'à 24 heures après la publication pour que votre application soit disponible sur l'App Store. Voir: AppStore - Le statut de l'application est prêt pour la vente, mais pas dans l'App Store

Informations requises pour Apple TV :

  • Nom de l'application
  • Langue principale
  • Identifiant du bundle (par exemple : com. Nom de la compagnie . nom du produit )
  • SKU (par exemple : [appname] applettv)
  • Sous-titre - Un résumé de votre application qui apparaîtra sous le nom de votre application sur l'App Store pour les clients disposant d'appareils exécutant tvOS 11 ou version ultérieure - Champ facultatif
  • Politique de confidentialité d'Apple TV
  • Catégorie
    • Primaire et secondaire (facultatif)
    • https://developer.apple.com/app-store/categories/
  • Évaluation
  • Prix de l'application
  • Disponibilité du territoire (Pays où vous souhaitez publier l'application)
  • Voir le Spécifications des captures d'écran document du site d'Apple pour les exigences relatives aux aperçus d'applications et aux captures d'écran
  • Texte promotionnel (jusqu'à 170 caractères)
  • Mots clés
  • Description (jusqu'à 4000 caractères)
  • URL d'assistance
  • URL marketing (facultatif)
  • Version
  • Évaluation
  • Droits d'auteur: Le nom de la personne ou de l'entité qui détient les droits exclusifs sur votre application, précédé de l'année d'obtention des droits (par exemple, 2008 Acmé inc. ). Ne fournissez pas d'URL.
  • Coordonnées du représentant
    • Prénom et nom
    • Adresse
    • Ville
    • État
    • Code postal
    • Pays
    • Numéro de téléphone
    • E-mail
  • Informations sur l'examen de l'application
    • Informations de connexion (nom d'utilisateur et mot de passe)
    • Coordonnées (nom et prénom, numéro de téléphone et e-mail)
    • Remarques
  • Sortie de version

Périodiquement, vous devrez accepter les accords fiscaux et généraux en tant que propriétaire du compte. Ceux-ci peuvent être trouvés à appstoreconnect.apple.com/agreements/#/. De plus, toutes les conventions fiscales doivent être soumises si elles peuvent être soumises sur le Impôt onglet sur cette même page.
