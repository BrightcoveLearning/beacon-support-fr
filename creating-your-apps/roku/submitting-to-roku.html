introduction

Avec Brightcove Beacon, vous recevrez une application de périphérique générée qui peut être ajoutée au Roku Channel Store.

  • Détails du magasin : https://developer.roku.com/overview
  • Configuration du compte développeur : https://my.roku.com/signup
  • Implémentation de la recherche Roku : https://developer.roku.com/en-gb/docs/developer-program/discovery/search/implementing-search.md
  • Délai suggéré pour l'approbation : Trois semaines
Notez que la recherche de votre chaîne peut prendre jusqu'à 48 heures après la publication sur tous les appareils. Depuis le site Roku : Ma chaîne vient d'être publiée, pourquoi ne puis-je pas la trouver avec la recherche ?

Informations requises pour Roku :

  • Exigences relatives aux affiches et aux captures d'écran
    • Affiche de la chaîne (540x405)
    • Captures d'écran jusqu'à 1920x1080
  • E-mail du développeur
  • URL préférée pour plus d'informations
  • Contact administratif (Nom / Téléphone / Email)
  • Contact technique (Nom / Téléphone / Email)
  • Client pour configurer les connexions d'application pour l'assurance qualité en magasin

Commencer

Pour accéder aux pages des développeurs Roku, procédez comme suit :

  1. Dans un navigateur, accédez au Se connecter page.

  2. Connectez-vous avec votre compte ou créez un nouveau compte.

Ajout d'une chaîne

Découvrez comment ajouter une chaîne.

  1. Dans la navigation supérieure, cliquez sur l'icône de profil.

  2. Dans le menu déroulant, cliquez sur le Développeurs option.

  3. Dans la navigation supérieure, sélectionnez le Tableau de bord relier.

  4. Depuis le tableau de bord, vous pouvez gérer vos canaux. Sélectionnez le Ajouter une chaîne bouton.

  5. Quand le SDK développeur est choisie, vous verrez des options pour une chaîne publique ou non certifiée.

  6. Sélectionner Publique quand vous voulez que la chaîne aille au magasin. Sélectionner Non certifié lorsque vous souhaitez créer un canal de test.

    Le nom du canal doit être unique dans les deux Publique et Non certifié canaux.

    Nous recommandons ce qui suit :

    • Crée ton Publique nom de la chaîne, par exemple Destinations.

    • Créer un Non certifié canal de test avec un suffixe de AQ. Donc, dans ce cas, il serait nommé Contrôle qualité des destinations.

    Premièrement, nous créons la chaîne publique.

  7. Ensuite, nous créons le canal de test.

Propriétés du canal

Des propriétés sont associées à chaque canal.

  1. Retournez à votre tableau de bord du développeur et sélectionnez le Gérer mes chaînes relier.

  2. Pour votre chaîne, sélectionnez le Aperçu et mise à jour bouton.

  3. Élargir la Prévisualiser et publier menu déroulant et sélectionnez Propriétés.

  4. Vous devriez voir une liste des propriétés du canal. Sur le côté gauche, vous verrez une liste des pays Channel Store où votre chaîne apparaîtra. Pour les tests, vous pouvez sélectionner tous les pays. Pour votre chaîne en direct, choisissez les pays où vous souhaitez que votre chaîne soit vue.

  5. Pour les fonctionnalités d'accessibilité et la conformité, sélectionnez Oui. Toutes les applications Brightcove Beacon répondent à ces exigences.

  6. En fonction de votre contenu, choisissez la classification de contenu appropriée.

  7. Indiquez si un compte client est requis pour accéder aux fonctionnalités de votre chaîne.

  8. Saisissez l'URL de la politique de confidentialité associée à l'ensemble de votre application.

  9. Le code d'accès Vanity est généré automatiquement par Roku. Vous pouvez laisser cette valeur.

  10. Sélectionnez les langues que vous souhaitez prendre en charge.

  11. Pour Fonctionnalités requises , choisissez d'abord le Classification qui décrit le mieux votre chaîne.

  12. Ensuite, spécifiez si votre application nécessite une connexion Internet. Lorsque vous avez terminé de mettre à jour les propriétés de votre chaîne, sélectionnez le sauvegarder bouton.

Informations sur la chaîne de magasins

Ici, vous allez ajouter des descriptions et des icônes pour votre chaîne.

  1. Revenez à la page Aperçu et publication, développez le Prévisualiser et publier menu déroulant et sélectionnez Informations sur la chaîne de magasins.

  2. Pour Channel Store Info, saisissez les valeurs de champ suivantes :

    • Nom du canal - Cela devrait déjà être renseigné avec le nom de votre chaîne.
    • La description - Ajoutez une description détaillée de la chaîne, qui apparaîtra dans la description du Channel Store. (300 caractères)
    • Description Web - Saisissez une description de canal pour le marketing Web américain de votre canal. Cela ne s'affichera pas sur les appareils. (1500 caractères)
    • Affiche de la chaîne - Ajoutez une image d'affiche qui s'affichera dans la boutique Roku Channel. (540 x 405 JPEG ou PNG)
    • Catégorie descriptive - Sélectionnez la catégorie dans laquelle vous souhaitez diffuser votre chaîne. (Exemple: Sports, Drame, Comédie, etc.)
    • Région domestique - Sélectionnez le pays où votre chaîne est considérée comme native et/ou domestique. Votre chaîne apparaîtra dans la catégorie « International » de toutes les boutiques de chaînes régionales qui incluent votre chaîne, à l'exception de la boutique de chaînes associée à votre région nationale.

  3. Lorsque vous avez terminé d'ajouter Channel Store Info, sélectionnez le sauvegarder bouton.

Vérifiez votre compte et associez un appareil

Après avoir terminé la configuration du canal Roku, il est important de vérifier votre compte de développeur.

  1. Retour au Gérer les canaux l'écran et sélectionnez le Aperçu et mise à jour bouton.

  2. Développer Prévisualiser et publier et sélectionnez Téléchargement de paquet.

  3. Sélectionner Envoyer email de vérification bouton.

    Au début de la configuration de la chaîne, assurez-vous d'utiliser l'adresse e-mail de votre organisation qui sera utilisée pour les droits de développeur.

  4. Une fois que vous avez vérifié votre compte de messagerie de développeur, vous serez invité à associer votre appareil et à ajouter un mode de paiement. Si vous l'avez déjà fait, vous pouvez passer à la section suivante.

  5. Lorsque vous associez votre appareil Roku, assurez-vous que votre appareil a le même e-mail enregistré que lors des étapes de vérification. Pour de meilleurs résultats, nous vous recommandons de réinitialiser votre appareil Roku.

    Pour réinitialiser votre appareil Roku :

    1. Sélectionner Paramètres.
    2. Sélectionner Système.
    3. Sélectionner Réglages avancés du système.
    4. Sélectionner Retour aux paramètres d'usine.

    • Comment configurer le Roku Express/Express+

Ajouter une chaîne avec un code

Vous pouvez utiliser un code personnalisé pour ajouter votre chaîne d'applications à votre appareil Roku.

  1. Développez le menu déroulant de votre profil et sélectionnez Mon compte.

  2. Sélectionnez le Ajouter une chaîne avec un code relier.

  3. Dans le Ajouter une chaîne dialogue, procédez comme suit :

    1. Entrer le Code d'accès à la chaîne que nous vous fournissons.
    2. Vérifier la je ne suis pas un robot boîte.
    3. Sélectionner Ajouter une chaîne.

  4. Une fois la chaîne ajoutée à votre compte Roku, suivez ces étapes pour examiner votre application sur votre téléviseur Roku :

    1. Sélectionner Paramètres.
    2. Sélectionner Système.
    3. Sélectionner Mise à jour du système.

    Votre application devrait maintenant être visible pour votre examen.

Monétisation

Ici, vous allez ajouter des informations de monétisation sur votre chaîne.

  1. Revenez à la page Aperçu et publication, développez le Prévisualiser et publier menu déroulant et sélectionnez Monétisation.

  2. Indiquez si votre chaîne contiendra des publicités ou prendra en charge un modèle d'abonnement.

    Notez que même si vous utilisez Brightcove SSAI, vous devrez vous inscrire au programme Roku Programme de paiement des partenaires. C'est une exigence du magasin Roku d'effectuer le rappel des annonces pendant la lecture des annonces.

    Pour les chaînes d'abonnement, vous inclurez les informations suivantes :

    • type d'achat - Comment les utilisateurs sont facturés pour la chaîne. (Une fois, mensuellement, annuellement, etc.)
    • Niveau de prix - Sélectionnez une valeur dans le menu déroulant. Le prix d'achat pour les pays de l'EU Channel Store inclut la TVA. Le produit est basé sur des prix avant taxes (nets).

  3. Lorsque vous avez terminé d'ajouter des informations de monétisation, sélectionnez le sauvegarder bouton.

Captures d'écran

Ici, vous pouvez ajouter des captures d'écran à votre chaîne.

  1. Revenez à la page Aperçu et publication, développez le Prévisualiser et publier menu déroulant et sélectionnez Captures d'écran.

  2. Ajoutez des captures d'écran qui apparaîtront dans le Channel Store.

    Vous pouvez ajouter jusqu'à 6 captures d'écran. Chacun doit être des fichiers JPEG ou PNG 1920 x 1080.

  3. Lorsque vous avez terminé d'ajouter des captures d'écran, sélectionnez le sauvegarder bouton.

Informations d'assistance

Fournissez des informations de contact pour aider à l'assistance de votre chaîne.

  1. Revenez à la page Aperçu et publication, développez le Prévisualiser et publier menu déroulant et sélectionnez Informations d'assistance.

  2. Pour les informations d'assistance, saisissez les valeurs de champ suivantes :

    • E-mail du développeur - Saisissez votre adresse e-mail de développeur. Si ce n'est pas le cas, mettez à jour vos coordonnées.
    • URL préférée - Spécifiez l'URL de votre chaîne. Cela peut être utilisé par le site Web ou la newsletter Roku pour créer un lien vers votre chaîne.
    • Contacter l'assistance à l'utilisateur final - Fournissez une page Web, une adresse e-mail ou un numéro de téléphone ou plusieurs de ceux-ci. Roku l'utilisera lorsque le service client sera contacté à propos de votre chaîne. (300 caractères) Ceci est obligatoire.

    • Comptes de test et paramètres de liens profonds - Dites à Brightcove quels actifs vous souhaitez ouvrir en tant que liens profonds, et nous mettrons à jour les paramètres de liens profonds. Si vous ne souhaitez pas configurer de partage de compte, Brightcove vous fournira les paramètres.

      Vous fournirez également des informations d'identification pour 3 comptes de test. Ces comptes seront utilisés pour le support client, les démonstrations et l'assurance qualité. (1500 caractères) Si vous avez des comptes avec différents abonnements, vous devez les lister ici afin que le Channel Store ait accès à tous.

    • Contact administratif - Fournissez les coordonnées administratives de votre application.
    • Contact technique - Fournissez les coordonnées techniques de votre application.

  3. Lorsque vous avez terminé d'ajouter des informations d'assistance, sélectionnez le sauvegarder bouton.

Téléchargement de paquet

Téléchargez un package sur votre chaîne.

  1. Revenez à la page Aperçu et publication, développez le Prévisualiser et publier menu déroulant et sélectionnez Téléchargement de paquet.

  2. Tout d'abord, vous devez vérifier votre adresse e-mail. Ensuite, vous pourrez télécharger.

  3. Pour le téléchargement de package, saisissez les valeurs de champ suivantes :

    • Version de la chaîne - Spécifiez la version de votre application, en commençant par la version 1.0.
    • Micrologiciel minimal - Sélectionnez la version du micrologiciel Roku. Le package ne fonctionnera que sur les appareils prenant en charge la version minimale du micrologiciel ou supérieure. La dernière version de build stable que nous recommandons est Roku v9.2 bl.
    • Fichier de package - Le fichier de package peut être téléchargé par vous ou par Brightcove. Ce fichier aura un nom similaire à : Balise_1.0.6138.pkg.

  4. Lorsque vous avez terminé d'ajouter les informations de téléchargement de package, sélectionnez le sauvegarder bouton.

Analyse statique

Créez un rapport d'analyse sur votre chaîne.

  1. Revenez à la page Aperçu et publication, développez le Prévisualiser et publier menu déroulant et sélectionnez Analyse statique.

  2. Lorsque vous téléchargez un package dans la section précédente, votre package est analysé pour détecter les erreurs de structure ou de syntaxe. Un tableau peut être affiché avec des erreurs et des avertissements.

Prévisualiser et publier

Lorsque vous aurez terminé de remplir tous les formulaires liés à votre chaîne, vous verrez le Prévisualiser et publier filtrer.

  1. Revenez à la page Aperçu et publication, développez le Prévisualiser et publier menu déroulant et sélectionnez Prévisualiser et publier.

  2. Cette page vous donne un aperçu récapitulatif de votre chaîne.

  3. Lorsque vous êtes prêt, sélectionnez le Soumettre bouton pour publier votre chaîne.

  4. Votre application sera vérifiée par le magasin Roku pour voir si elle a des problèmes. Si votre chaîne est rejetée pour quelque raison que ce soit, un e-mail sera envoyé au E-mail du développeur que vous placez dans le Informations d'assistance filtrer. Passez en revue les raisons du rejet et créez un ticket d'assistance Brightcove pour résoudre les problèmes.

Mise à jour de votre chaîne

Si vous souhaitez mettre à jour une application existante, procédez comme suit :

  1. Créez un nouveau package Roku.

    Donnez au nouveau package un identifiant et un mot de passe uniques. Fournissez à Brightcove le nouveau package ainsi que les clés de ce package. Nous avons besoin des nouvelles informations pour soumettre le nouveau package au magasin.

    N'oubliez pas que si vous souhaitez modifier les noms de chaînes, ils doivent être uniques dans le Channel Store.
