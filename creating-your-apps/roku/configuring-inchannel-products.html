introduction

Les produits In-channel sont des solutions de monétisation que vous pouvez proposer dans vos canaux, comme un abonnement mensuel ou annuel. Ce document expliquera comment créer des produits intégrés pour monétiser vos canaux d'application Roku.

Création d'un produit dans le canal dans Roku

Découvrez comment ajouter un produit dans le canal sur la page Roku Developers en suivant ces étapes :

  1. Dans la navigation supérieure, cliquez sur l'icône de profil.

  2. Dans le menu déroulant, cliquez sur le Développeurs option.

  3. Dans la navigation supérieure, sélectionnez le Tableau de bord relier.

  4. Sélectionner Gérer mes produits In-Channel.

  5. Clique le Ajouter un nouveau produit bouton.
  6. Dans le Canaux champ, sélectionnez les canaux où le produit sera disponible à la vente.
  7. Remplissez le formulaire (capture d'écran de référence sous la liste des étapes) :
    1. Pour le catégorie de produit sélectionner Vidéo.
    2. Pour le Nom du produit , saisissez le nom qui sera visible pour les clients. Vous pouvez ajouter le nom du produit dans différentes langues en cliquant sur le Ajouter le nom du produit dans une autre langue bouton.
    3. Pour le Étiquette d'un produit , saisissez votre identifiant de référence interne.
      Notez que l'ID de référence interne doit être unique, et vous en aurez besoin plus tard pour ajouter votre produit In-Channel à la configuration Brightcove Beacon CMS.

    4. Pour le Type d'achat , sélectionnez le type d'abonnement en fonction de la façon dont vos clients sont facturés pour votre produit en canal. Dans cet exemple un Souscription mensuelle.

      Notez que l'intervalle d'abonnement pour Roku est différent des autres magasins. Pour plus de détails, consultez le Prise en charge des intervalles d'abonnement section du document Utilisation d'un forfait d'abonnement
    5. Pour le Niveau de prix option, sélectionnez un niveau de prix basé sur le prix de votre produit dans le canal.
  8. Dans le Essais et offres section, en fonction de votre stratégie produit dans le canal, vous pouvez sélectionner Essai gratuit et tapez une durée de nombre dans le Durée de l'essai champ et sélectionnez Jours ou Mois à partir du moment du dépôt.
  9. Allez au bas de la page, vérifiez le Autorisé à la vente case et cliquez sur le sauvegarder bouton.
  10. Après la création, vérifiez que votre produit est visible sur le Gérer mes produits en chaîne page.
    Vous pouvez créer autant de produits supplémentaires que vous le souhaitez en fonction du type d'abonnement que vous souhaitez proposer. Exemple : annuel ou mensuel.

Associez votre produit intégré à un canal dans Brightcove Beacon

Une fois vos produits intégrés au canal créés, accédez à la plate-forme Brightcove Beacon pour associer vos nouveaux produits intégrés à un canal.

  1. Dans le menu du haut, cliquez sur le Commerce languette.
  2. Dans le menu de gauche, sélectionnez le Paquets option.
  3. Cliquez sur votre colis.
  4. Du Emballer onglet, descendez jusqu'au Détails des prix section.
  5. Dans le Prix champ, saisissez le prix de votre produit dans le canal.
  6. Dans le Monnaie champ, sélectionnez le type de devise.
  7. Dans le Identifiant du magasin Roku , saisissez l'ID d'identification de votre produit dans le canal créé dans la plate-forme de développeurs Roku.
  8. Allez au bas de la page et cliquez sur le Modifier le package bouton pour enregistrer vos modifications.
