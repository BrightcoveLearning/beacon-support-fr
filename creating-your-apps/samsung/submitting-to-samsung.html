    Soumettre des applications dans le Samsung Store

    Dans cette rubrique, vous apprendrez comment soumettre votre application de votre appareil à la boutique Samsung.

    Introduction

    Avec Brightcove Beacon, vous recevrez une application générée qui peut être ajoutée à l'App Store Samsung.

    • Détails de la boutique : http://seller.samsungapps.com/tv/app/registration/list
    • Présentation du développeur : https://developer.samsung.com/galaxy-store
    • Configuration du compte développeur : https://program.developer.samsung.com/sign-in/
    • Informations sur l'enregistrement de la demande : https://developer.samsung.com/smarttv/develop/distribute/seller-office/applications/application-registration.html
    • Délai suggéré pour l'approbation : Trois semaines
    Si vous prévoyez de lancer votre application en dehors des États-Unis ou si vous souhaitez accéder à certaines API, vous aurez besoin d'un partenariat Samsung Apps TV. Pour plus d'informations, consultez le document Devenir un partenaire Samsung .

    Informations requises pour Samsung :

    • Applications
      • Tizen — Fichier Widget
    • Description
    • Icônes pour 16 Model Group
    • Logo Actif
      • Avec Transparence
      • 1920x1080 pixels 32 bits
      • PNG (RGBA) avec 300kb
    • Image d'arrière-plan
      • 1920x1080 pixels 24 bits
      • PNG (RGBA) avec 300kb
    • Icône pour groupe de modèles hérités
    • Icône
      • 512x423 pixel
      • 300ko
    • Images de capture d'écran (x4)
    • 1280x720 ou 1920x1080 pixels, 500 ko
    • Description
    • Document de soumission PowerPoint : doit être téléchargé sur le portail lors de la soumission

    Premiers pas

    Pour accéder aux pages développeur Samsung, procédez comme suit :

    1. Dans un navigateur, accédez à la page Connexion Samsung .

    2. Connectez-vous avec votre compte ou créez un nouveau compte.

    3. Lorsque vous vous inscrivez, vous devez demander un compte Partenaire auprès de Samsung pour pouvoir soumettre vos applications dans le monde entier. Sinon, vous serez limité à un certain emplacement.

    Enregistrement de votre application

    Enregistrez votre application auprès de Samsung.

    1. Sur la page d'accueil, cliquez sur le bouton Enregistrement des applications Web .

    2. Vous verrez un ensemble d'étapes à suivre avant de pouvoir soumettre votre application.

    3. Tout d'abord, entrez un nom d'application. Il s'agit de votre identifiant d'application et n'est utilisé qu'à des fins de gestion. Le titre de l'application affiché sera saisi ultérieurement.

    Informations de base

    Saisissez des informations de base sur votre application TV. Il s'agit d'informations qui sont affichées à l'utilisateur.

    1. Pour obtenir des informations sur l'application TV de base, entrez les valeurs de champ suivantes :

      • Nom du vendeur - Saisissez le nom du vendeur. Ça pourrait être le nom de votre entreprise.
      • E-mail de support - Adresse e-mail à laquelle vous souhaitez que les utilisateurs envoient des questions d'assistance technique
      • Page d'accueil du vendeur - Saisissez l'URL de la page d'accueil de votre entreprise.
      • Représentant - Nom de votre représentant
      • Numéro de téléphone - Numéro de téléphone de contact de l'entreprise
      • Adresse - Adresse de votre entreprise
      • App Images - Brightcove ajoutera normalement les images d'icône pour vous. Les images de capture d'écran proviennent de l'application elle-même. Vous pouvez vouloir 4 captures d'écran :
        • 1 pour la page d'accueil
        • 1 pour la page détaillée
        • 1 pour joueur
        • 1 pour les réglages

        Ensuite, vous pouvez choisir les captures d'écran que vous souhaitez utiliser dans votre application.

    2. Choisissez une ou plusieurs langues pour votre application.

      • Langue : il s'agit de la langue utilisée pour le titre, la description et les informations de balise de votre application. Vous pouvez sélectionner plusieurs langues.
      • Informations offertes dans TV - Ici, vous verrez les langues que vous avez sélectionnées dans le champ Langue. La première langue est définie par défaut. Entrez ces valeurs de champ pour chaque langue.
        • Titre de l'application - Titre de votre application. Utilisez le même titre d'application pour chaque langue. Ils devraient correspondre entre les langues. (exemple « BC App »)
        • Description - Entrez une description spécifique à chaque langue.
        • Tag - Saisissez des mots-clés pour rechercher l'application (exemple : « BC, BC App, App »

    3. Continuez avec le reste des champs de la page :

      • Catégorie - Indiquez la catégorie pour votre application (vidéo, jeu, sport, etc.)
      • Évaluation - Sélectionnez le groupe d'âge approprié pour votre demande. Toute demande de 18 ans et plus ne peut être signifiée en Corée, en Arabie Saoudite ou aux Émirats arabes unis.
      • Pays ou région - Vous devez être un partenaire pour assurer le service de votre application dans des pays en dehors des États-Unis. Si vous ne l'avez pas déjà fait, sélectionnez le lien pour demander un partenariat. Sélectionnez les pays dans lesquels vous souhaitez que votre application apparaisse.
      • Certification de notation - Pour servir le Brésil, vous devez télécharger les certificats appropriés.
      • Politique de confidentialité - Indiquez si votre application recueille des informations personnelles auprès des utilisateurs.

    4. Sélectionnez le bouton Suivant .

    Informations sur les essais

    Saisissez les informations de test sur votre application TV.

    1. Pour obtenir des informations sur le test, entrez les valeurs de champ suivantes :

      • Groupe de modèles - Sélectionnez les plates-formes Samsung Smart TV que vous souhaitez tester. Normalement, vous devriez voir 3 ans de retour aux modèles actuels.
      • Date de demande de lancement - Choisissez l'une des options suivantes :
        • Relâchez votre application dès qu'elle réussit les tests QA Samsung.
        • Définissez la date et l'heure de sortie de votre application. Il ne sera publié que s'il réussit les tests QA Samsung avant la date demandée.

    2. Continuez avec les notes de test :

      • Note pour Tester - Incluez des notes spécifiant les nouveautés de votre application. Les testeurs Samsung examineront ces notes.
      • Fonction d'application - Sélectionnez toutes les fonctionnalités qui s'appliquent à votre application, y compris les suivantes :
        • Indique si un clavier ou une souris doit être connecté au téléviseur
        • Application Smart View activée - Active les miniatures de prévisualisation pour la Smart TV. Brightcove recommande d'activer cette fonctionnalité.
      • Fichier de description de l'application - Créez un document qui répertorie toutes les pages, fonctionnalités et options de votre application. Cela donne aux testeurs Samsung un aperçu de ce qu'il faut tester. Téléchargez le modèle pour commencer. Brightcove peut fournir des conseils pour ce document.
        Le modèle peut être téléchargé à partir de la page Description de l'interface utilisateur de Samsung en cliquant sur le lien surligné :
      • Légende - Normalement, vous sélectionnez Aucune légende. Cette fonctionnalité n'est pas des sous-titres, mais des sous-titres utilisés principalement aux États-Unis. Même si votre application utilise des sous-titres, vous allez sélectionner Aucune légende ici.

    3. Continuer avec la spécification du joueur :

      Sélectionnez les valeurs appropriées pour les informations de votre joueur :

      • Codec vidéo
      • Conteneur
      • Moteur de streaming
      • Codec audio
      • DRM
      • API du lecteur
      • Autre - Entrez les informations du joueur

    4. Sélectionnez le bouton Suivant .

    Pré-test

    Le pré-test est l'endroit où vous pouvez tester votre application.

    1. Tizen Studio est l'endroit où nous créons les builds qui seront soumis à la boutique Samsung. Vous verrez un fichier zip avec l'application à l'intérieur. L'exécution du pré-test vous permet de confirmer que tout va bien avant de soumettre votre application au magasin.

    2. Testez votre application dans Tizen Studio. Le pré-test vous dira s'il y a des informations manquantes pour votre application.

    3. Dites à Brightcove si tout va bien avec votre application. Si des problèmes existent, consultez la section Informations sur le test .

    Aperçu

    Prévisualisez votre application avant de la soumettre au magasin.

    1. L'aperçu vous montrera tous les groupes de modèles Samsung que vous avez sélectionnés. La colonne d'état contient des valeurs telles que Révision , Test , puis Lancé. Lorsque le statut est Lancé, votre application se trouve dans le magasin de production.

    Envoyer

    Soumettez votre application à la boutique Samsung.

    1. Lorsque vous êtes prêt, vous pouvez soumettre votre application à la boutique Samsung.

    Aperçu du Smart Hub

    Grâce à la fonctionnalité Smart Hub Preview de Samsung, le contenu s'affiche lorsque le spectateur survole l'icône de votre application dans le lanceur Smart Hub. Cela fournit un bon moyen de promouvoir à la fois du contenu nouveau et personnalisé. Voici comment cela fonctionne :

    • Lorsque les utilisateurs passent le curseur sur l'icône de votre application, l'aperçu Smart Hub s'affiche. Les spectateurs peuvent ensuite sélectionner la vignette d'aperçu pour ouvrir le contenu dans votre application.

    • Il existe deux types d'aperçus :

      • Aperçu public - Cette option affiche le même contenu d'aperçu pour tous les utilisateurs.

      • Aperçu personnel - Cette option affiche le contenu d'aperçu personnalisé pour le spectateur.

      Un fichier JSON hébergé à distance est utilisé pour définir le contenu de l'aperçu.

    • Le contenu de l'aperçu peut être personnalisé pour correspondre au contenu de l'application :

      • Vidéo - Vidéos populaires, reprendre la lecture, mes playlists
      • Sports - Les meilleurs matchs de la semaine, mon équipe
      • Actualités - Les gros titres du jour, Météo locale

      • Jeux - Top jeux, récemment joué

    • Pour plus d'informations, consultez la documentation de Samsung Smart Hub Preview .

    Astuces utiles

    Les éléments suivants peuvent être utiles lors de la soumission de votre application à Samsung :

    • Si vous devez ajouter un nouvel utilisateur à votre compte, vous pouvez utiliser l'option Gestion des > groupes d'adhérents pour le faire.
    • Soyez très prudent de NE PAS placer des espaces devant les adresses e-mail car dans certains cas, ils ne seront PAS dépouillés et provoquent des échecs dans le processus de soumission de l'application.
