introduction

Avec Brightcove Beacon, vous recevrez une application d'appareil générée qui peut être ajoutée à l'App Store de Samsung.

  • Détails du magasin : http://seller.samsungapps.com/tv/app/registration/list
  • Présentation du développeur : https://developer.samsung.com/galaxy-store
  • Configuration du compte développeur : https://program.developer.samsung.com/sign-in/
  • Informations sur l'inscription de l'application : https://developer.samsung.com/smarttv/develop/distribute/seller-office/applications/application-registration.html
  • Délai suggéré pour l'approbation : Trois semaines
Si vous envisagez de lancer votre application en dehors des États-Unis ou si vous souhaitez accéder à certaines API, vous aurez besoin d'un partenariat Samsung Apps TV. Pour plus de détails, consultez le Devenir partenaire Samsung document.

Informations requises pour Samsung :

  • Applications
    • Tizen – Fichier widget
  • La description
  • Icônes pour 16 groupes de modèles
  • Actif du logo
    • Avec transparence
    • 1920x1080 pixels 32 bits
    • PNG (RVBA) avec 300ko
  • Image de fond
    • 1920x1080 pixels 24 bits
    • PNG (RVBA) avec 300ko
  • Icône pour le groupe de modèles hérités
  • Icône
    • 512x423 pixels
    • 300 Ko
  • Images de capture d'écran (x4)
  • 1280x720 ou 1920x1080 pixels, 500 Ko
  • La description
  • Document de soumission PowerPoint - Doit être téléchargé sur le portail lors de la soumission

Commencer

Pour accéder aux pages des développeurs Samsung, procédez comme suit :

  1. Dans un navigateur, accédez au Connexion Samsung page.

  2. Connectez-vous avec votre compte ou créez un nouveau compte.

  3. Lorsque vous vous inscrivez, vous devez demander un compte partenaire avec Samsung pour pouvoir soumettre vos applications dans le monde entier. Sinon, vous serez limité à un certain endroit.

Enregistrement de votre application

Enregistrez votre application auprès de Samsung.

  1. Depuis la page d'accueil, sélectionnez le Inscription à l'application Web bouton.

  2. Vous verrez un ensemble d'étapes à suivre avant de pouvoir soumettre votre application.

  3. Entrez d'abord un Nom de l'application. Il s'agit de votre identifiant d'application et est utilisé à des fins de gestion uniquement. Le titre de l'application affiché sera saisi ultérieurement.

Informations de base

Saisissez les informations de base sur votre application TV. Il s'agit d'informations qui sont présentées à l'utilisateur.

  1. Pour les informations de base sur l'application TV, saisissez les valeurs de champ suivantes :

    • Nom du Vendeur - Entrez le nom du vendeur. Cela pourrait être le nom de votre entreprise.
    • Courriel d'assistance - Adresse e-mail à laquelle vous souhaitez que les utilisateurs envoient des questions d'assistance
    • Page d'accueil du vendeur - Saisissez l'URL de la page d'accueil de votre entreprise.
    • Représentant - Le nom de votre représentant
    • Numéro de téléphone - Numéro de téléphone de contact de l'entreprise
    • Adresse - L'adresse de votre entreprise
    • Images de l'application - Brightcove ajoutera normalement les images d'icônes pour vous. Les images de capture d'écran proviennent de l'application elle-même. Vous voudrez peut-être 4 captures d'écran :
      • 1 pour la page d'accueil
      • 1 pour la page de détail
      • 1 pour le joueur
      • 1 pour les réglages

      Ensuite, vous pouvez choisir les captures d'écran que vous souhaitez utiliser dans votre application.

  2. Choisissez 1 ou plusieurs langues pour votre application.

    • Langue - Il s'agit de la langue du titre, de la description et des informations de balise de votre application. Vous pouvez sélectionner plusieurs langues.
    • Informations proposées à la télévision - Ici, vous verrez les langues que vous avez sélectionnées dans le champ Langue. La première langue est définie par défaut. Saisissez ces valeurs de champ pour chaque langue.
      • Titre de l'application - Le titre de votre application. Utilisez le même titre d'application pour chaque langue. Ils doivent correspondre entre les langues. (exemple « Application BC »)
      • Description - Saisissez une description spécifique à la langue pour chaque langue.
      • Balise : saisissez des mots-clés pour rechercher l'application (exemple : "C.-B., appli C.-B., appli"

  3. Continuez avec les autres champs de la page :

    • Catégorie - Spécifiez la catégorie de votre application (Vidéo, Jeu, Sports, etc.)
    • Évaluation - Sélectionnez la tranche d'âge appropriée pour votre candidature. Les demandes classées 18+ ne peuvent pas être traitées en Corée, en Arabie saoudite ou aux Émirats arabes unis.
    • Pays ou région - Vous devez être un partenaire pour entretenir votre application dans des pays en dehors des États-Unis. Si vous ne l'avez pas déjà fait, sélectionnez le lien pour demander un partenariat. Sélectionnez les pays dans lesquels vous souhaitez que votre application apparaisse.
    • Certification de notation - Pour desservir le Brésil, vous devez télécharger les certificats appropriés.
    • Politique de confidentialité - Précisez si votre application collecte des informations personnelles auprès des utilisateurs.

  4. Sélectionnez le Prochain bouton.

Informations sur les tests

Saisissez les informations de test sur votre application TV.

  1. Pour les informations de test, saisissez les valeurs de champ suivantes :

    • Groupe de modèles - Sélectionnez les plates-formes Samsung Smart TV avec lesquelles vous souhaitez tester. Normalement, vous devriez voir 3 ans en arrière pour les modèles actuels.
    • Date de demande de libération - Choisissez soit :
      • Lancez votre application dès qu'elle réussit les tests d'assurance qualité de Samsung.
      • Réglez la date et l'heure de sortie de votre application. Il ne sera publié que s'il réussit les tests d'assurance qualité de Samsung avant la date demandée.

  2. Continuez avec les notes de test :

    • Remarque pour le testeur - Incluez des notes spécifiant les nouveautés de votre application. Les testeurs de Samsung examineront ces notes.
    • Fonctionnalité de l'application - Sélectionnez toutes les fonctionnalités qui s'appliquent à votre application, notamment :
      • Si un clavier ou une souris doit être connecté au téléviseur
      • Application Smart View activée - Active les vignettes d'aperçu pour Smart TV. Brightcove recommande d'activer cette fonctionnalité.
    • Fichier de description de l'application - Créez un document qui répertorie toutes les pages, fonctionnalités et options de votre application. Cela donne aux testeurs de Samsung un aperçu de ce qu'il faut tester. Téléchargez le modèle pour commencer. Brightcove peut fournir des conseils pour ce document.
      Le modèle peut être téléchargé à partir de Samsung Description de l'interface utilisateur de l'application page en cliquant sur le lien en surbrillance :
    • Légende - Normalement, vous sélectionnerez Pas de légende. Cette fonctionnalité ne concerne pas les sous-titres, mais les sous-titres utilisés principalement aux États-Unis. Même si votre application utilise des sous-titres, vous sélectionnerez Pas de légende ici.

  3. Continuer avec les spécifications du lecteur :

    Sélectionnez les valeurs appropriées pour les informations de votre joueur :

    • Codec vidéo
    • Récipient
    • Moteur de streaming
    • Un codec audio
    • GDN
    • API de lecteur
    • Autre - Entrez d'autres informations sur le joueur

  4. Sélectionnez le Prochain bouton.

Pré-test

Le pré-test est l'endroit où vous pouvez tester votre application.

  1. Tizen Studio est l'endroit où nous créons les versions qui seront soumises au magasin Samsung. Vous verrez un fichier zip avec l'application à l'intérieur. L'exécution du pré-test vous permet de confirmer que tout va bien avant de soumettre votre application au magasin.

  2. Testez votre application dans Tizen Studio. Le pré-test vous dira s'il manque des informations pour votre application.

  3. Informez Brightcove si tout va bien avec votre application. S'il y a des problèmes, consultez le Informations sur les tests section.

Aperçu

Prévisualisez votre application avant de la soumettre au magasin.

  1. L'aperçu vous montrera tous les groupes de modèles Samsung que vous avez sélectionnés. Les statut colonne contiendra des valeurs telles que Révision , Test puis Lancé. Lorsque le statut est Lancé , votre application se trouve dans le magasin de production.

Soumettre

Soumettez votre application à la boutique Samsung.

  1. Lorsque vous êtes prêt, vous pouvez soumettre votre application à la boutique Samsung.

Aperçu du hub intelligent

Avec la fonction Aperçu Smart Hub de Samsung, le contenu s'affiche lorsque le spectateur survole l'icône de votre application dans le lanceur Smart Hub. Cela constitue un bon moyen de promouvoir à la fois du contenu nouveau et personnalisé. Voilà comment cela fonctionne:

  • Lorsque les téléspectateurs survolent l'icône de votre application, l'aperçu Smart Hub s'affiche. Les téléspectateurs peuvent ensuite sélectionner la vignette d'aperçu pour ouvrir le contenu dans votre application.

  • Il existe deux types d'aperçus :

    • Aperçu public - Cette option affiche le même contenu d'aperçu à tous les téléspectateurs.

    • Aperçu personnel - Cette option affiche le contenu d'aperçu personnalisé pour le spectateur.

    Un fichier JSON hébergé à distance est utilisé pour définir le contenu de l'aperçu.

  • Le contenu de l'aperçu peut être personnalisé pour correspondre au contenu de l'application :

    • Vidéo - Vidéos populaires, reprendre la lecture, mes playlists
    • Des sports - Les meilleurs matchs de la semaine, mon équipe
    • Nouvelles - Les gros titres du jour, la météo locale

    • Jeux - Meilleurs jeux, récemment joués

  • Pour plus de détails, voir Samsung Aperçu du hub intelligent Documentation.

Astuces utiles

Les éléments suivants peuvent être utiles lorsque vous soumettez votre application à Samsung :

  • Si vous devez ajouter un nouvel utilisateur à votre compte, vous pouvez utiliser le Adhésion > Gestion de groupe possibilité de le faire.
  • Faites très attention à ne PAS mettre d'espaces devant les adresses e-mail, car dans certains cas, elles ne seront PAS supprimées et entraîneront des échecs dans le processus de soumission de l'application.
