Aperçu

Vous aurez besoin d'un partenariat Samsung Apps TV et d'un groupe de niveau partenaire si vous prévoyez de lancer votre application en dehors des États-Unis ou si vous souhaitez accéder à certaines API.

Voici le processus de demande de partenariat :

  1. Si vous n'en avez pas déjà un, créez un groupe public. Ceci est requis pour la demande de partenariat.

  2. Demandez un accord hors ligne avec le gestionnaire de contenu Samsung. Dans certains cas, cette exigence peut être levée par le gestionnaire de contenu.

  3. Une fois que le gestionnaire de contenu Samsung aura approuvé votre demande, vous deviendrez un partenaire Samsung Apps TV.

Création d'un groupe public

Tout d'abord, si vous n'en avez pas déjà un, créez un groupe public.

  1. Dans un navigateur, accédez au Connexion Samsung page.

  2. Connectez-vous avec votre compte ou créez un nouveau compte.

  3. Si vous n'êtes pas déjà membre d'un groupe public, créez un nouveau groupe. Dans la navigation de gauche, sélectionnez Gestion de groupe.

  4. Sélectionnez le Allez dans "Créer un groupe" bouton.

  5. Saisissez des valeurs pour Informations sur le groupe et Information du vendeur. Lorsque vous avez terminé, sélectionnez le Créer un groupe bouton.

Demande de partenariat

Ensuite, demandez un partenariat.

  1. Depuis l'écran d'accueil du développeur, sélectionnez soit le Demander un partenariat bouton dans la navigation de gauche ou le Demande de partenariat lien de la navigation supérieure.

  2. Saisissez les informations sur le partenaire.

    La page se compose de trois sections de saisie :

    • Informations sur le Gestionnaire de contenu Samsung
    • Informations sur le groupe
    • Information du vendeur

    En fonction de votre statut de création de groupe, la page du formulaire de demande sera différente.

    • Si vous avez déjà créé un groupe, les valeurs des informations du groupe et des informations sur le vendeur seront définies à partir de votre groupe. Vous pouvez modifier les informations de votre groupe ici.
    • Si vous n'avez pas créé de groupe, tous les champs seront vides.

    Les champs d'informations sur le groupe et les informations sur le vendeur sont requis pour l'enregistrement.

    Étant donné que Samsung traite les demandes de partenariat en fonction du groupe, le bureau du vendeur créera votre groupe en tant que groupe public avant d'examiner votre demande.

  3. Si vous ne connaissez pas votre Gestionnaire de contenu Samsung , posez une question en utilisant 1:1 Q&R . Samsung vous fournira l'adresse e-mail de votre gestionnaire de contenu.

    • Dans la navigation supérieure, survolez Soutien.
    • Sélectionner 1:1 Q&R.
    • Sélectionnez le Poser une question bouton.
  4. Lorsque vous avez terminé de remplir la page d'informations sur le partenariat, sélectionnez le Demander un partenariat bouton.

Vérification du statut de partenariat

Enfin, nous vous montrerons comment vérifier votre statut de partenaire.

  1. Une fois que le gestionnaire de contenu Samsung aura approuvé votre demande, vous deviendrez un partenaire Samsung Apps TV.

  2. Vous pouvez vérifier que vous êtes un vendeur partenaire, en consultant le Bureau du vendeur page d'accueil.
