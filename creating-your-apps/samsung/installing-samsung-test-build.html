Conditions préalables

Pour pouvoir installer l'application Brightcove Beacon sur votre Samsung Smart TV, vous aurez besoin de :

  • Samsung Smart TV avec le système d'exploitation Tizen
  • Assurez-vous que le Samsung Smart TV dispose du dernier firmware installé

Installation de l'application Brightcove Beacon sur les appareils Samsung Smart TV

Une fois que vous avez reçu l'e-mail de Brightcove, veuillez suivre les étapes suivantes :

  1. Dans l'e-mail envoyé par Brightcove, une pièce jointe ZIP viendra. Téléchargez le fichier ZIP.
  2. Double-cliquez sur le fichier ZIP et extrayez le dossier du fichier. Les fichiers d'installation de Brightcove Beacon se trouvent dans ce dossier.
  3. Copiez le dossier et collez-le à la racine d'une clé USB.
  4. Insérez la clé USB dans votre Samsung Smart TV. L'application Brightcove Beacon commencera à s'installer automatiquement.
  5. Dans le Samsung Smart TV, accédez au applications menu et ouvrez l'application Brightcove Beacon.
Chaque fois que vous souhaitez installer une nouvelle version de l'application Brightcove Beacon, vous devrez supprimer la version précédemment installée, sinon vous trouverez une erreur système.

Désinstallation de l'application Brightcove Beacon

Pour désinstaller votre application de l'appareil Samsung Smart TV, procédez comme suit :

  1. appuie sur le Accueil bouton de votre manette OneRemote.
  2. À l'aide du pavé directionnel de votre télécommande, accédez à et sélectionnez APPLICATIONS.
  3. Sélectionner Options.
  4. Sélectionnez l'application Brightcove Beacon.
  5. Sélectionner Effacer.
  6. Sélectionner Effacer de nouveau.
