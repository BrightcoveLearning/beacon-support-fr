introduction

Ce document vous montre comment installer une version de test afin que vous n'ayez pas à suivre le processus de soumission d'application avant de tester votre application.

Pour pouvoir installer l'application Brightcove Beacon sur votre appareil Android, vous aurez besoin de :
  • mode développeur activé sur votre appareil Android.
  • Android Debug Bridge (adb) installé sur votre ordinateur.

Une vue de haut niveau du processus est :

  1. Sur votre appareil mobile Android, ouvrez l'invitation à l'application dans votre e-mail.
  2. Acceptez l'invitation et téléchargez Firebase App Tester.
  3. Installez l'application Brightcove Beacon.
  4. Exécutez votre application Brightcove Beacon sur votre appareil mobile Android.

Installation de l'application mobile Android

Pour installer l'application mobile Android, veuillez suivre ces étapes :

  1. Sur un appareil Android, ouvrez l'invitation dans votre e-mail.
  2. Lorsque vous y êtes invité, connectez-vous avec Google et acceptez l'invitation à tester l'application.
  3. Une fois que vous avez accepté une invitation, vous avez accès à l'installation de Firebase App Tester.
  4. Téléchargez Firebase App Tester.
  5. Dans Firebase App Tester, sélectionnez l'application Brightcove Beacon et appuyez sur Télécharger.
  6. L'application Brightcove Beacon sera automatiquement ajoutée à l'écran d'accueil de votre appareil.
  7. Lancez l'application Brightcove Beacon.

Installation de l'application Android TV

Vous devez utiliser Android Debug Bridge (adb) pour connecter votre ordinateur à un appareil Android TV pour installer, tester et déboguer vos applications. Pour plus d'informations sur la façon d'activer adb, veuillez suivre le Android Documentation.

Pour installer l'application Android TV, procédez comme suit :

  1. Permettre Options de développeur dans votre appareil Android TV.
  2. Sélectionner Appareil (ou Android TV) > Options pour les développeurs.
  3. Allumer Débogage ADB.
  4. Allumer Applications de sources inconnues.
  5. Connectez votre ordinateur à votre appareil via Wi-Fi en utilisant Pont de débogage Android (adb).
  6. Ouvrez votre terminal ou votre ligne de commande.
  7. Tapez la commande suivante, où path-to-apk-file est le chemin du système de fichiers vers l'APK de votre application. 
    adb install path-to-apk-file
  8. Si votre installation a réussi, adb répondra avec un message similaire à celui-ci : 
    Paquet de 764 Ko/s (217246 octets en 0,277s) : /data/local/tmp/HelloWorld.apk Succès
  9. Pour réinstaller une application qui existe déjà sur l'appareil, vous pouvez utiliser le -r possibilité de le réinstaller. 
    adb install -r chemin d'accès au fichier apk
  10. L'application Android sera automatiquement ajoutée à l'écran d'accueil de votre appareil.
  11. Lancez l'application Brightcove Beacon.

Installation de l'application Android TV sur invitation

Pour pouvoir installer l'application Android TV sur votre appareil, vous aurez besoin d'une invitation par e-mail de Brightcove.

Pour ce processus, vous devrez utiliser un compte Google pour vous connecter au Jouer au magasin site Web et votre appareil Android TV, tous deux avec le même compte Google.

Une fois que vous avez reçu l'e-mail d'invitation, veuillez suivre les étapes suivantes :

  1. Brightcove vous enverra un e-mail avec une URL Play Store renvoyant vers l'application de test Brightcove Beacon.
  2. Depuis votre compte de messagerie, cliquez sur l'URL ou copiez-la et collez-la dans votre navigateur.
  3. Vous serez invité à la page de l'application. Clique sur le téléchargez-le sur Google Play relier.
  4. Dans le Play Store, cliquez sur le Installer bouton.
  5. Lorsque la fenêtre contextuelle apparaît, cliquez sur le Choisissez un appareil option.
  6. Une liste avec tous vos appareils Android connectés à votre compte Google apparaîtra. Sélectionnez l'appareil Android TV sur lequel vous souhaitez installer l'application Brightcove Beacon.
  7. Après avoir sélectionné votre appareil, cliquez sur le Installer bouton.
  8. Le compte Google vous demandera votre mot de passe. Tapez-le et cliquez sur le Suivant bouton. Un message apparaîtra vous indiquant que l'application Brightcove Beacon sera bientôt installée sur votre appareil Android TV.
  9. Accédez à votre appareil Android TV.
    10. Google peut prendre jusqu'à 30 minutes pour synchroniser l'application Brightcove Beacon et l'installer sur votre appareil Android TV.
  10. L'application Brightcove Beacon sera automatiquement ajoutée à l'écran d'accueil de votre appareil Android TV.
  11. Lancez l'application Brightcove Beacon.

Dépannage de l'application Android TV

Si votre application Brightcove Beacon n'apparaît pas sur votre appareil Android TV après 30 minutes d'installation sur le site Web Play Store, procédez comme suit :

  1. Redémarrez votre appareil Android TV.
  2. Après avoir redémarré votre appareil, accédez à Paramètres > Compte et connexion > Sélectionnez votre profil Google > Cliquez sur Synchroniser maintenant.
  3. Après ces étapes, l'application Brightcove Beacon doit être automatiquement ajoutée à l'écran d'accueil de votre appareil Android TV.
  4. Lancez l'application Brightcove Beacon.

Désinstallation de l'application Brightcove Beacon

Pour désinstaller votre application de l'appareil Android TV, procédez comme suit :

  1. Depuis l'écran d'accueil de votre appareil, sélectionnez Paramètres > Applications et notifications > Voir toutes les applications.
  2. Sélectionnez l'application Brightcove Beacon.
  3. Sélectionner Désinstaller > Désinstaller
