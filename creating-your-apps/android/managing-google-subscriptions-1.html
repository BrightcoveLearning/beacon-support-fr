introduction

Ce document vous explique comment utiliser la console Google Play pour créer un compte de service, puis effectuer quelques tâches de configuration.

Créer un compte de service

Effectuez les étapes suivantes pour créer un compte de service :

  1. Naviguez jusqu'au Console de jeu Google.
  2. Dans le coin supérieur droit, cliquez sur Accédez à la console de jeu.
  3. Dans la navigation de gauche, sélectionnez Paramètres → Compte développeur → Accès API.
  4. Cliquez sur Créer un nouveau compte de service.
  5. Dans la boîte de dialogue qui apparaît, cliquez sur Plateforme Google Cloud.
  6. Depuis la page vers laquelle vous êtes redirigé, cliquez sur + CRÉER UN COMPTE DE SERVICE.
  7. Dans la boîte de dialogue qui s'ouvre, pour le Nom du compte de service Entrer Abonnements Android puis clique CRÉER.
  8. À l'étape 2, dans la liste déroulante, choisissez le Rôle être Propriétaire puis clique CONTINUEZ.
  9. À l'étape 3, cliquez sur TERMINÉ.

Configurer le compte de service

Effectuez les étapes suivantes pour configurer votre compte de service :

  1. Pour le compte de service qui vient d'être créé, sous Actions cliquez sur les trois points verticaux. Puis clique Gérer les clés.
  2. Dans le Clés informations, sélectionnez AJOUTER UNE CLÉ → Créer une nouvelle clé
  3. À partir des informations sur la clé privée, laissez le Type de clé être JSON, puis clique CRÉER.
  4. Assurez-vous que ce message s'affiche :
  5. Vous devez envoyer le fichier de clé privée par e-mail à votre responsable d'intégration.
  6. Vous pouvez fermer l'onglet que vous avez ouvert à l'étape 5 et cliquer sur Terminé à partir de la boîte de dialogue qui a lancé votre processus de création de compte de service.
  7. Dans le Comptes de service rubrique, cliquez sur Accorder l'accès à côté de votre compte de service nouvellement créé.
  8. Du Inviter un utilisateur page, dans le Autorisations de compte rubrique, cliquez sur le Administrateur (toutes les autorisations) case à cocher.
  9. En bas à droite de la page, cliquez sur Inviter un utilisateur.
  10. Dans la boîte de dialogue qui s'ouvre, cliquez sur Envoyer une invitation.
