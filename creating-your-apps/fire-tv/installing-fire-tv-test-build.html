introduction

Ce document vous montre comment installer une version de test afin que vous n'ayez pas à suivre le processus de soumission d'application avant de pouvoir tester votre application.

Pour pouvoir installer l'application Brightcove Beacon sur votre appareil Fire TV, vous aurez besoin de :
  • Mode développeur activé sur votre appareil Fire TV.
  • Android Debug Bridge (adb) installé sur votre ordinateur.

Si vous souhaitez tester les méthodes de paiement internes de l'App, telles que les abonnements ou les paiements d'événements, veuillez suivre les Instructions d'installation de Live App Testing.

Une vue de haut niveau du processus est :

  1. Activez le débogage sur votre appareil Amazon Fire TV.
  2. Connectez-vous à votre appareil Fire TV via Android Debug Bridge (adb).
  3. Installez votre application Brightcove Beacon.
  4. Exécutez votre application Brightcove Beacon.

Activation du débogage sur Amazon Fire TV

Pour activer le débogage sur votre appareil Fire TV, veuillez suivre les étapes ci-dessous :

  1. Depuis l'écran principal de votre Fire TV, sélectionnez Paramètres.
  2. Sélectionner Appareil (ou My Fire TV) > Options de développeur.
  3. Allumer Débogage ADB.
  4. Allumer Applications de sources inconnues.

Connexion via Android Debug Bridge (adb)

Vous devez utiliser Android Debug Bridge (adb) pour connecter votre ordinateur à un appareil Amazon Fire TV pour installer, tester et déboguer vos applications. Pour plus d'informations sur la façon d'activer adb, veuillez suivre le Amazone et Android Documentation.

Installation de votre application Brightcove Beacon

Pour installer et tester votre application Brightcove Beacon avant de la soumettre à l'Amazon Appstore, vous devez utiliser Android Debug Bridge (adb).

L'installation de votre propre application (en dehors de l'Appstore) est parfois appelée chargement latéral d'une application.

Pour suivre ces étapes, vous devez avoir déjà utilisé adb pour connecter votre ordinateur de développement à votre appareil Fire TV.
  1. Ouvrez votre terminal ou votre ligne de commande.
  2. Tapez la commande suivante, où path-to-apk-file est le chemin du système de fichiers vers l'APK de votre application. 
    adb install path-to-apk-file
  3. Si votre installation a réussi, adb répondra avec un message similaire à celui-ci : 
     Paquet de 764 Ko/s (217246 octets en 0,277s) : /data/local/tmp/HelloWorld.apk Succès
  4. Pour réinstaller une application qui existe déjà sur l'appareil, vous pouvez utiliser le -r possibilité de le réinstaller. 
     adb install -r chemin d'accès au fichier apk

Exécution de votre application Brightcove Beacon

Les applications téléchargées latéralement apparaissent à la fois dans le Récent rangée et dans le Ma bibliothèque rang dans le applications section. Vous pouvez également trouver votre application dans le Paramètres menu.

  1. Depuis l'écran principal d'Amazon Fire TV, sélectionnez Paramètres > Applications > Gérer les applications installées.
  2. Sélectionnez l'application Brightcove Beacon.
  3. Sélectionner Lancer l'application.

Installation de l'application Fire TV avec le service Amazon Live App Testing

Dans cette section, vous apprendrez comment installer et tester l'application Fire TV avec le service Amazon Live App Testing.

Utilisez cette procédure d'installation au cas où vous souhaiteriez tester les méthodes de paiement internes de l'application, telles que les abonnements ou les paiements d'événements.

Pour pouvoir installer l'application Fire TV sur votre appareil, vous aurez besoin d'une invitation par e-mail de test d'application en direct de Brightcove.

Dans le cas où vous ne recevez pas l'e-mail d'invitation pour le Live App Testing, veuillez suivre ce relier et activez la communication marketing pour l'Appstore en sélectionnant l'option intitulée Envoyez-moi un e-mail marketing parmi les catégories suivantes et cliquez sauvegarder.

Après avoir défini les préférences de communication marketing, le changement commencera avec le prochain et tous les tests d'invitation par e-mail suivants.

Une fois que vous avez reçu l'e-mail d'invitation à participer au Live App Testing, veuillez suivre les étapes suivantes :

  1. Dans votre e-mail d'invitation, cliquez sur le lien de la place de marché que vous testerez

    Le lien ouvrira la page de vente au détail de l'application Brightcove Beacon sur le marché spécifié.

  2. Sur la page de vente au détail de l'application, à partir du Livrer à: menu déroulant, sélectionnez l'appareil sur lequel vous souhaitez installer l'application Brightcove Beacon.
  3. Si un appareil particulier n'est pas disponible sur le marché actuel du testeur, cet appareil n'apparaîtra pas dans le menu déroulant. Si aucun des appareils ciblés par votre application n'est disponible sur une place de marché donnée, le menu déroulant sera remplacé par un Continuer bouton sur la page de vente au détail de l'application.
  4. Installez l'application Brightcove Beacon sur l'appareil spécifié. Vous pouvez tester sur autant d'appareils que vous avez associés à votre compte Amazon que l'application cible.
  5. Depuis l'écran principal d'Amazon Fire TV, sélectionnez Paramètres > Applications > Gérer les applications installées.
  6. Sélectionnez l'application Brightcove Beacon.
  7. Sélectionner Lancer l'application.

Désinstallation de votre application Brightcove Beacon

Pour désinstaller votre application de l'appareil Fire TV, procédez comme suit :

  1. Depuis l'écran principal de Fire TV, sélectionnez Paramètres > Applications > Gérer les applications installées.
  2. Sélectionnez votre application Brightcove Beacon.
  3. Sélectionner Désinstaller > Désinstaller
