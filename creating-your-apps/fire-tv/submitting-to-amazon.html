Aperçu

Ce document vous aidera à soumettre votre application Brightcove Beacon à l'Amazon Appstore.

Une vue de haut niveau du processus est :

  1. Ajoutez une nouvelle application Android dans la console développeur Amazon.
  2. Définissez la disponibilité et le prix de votre nouvelle application Fire TV.
  3. Ajoutez des descriptions textuelles pour votre application Fire TV dans l'Appstore d'Amazon.
  4. Ajoutez les icônes et les images de votre application Fire TV.
  5. Sélectionnez une classification de contenu pour votre application Fire TV.
  6. Téléchargez le fichier APK fourni par Brightcove.

Le reste de ce document fournira des détails pour chacune des étapes ci-dessus.

Ressources Amazon

  • Détails de l'Appstore : https://developer.amazon.com/home.html (nécessite un compte développeur Amazon - sans frais)
  • Guide de soumission de l'application Fire TV : https://developer.amazon.com/docs/fire-tv/submitting-your-app-to-the-amazon-appstore.html et plus généralement : https://developer.amazon.com/apps-and-games/app-submission
  • Délai suggéré pour l'approbation : Trois semaines

Informations requises pour Amazon

  • Icône de l'application Fire TV (1280 x 720px PNG (pas de transparence)).
  • Entre trois et dix captures d'écran ( 1920 x 1080px JPG ou PNG (paysage, pas de transparence).
    • Ces images ne doivent pas apparaître déformées, étirées ou pixelisées lorsqu'elles sont affichées dans l'interface utilisateur.
    • Veuillez fournir des captures d'écran de Fire TV lorsque vous soumettez votre application, ou utilisez des captures d'écran qui peuvent représenter votre application de manière générique pour tous les appareils.
    • Notez que si vous ne fournissez pas de captures d'écran spécifiques à Amazon Fire TV pour votre application lors du processus de soumission, toutes les captures d'écran de téléphone ou de tablette que vous avez fournies sont utilisées pour représenter votre application dans l'App Store Fire TV.
  • Image de fond 1920 x 1080px JPG ou PNG (paysage, pas de transparence).
  • Package APK fourni par Brightcove.
  • Afficher le titre.
  • Brève description.
  • Longue description.
  • Puces de fonctionnalités du produit (jusqu'à 10 fonctionnalités clés de votre application Fire TV).
  • Mots-clés (termes de recherche utilisés pour augmenter la visibilité de votre application).

Soumettre votre application FireTV à l'Amazon Appstore

Suivez ces étapes pour soumettre votre application FireTV à l'Amazon Appstore.

  1. Entrer le Console développeur Amazon. Tu devras créer un compte développeur Amazon pour utiliser la console.
  2. Dans le Boutique d'applications Amazon rubrique, cliquez sur le Ajouter une nouvelle application bouton.
  3. Dans le menu déroulant, sélectionnez Android. (Pour le moment, Brightcove ne prend en charge que la version Android pour une soumission d'application Fire TV.)
  4. Les Nouvelle soumission d'application formulaire s'affichera.
  5. Entrez votre Titre de l'application.
  6. Choisissez un SKU de l'application. Il s'agit d'une chaîne unique à votre application que vous définissez vous-même. Le SKU (Stock Keeping Unit) devient l'ID de votre application pour vos propres besoins de suivi, par exemple : com.amazon.example.myapp.v1.
  7. Sélectionnez un Catégorie d'application. En catégorisant correctement votre application, les utilisateurs pourront trouver votre application plus facilement, ce qui augmente les téléchargements pour votre application. Pour plus d'informations, veuillez consulter le Documentation de la catégorie Amazon.
  8. Vous pouvez marquer le Utiliser mes informations d'assistance par défaut case à cocher ou introduisez différentes informations (e-mail, téléphone, site Web) pour le contact du support client de l'application Fire TV.
  9. Clique sur le sauvegarder bouton.
  10. Après l'enregistrement, vous atteindrez les paramètres de l'application Fire TV et le Informations générales languette.
  11. Dans cet onglet, vous pourrez voir votre Clés API de soumission d'applications généré automatiquement par Amazon et toutes les informations fournies au moment de la création de votre application Fire TV.

Remplissez les informations sur la disponibilité et les prix

Dans cette section, vous pourrez définir le prix et la disponibilité de votre application Fire TV. N'oubliez pas que les prix que vous sélectionnez doivent correspondre à ceux définis sur la plate-forme Brightcove Beacon.

  1. Cliquez dans le Disponibilité et prix languette.
  2. Dans la première section, vous pourrez choisir où vous souhaitez que votre application Fire TV soit disponible :
    • Dans tous les pays et régions où Amazon vend des applications.
    • Uniquement dans certains pays et régions.
  3. Si vous sélectionnez Uniquement dans certains pays et régions , cette option vous permettra de sélectionner les pays ou les régions spécifiques qui pourront trouver votre application Fire TV sur la boutique Amazon. Dans l'exemple ci-dessous, tous les Amérique du Nord et trois pays en L'Europe  sont sélectionnés pour l'application Fire TV.
  4. Pour sélectionner un pays spécifique de la région, cliquez sur le bouton + Sélectionnez les pays/régions bouton.
  5. Sélectionnez les pays dans lesquels vous souhaitez que votre application Fire TV soit publiée dans la région de votre choix.
  6. Clique le Ajouter bouton.
  7. L'option suivante vous permettra de sélectionner si votre application Fire TV est Libérer ou Payé. Normalement, l'application Brightcove Beacon par défaut est définie comme Libérer.
  8. Si vous sélectionnez Payé , tu vas devoir Fixer un prix de base et un type de monnaie. (Veuillez contacter votre représentant de compte Brightcove Beacon si vous souhaitez attribuer un prix à votre application Fire TV).
  9. (Facultatif) Laissez ce champ vide pour démarrer le processus de publication dès que votre application Fire TV a été approuvée. Pour reporter le début du processus de publication à une date et une heure futures, attribuez une date de publication à votre application Fire TV.
  10. Clique le sauvegarder bouton.

Créer un abonnement aux achats intégrés (IAP)

Dans cette section, vous apprendrez à créer un abonnement aux achats intégrés pour votre application Fire TV.

  1. Après avoir enregistré votre Disponibilité et prix paramètres, un champ supplémentaire pour Ajouter IAP (Achats intégrés) apparaîtra.
  2. Cliquez dans le Ajouter IAP bouton pour ajouter des achats intégrés en tant qu'abonnements.
  3. Vous voyez le Éléments intégrés à l'application languette.
  4. Pour le moment, vous n'aurez aucun élément intégré à l'application. Pour créer un clic dans le Ajouter un IAP unique bouton.
  5. Notez que vous avez trois types différents d'achats intégrés à sélectionner.
    • Ajouter un consommable : Un consommable comprend tout type de contenu que vous vendez dans votre application et qui est consommé dans l'application elle-même. Ce type de contenu est disponible uniquement sur l'appareil à partir duquel il est acheté.
    • Ajouter un droit : Un droit comprend tout type de contenu que vous vendez dans votre application et qui nécessite des droits d'accès. Ce type de contenu est disponible partout où le client est connecté au client Amazon Apps. Le droit au contenu n'expire pas.
    • Ajouter un abonnement : Un abonnement comprend tout type de contenu qui :
      • porte un droit
      • est lié par une période de temps
      • Renouvellement automatique en fin de période
      • Est disponible sur tous les appareils éligibles enregistrés sur le compte Amazon du client
      • Votre application peut avoir plusieurs abonnements définis pour elle, chaque abonnement ayant plusieurs périodes.
  6. Veuillez sélectionner l'IAP de votre choix ou contacter votre représentant de compte Brightcove Beacon pour en savoir plus sur les options les mieux adaptées à votre contenu.
  7. Dans cet exemple, Abonnement est sélectionné.
  8. Donne un Titre à votre nouvel abonnement.
  9. Choisissez un UGS. Le SKU (Stock Keeping Unit) devient l'ID de votre abonnement pour vos propres besoins de suivi, par exemple abonnement.gold.
  10. Clique le sauvegarder bouton.
  11. Après avoir enregistré votre IAP, vous serez invité à Période d'abonnement languette.
  12. Sélectionner un Période d'abonnement. Vous pourrez sélectionner l'heure à laquelle l'abonnement sera facturé à vos utilisateurs. Par exemple: mensuel, hebdomadaire, annuel , etc.
  13. Choisissez un UGS. Le SKU (Stock Keeping Unit) devient l'ID de votre période d'abonnement pour vos propres besoins de suivi, par exemple Souscription mensuelle.
  14. (Facultatif) Sélectionnez si vous souhaitez donner un Essai gratuit de votre abonnement. Par exemple: 7 jours, 1 mois , etc.
  15. Sélectionnez si vous facturerez cet abonnement. Pour charger, sélectionnez le Oui, mon prix catalogue de base est... option.
  16. Saisissez le prix de votre abonnement et le type de devise.
  17. Les Prix calculés apparaîtra, ce qui vous permettra de décider si vous souhaitez qu'Amazon calcule les prix catalogue pour vous ou si vous définissez les prix manuellement.
  18. Clique le sauvegarder bouton.
  19. Allez au La description et ajoutez une description à votre nouvel abonnement.
  20. Clique le sauvegarder bouton.
  21. Lorsque tous vos paramètres IAP sont terminés, le Soumettre un élément intégré à l'application le bouton sera activé. Cliquez sur le bouton pour envoyer votre IAP pour approbation à Amazon.
  22. Clique le Soumettre un élément intégré à l'application et vous pourrez voir votre article IAP dans le Éléments intégrés à l'application liste d'onglets.

Complétez les informations de description

Dans cette section, vous pourrez ajouter toutes les descriptions textuelles de l'application Fire TV qui seront visibles par les utilisateurs dans la boutique Amazon.

  1. Cliquez dans le Version actuelle onglet et sélectionnez le La description dans le menu de l'application Fire TV.
  2. Remplissez les champs suivants dans le formulaire :
    • Brève description: Une brève description de votre application, affichée sur les appareils mobiles.
    • Longue description: Une description plus longue de votre application, pour le site Web de l'App Store.
    • Balles de caractéristique du produit: Jusqu'à 10 fonctionnalités clés de votre application, une fonctionnalité par ligne. Ces fonctionnalités apparaîtront sur le site Web de l'App Store, sous la forme d'une liste à puces.
    • (Optionnel) Mots clés: Termes de recherche utilisés pour augmenter la visibilité de votre application. Utilisez une virgule ou un espace blanc pour séparer vos termes.
  3. Par défaut, la description sera affichée en anglais mais vous pouvez ajouter des traductions en cliquant sur le Ajouter des descriptions localisées bouton.
  4. Un formulaire apparaîtra vous permettant de sélectionner les langues de traduction que vous souhaitez ajouter.
  5. Clique le Ajouter bouton.
  6. En haut du formulaire, vous pourrez choisir entre les formulaires de langue et ajouter chacune des descriptions de langue.
  7. Clique le sauvegarder bouton.

Informations complètes sur l'évaluation du contenu

La section d'évaluation du contenu vous demande de catégoriser les types de contenu que vous présenterez dans votre application Fire TV. La catégorisation Amazon divise le contenu en trois types :

  • Modérer: Se produit une fois ou rarement et n'est pas fondamental pour l'objectif général et/ou l'intention de l'application.
  • Fort: Se produit régulièrement et est fondamental pour l'objectif général et/ou l'intention de l'application.
  • Rien: Ne se produit pas dans l'application.

Les types de sujets sont violence, académique, violence des dessins animés.

  1. Cliquez dans le Évaluation du contenu languette.
  2. Sélectionnez un type de catégorie pour chacun des sujets. Toutes ces informations correspondent aux types de contenu de votre application Fire TV. Si vous ne savez pas quoi répondre, veuillez contacter votre représentant de compte Brightcove Beacon.
  3. Dans le Information additionnelle section, répondre Oui ou Non selon les fonctionnalités que contient votre application Fire TV. Toutes ces informations correspondent aux informations et fonctionnalités supplémentaires que votre application Fire TV peut contenir ou non. Si vous ne savez pas quoi répondre, veuillez contacter votre représentant de compte Brightcove Beacon.
  4. Si votre application Fire TV collecte des informations personnelles, vous devez ajouter un URL de la politique de confidentialité pour examiner vos déclarations de politique de confidentialité pour les utilisateurs.
  5. Clique sur le sauvegarder bouton.

Informations complètes sur les fichiers du kit de package Android (APK)

Les Fichiers APK section est l'endroit où le fichier d'application Android doit être téléchargé. L'APK vous est fourni par Brightcove.

  1. Cliquez dans le Fichiers APK languette.
  2. Dans le Appliquer Amazon DRM ? option, sélectionnez Non.
  3. Dans le Fichier APK section, cliquez sur la case de droite et téléchargez le fichier APK qui vous a été fourni par Brightcove.
  4. Une fois le fichier téléchargé, vous pourrez voir votre fichier à l'écran. Cliquez dans le Vue bouton.
  5. Les Prise en charge de l'appareil Une fenêtre s'ouvrira, sur laquelle vous pourrez sélectionner les appareils sur lesquels votre application Fire TV sera prise en charge.
  6. Cliquez dans le Amazon Fire TV onglet et sélectionnez tous les appareils. Brightcove Beacon n'est disponible que pour Amazon Fire TV, alors ne sélectionnez pas le Tablettes et téléphones Amazon Fire et Appareils non Amazon onglets.
  7. Fermer la Prise en charge de l'appareil la fenêtre.
  8. Dans le Ajouter d'autres détails section, sélectionnez au moins une langue prise en charge dans la liste. Il s'agit de la langue dans laquelle les utilisateurs communiqueront pour obtenir de l'aide.
  9. Pour le Instructions de test champ, fournissez des instructions spéciales à l'équipe de test Amazon, telles que les informations de connexion requises pour tester les flux vidéo, ou des notes qui expliquent comment utiliser les différents niveaux d'abonnement.
  10. marque le Conformité à l'exportation et Utiliser la redirection Amazon Maps cases à cocher.
  11. Clique le sauvegarder bouton.
  12. Après l'enregistrement, le État de l'examen onglet apparaîtra dans le menu.
  13. Vous pourrez vérifier les informations sur votre application Fire TV comme votre dernière activité et l'historique de vos soumissions. S'il n'y a pas d'erreurs, cliquez sur le Soumettre l'application dans le coin supérieur gauche de l'écran pour soumettre votre application Fire TV.

Il ne vous reste plus qu'à attendre qu'Amazon approuve votre application Fire TV et la publie dans l'Amazon Appstore.
