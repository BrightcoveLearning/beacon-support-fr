Qu'est-ce que la balise Brightcove ?

Brightcove Beacon est une plate-forme qui vous permet de créer et de configurer une expérience de streaming multimédia unique qui est ensuite utilisée sur plusieurs appareils. Avec Brightcove Beacon, vous pouvez organiser votre contenu, mettre en œuvre des stratégies de monétisation et concevoir des expériences de visionneuse. Ce faisant, vous pouvez :

  • Réduisez considérablement les coûts globaux. Les coûts de développement d'applications pour chaque plate-forme sont considérables, allant de 50 000 $ à 250 000 $ pour une application personnalisée ciblée sur UNE seule plate-forme. Brightcove Beacon permet la création de plusieurs applications à partir d'une seule expérience configurée.
  • Offrez une expérience cohérente aux téléspectateurs sur toutes les plateformes.
  • Autorisez la monétisation du contenu vidéo via des options telles que AVOD et SVOD.

Les applications peuvent être générées pour s'exécuter sur les quatre types d'appareils suivants (quelques exemples spécifiques inclus) :

  • Mobile (Téléphone / Tablettes)
    • Android
    • iOS
  • Téléviseurs connectés
    • Télévision Android
    • Apple TV
    • Télé d'incendie
    • Roku
  • Téléviseurs intelligents
    • Samsung
    • Sony
    • Toshiba
  • la toile

Non seulement vous obtenez plusieurs applications d'appareil à partir d'une seule source, mais Brightcove Beacon vous permet également de contrôler le contenu et la mise en page de chaque application d'appareil à partir de votre expérience Brightcove Beacon unique.

Comment Brightcove Beacon est-il implémenté ?

Les phases répertoriées dans ce tableau sont détaillées dans le Phases de mise en œuvre document.

Phase de mise en oeuvre La description
Préparez-vous à la création d'une expérience Brightcove Beacon* Rassemblez des artefacts de conception, créez la conception de votre application et planifiez des stratégies de publicité et de monétisation.
Établir des relations avec des fournisseurs tiers* Créez des comptes pour : Les magasins auxquels vous souhaitez soumettre des applications d'appareils, un serveur publicitaire, un fournisseur de paiement en ligne, etc.
Créez votre conception d'application Déterminez votre architecture d'informations autour des vidéos qui seront disponibles pour la visualisation, ainsi que la manière dont elles sont regroupées et ordonnées.
Ingérez ou ajoutez du contenu et configurez votre expérience Brightcove Beacon Configuration et éléments multimédias ajoutés à votre Brightcove Beacon Experience.
Créez des applications de périphérique à partir de votre expérience Brightcove Beacon Travaillez avec un représentant Brightcove pour finaliser la configuration de votre Brightcove Beacon Experience, à partir de laquelle les applications de l'appareil sont ensuite générées.
Soumettez les applications de votre appareil aux magasins correspondants Soumettez la ou les applications d'appareil nouvellement générées aux magasins correspondants pour approbation, puis vos applications sont prêtes à être utilisées par les clients.

* Les deux premières phases doivent être exécutées simultanément.

Comment les applications de l'appareil sont-elles mises à jour ?

Vos applications d'appareils spécifiques sont initialement soumises aux fournisseurs pour approbation et ne doivent être soumises à nouveau que dans de très rares cas, comme une mise à jour du système d'exploitation. Bien sûr, vous souhaiterez modifier les médias proposés pour la visualisation, modifier la mise en page de l'application ou modifier votre stratégie de monétisation pour la publicité. Cela se fait en modifiant l'expérience, ce qui ne nécessite PAS de resoumission à un magasin de fournisseur.
