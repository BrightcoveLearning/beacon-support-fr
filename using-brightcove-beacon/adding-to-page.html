Ajout d'un carrousel à une page existante

Dans cette rubrique, vous apprendrez à ajouter un groupe de vidéos dans un carrousel à une page Brightcove Beacon.

Aperçu

La page Brightcove Beacon est l'endroit où les utilisateurs sélectionnent et regardent votre contenu vidéo. Les carrousels affichent un groupe de vidéos ou de listes de lecture que vous avez définies dans Video Cloud Studio. À l'aide de la console d'administration Brightcove Beacon, vous pouvez ajouter des carrousels à une page existante.

Ajout d'un carrousel

Pour ajouter un carrousel à une page existante, procédez comme suit :

  1. Connectez-vous à Brightcove Beacon.

  2. Dans la navigation supérieure, sélectionnez Pages.

  3. Vous pouvez utiliser le Ajouter une nouvelle page bouton pour créer une nouvelle page. Une fois les pages définies, vous pouvez facilement ajouter un carrousel à une page existante. Sélectionnez la page que vous souhaitez mettre à jour. Ici, nous sélectionnons le lien pour le Accueil page.

  4. Notez que le Donnee de base est déjà peuplé.

  5. Sélectionnez le Ajouter un bloc bouton.

  6. Choisissez le type de bloc que vous souhaitez. Dans ce cas, nous choisissons le liste-bloc. Sélectionnez le Ajouter un bloc bouton associé au bloc que vous souhaitez.

  7. Sélectionnez le Ajouter une mise en page bouton.

  8. Choisissez le type de mise en page que vous souhaitez, qu'il s'agisse d'une présentation, d'un carrousel ou d'une grille. Dans cet exemple, nous sélectionnons le Étiquettes de paysage de carrousel dans disposition.

  9. Maintenant que vous avez défini le conteneur pour le carrousel, vous êtes prêt à ajouter du contenu. Dans l'élément de bloc de liste, sélectionnez le Ajouter une liste de lecture bouton.

  10. Choisissez une liste de lecture. Dans cet exemple, nous sélectionnons le Ajouter une liste de lecture bouton associé au Humour playlist.

    Vous créez des listes de lecture à l'aide de Video Cloud Studio. Ils apparaîtront ici une fois que vous aurez synchronisé avec votre compte Video Cloud.

  11. Nous pouvons voir que le Humour playlist a été ajoutée à notre carrousel. Vous pouvez faire glisser et déposer pour réorganiser les articles. Sélectionnez le Mettre à jour la page bouton pour ajouter le nouveau carrousel à la page.

  12. Vous êtes prêt. Vous devriez voir le nouveau carrousel sur votre page Brightcove Beacon. Consultez les sections suivantes pour plus de détails sur les valeurs de champ.
