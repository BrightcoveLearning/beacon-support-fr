Aperçu

UNE séries est une collection de vidéos similaires présentées dans un ordre particulier, qui partagent généralement un thème commun. Lors de la création d'une série, vous devez définir les relations entre la série, ses saisons et les épisodes associés.

UNE saison est un groupe de vidéos qui couvrent une période de temps définie. Chaque série peut avoir une ou plusieurs saisons.

Un épisode est une vidéo individuelle. Vous pouvez considérer cela comme un film spécifique.

Une fois que vous avez défini vos séries et saisons, et ingéré vos épisodes, ils peuvent être utilisés dans votre application Brightcove Beacon. Pour le faire, suivez ces étapes:

  1. Série (Brightcove Beacon)

    Dans Brightcove Beacon, créez la série. Ici, vous nommerez la série et fournirez des informations, y compris la disponibilité et des images.

  2. Saisons (Brightcove Beacon)

    Dans Brightcove Beacon, créez les saisons liées à la série. Vous fournirez des informations similaires à la série, en plus de sélectionner la série à laquelle appartient la saison.

  3. Épisodes (Cloud vidéo)

    La vérité du contenu des épisodes de Brightcove Beacon réside dans Video Cloud.

    Dans Video Cloud Studio, attribuez les épisodes vidéo à la saison correspondante à l'aide des valeurs fournies aux champs personnalisés. Ces champs personnalisés incluent :

    • beacon.episode.number
    • beacon.episode.seasonNumber
    • beacon.episode.serieName

  4. Retournez dans Brightcove Beacon et ingérez vos épisodes vidéo.

Création d'une série

Pour créer une série, procédez comme suit :

  1. Connectez-vous à Brightcove Beacon et sélectionnez Programmes.
  2. Dans la navigation de gauche, sélectionnez Séries.
  3. Clique le Ajouter une nouvelle série bouton.

  4. Sur le Donnee de base onglet, remplissez les champs suivants :

    Nom du champ de formulaire Description / Exigences
    Nom L'identifiant de la série, le nom face au client est fourni plus loin dans le Données textuelles section
    Année de production L'année de production de la série ; ces informations apparaîtront dans le Plus d'informations filtrer
    Pays de production Le pays où la série est produite ; ces informations apparaîtront dans le Plus d'informations filtrer
    Genres Les genres de la série ; notez que vous pouvez créer des genres selon vos besoins :
    Sous-genres Ne pas compléter, ces données ne sont pas utilisées
    Aperçu de la langue du genre La langue de l'aperçu
    Type de tri des épisodes Comment trier les épisodes dans l'interface utilisateur de l'application
    Jeter Les métadonnées sous JETER apparaîtra dans le Plus d'informations filtrer.
  5. Clique sur le Données textuelles languette.
  6. Remplissez le Titre (qui est le nom du spectateur pour la série), La description et Synopsis. Ces données seront affichées dans l'application Brightcove Beacon.
  7. Clique sur le Données non textuelles languette.
  8. Fournissez une image pour les éléments suivants (correspondant aux exigences ci-dessous) :
    • Affiche de portrait (obligatoire)
    • Affiche de paysage (obligatoire)
    • La vignette
    • Large bannière (obligatoire)

    Si elle n'est pas fournie, l'image miniature sera générée à partir de l'affiche portrait.

    Image Conditions
    Affiche de portrait

    Image JPEG (2:3)

    Le minimum Largeur: 1000px Hauteur: 1500px
    Maximum Largeur: 2000px Hauteur: 3000px
    Affiche de paysage

    Image JPEG (16:9)

    Le minimum Largeur: 1000px Hauteur: 563px
    Maximum Largeur: 4000px Hauteur: 2250px
    La vignette Image JPEG (4:3)
    Bannière large Image JPEG (21:9)
    Le minimum Largeur: 1000px Hauteur: 428px
    Maximum Largeur: 4000px Hauteur: 1714px
  9. Les informations sur le Mots clés n'est pas utilisé pour le moment, donc n'entrez pas d'informations sur cet onglet.
  10. Les informations sur le Métadonnées n'est pas utilisé pour le moment, donc n'entrez pas d'informations sur cet onglet.
  11. Clique sur le Ruisseaux onglet, où vous sélectionnerez une bande-annonce pour la série. La capture d'écran suivante vous guidera tout au long du processus de sélection de cette bande-annonce.
  12. Sélectionnez votre Fournisseur. Vous n'aurez probablement qu'un seul fournisseur, qui fait référence à votre compte Brightcove Video Cloud.
  13. Pour le Identifiant de la vidéo en streaming , obtenez l'ID vidéo de la bande-annonce dans Studio et placez-le dans la zone de texte.
  14. Assurez-vous que le Type de flux est réglé sur Bande-annonce.
  15. Clique sur le Droits et planification onglet, comme indiqué ici :

  16. Ajoutez des valeurs pour les champs suivants :

    Champ Obligatoire Remarques
    Statut Oui Sélectionnez l'un des types suivants :
    • Publié
    • Inédit
    Type de droits Oui Choisissez la valeur appropriée pour votre série, parmi les options :
    • Publicité - AVOD
    • Libérer
    • Abonnement - SVOD
    Heure de début Oui Planifiez une date de début pour laquelle votre série sera disponible
    Heure de fin Oui Planifiez une date de fin à laquelle votre série ne sera plus disponible
    Flux max. Oui Définir le nombre maximum de streams pouvant regarder la série à partir du même compte
    Appareils à publier Oui Ajoutez les appareils sur lesquels vous souhaitez publier. Sélectionnez le Sélectionner / Désélectionner tous les appareils pour voir une liste des appareils disponibles
    Emplacements autorisés Oui

    Sélectionnez les emplacements où vous souhaitez diffuser votre contenu, si aucune restriction entrez Monde.

    Si vous ne spécifiez pas au moins 1 emplacement autorisé, ce message peut s'afficher dans votre application :
    Emplacements refusés Non Sélectionnez les emplacements où vous ne souhaitez PAS diffuser votre contenu
  17. Assurez-vous de cliquer sur le Créer une nouvelle série bouton pour enregistrer les informations de la série.

Vous avez maintenant créé votre série. L'étape suivante consiste à créer une ou plusieurs saisons qui appartiennent à cette série. La section suivante vous guide à travers ce processus.

Créer des saisons

Pour créer une saison, procédez comme suit :

  1. Connectez-vous à Brightcove Beacon et sélectionnez Programmes.
  2. Dans la navigation de gauche, sélectionnez Saisons.
  3. Clique le Ajouter une nouvelle saison bouton.
  4. Sur le Donnee de base onglet, remplissez les champs suivants :
    Nom du champ de formulaire Description / Exigences
    Sélectionnez une série La liste déroulante sera remplie avec la série que vous avez créée. Choisissez la série à laquelle appartient cette saison.
    Nom L'identifiant de la série, le nom face au client est fourni plus loin dans le Données textuelles section
    Numéro de saison Il doit s'agir d'un nombre entier représentant la séquence de saison dans la série. Ce numéro sera utilisé lors de l'ingestion de vidéos d'épisodes à partir de Video Cloud.
    Année de production L'année de production de la saison ; ces informations apparaîtront dans le Plus d'informations filtrer.
    Pays de production Le pays où la saison est produite ; ces informations apparaîtront dans le Plus d'informations filtrer.
  5. Clique sur le Données textuelles languette.
  6. Remplissez le Titre, La description et Synopsis pour la saison. Ces données seront affichées dans l'application Brightcove Beacon.
    Pour la saison Titre, vous pouvez entrer un nom comme Saison 1, mais si une série a des sujets pour des saisons distinctes, cela peut être un meilleur choix. Par exemple, une série sur les continents où chaque saison se concentre sur un continent, de bons noms pour les saisons seraient L'Europe , Afrique, Asie, etc.
  7. Clique sur le Données non textuelles languette.
  8. Pour votre saison, fournissez une image pour les éléments suivants (correspondant aux exigences ci-dessous) :
    • Affiche de portrait (obligatoire)
    • Affiche de paysage (obligatoire)
    • La vignette
    • Large bannière (obligatoire)

    Si elle n'est pas fournie, l'image miniature sera générée à partir de l'affiche portrait.

    La vignette Image JPEG (4:3)

    Le minimum Largeur: 1000px Hauteur: 428px

  10. Les informations sur le Métadonnées n'est pas utilisé pour le moment, donc n'entrez pas d'informations sur cet onglet.
  11. Clique sur le Ruisseaux onglet, où vous sélectionnerez une bande-annonce pour la saison.
  12. Sélectionnez votre Fournisseur. Vous n'aurez probablement qu'un seul fournisseur, qui fait référence à votre compte Brightcove Video Cloud.
  13. Clique le Ajouter un nouveau flux bouton.
  14. Pour le Identifiant de la vidéo en streaming , obtenez l'ID vidéo de la bande-annonce dans Studio et placez-le dans la zone de texte.
  15. Assurez-vous que le Type de flux est réglé sur Bande-annonce.
  16. Clique sur le Droits et planification onglet, comme illustré ici.

  17. Ajoutez des valeurs pour les champs suivants :

    Champ Obligatoire Remarques
    Statut Oui Sélectionnez l'un des types suivants :
    • Publié
    • Inédit
    Type de droits Oui Choisissez la valeur appropriée pour votre saison, parmi les options :
    • Publicité - AVOD
    • Libérer
    • Hériter
    • Abonnement - SVOD

Vous avez maintenant créé une de vos saisons. Si nécessaire, créez plus de saisons pour la série. L'étape suivante consiste à ingérer les vidéos des épisodes de Video Cloud dans Brightcove Beacon. La section suivante vous guide à travers ce processus.

Ingérer des épisodes

Alors que la création de la série et des saisons associées se fait dans Brightcove Beacon, la création des épisodes de la série se fait dans Video Cloud Studio. Vous attribuerez des valeurs sur la série et les saisons dans des champs personnalisés pour chaque vidéo d'épisode, puis ingérerez ces vidéos dans Brightcove Beacon.

N'oubliez pas que la vérité du contenu des épisodes de Brightcove Beacon réside dans Video Cloud. Toutes les modifications que vous apportez aux épisodes dans Brightcove Beacon seront écrasées lorsque vous ingérez du contenu à partir de Video Cloud.

Conditions préalables

Avant de poursuivre les étapes ci-dessous, assurez-vous d'avoir effectué les opérations suivantes :

  1. Connectez-vous à Video Cloud Studio.

  2. Dans le menu déroulant en haut de la page, sélectionnez TÉLÉCHARGER.

  3. Téléchargez vos vidéos d'épisodes.

  4. Dans le menu déroulant en haut de la page, sélectionnez MÉDIAS.

  5. Mettez à jour vos images vidéo d'épisode comme spécifié dans le Images et leurs spécifications document.

  6. Dans l'en-tête de navigation, cliquez sur ADMINISTRER puis Champs vidéo.

    Vous devez être administrateur de compte pour accéder au ADMINISTRER menu.
  7. Créez les champs personnalisés requis pour ingérer des vidéos dans Brightcove Beacon. Pour plus de détails, consultez le Création de champs personnalisés Video Cloud document.

Pas

Pour ingérer des vidéos d'épisodes dans Brightcove Beacon, procédez comme suit :

  1. Revenez à Video Cloud Studio.
  2. Dans le menu déroulant en haut de la page, sélectionnez MÉDIAS.

  3. Cliquez sur le titre de l'une des vidéos de l'épisode pour accéder à la page de propriétés de la vidéo.

  4. Faites défiler vers le bas jusqu'à ce que vous voyiez sur le côté droit le LES CHAMPS PERSONNALISÉS section.
  5. Clique le Éditer bouton.
  6. Localisez les trois champs personnalisés associés aux épisodes.
  7. Pour la vidéo de l'épisode, entrez le numéro de l'épisode, le numéro de la saison et le nom de la série dans les champs personnalisés correspondants.
  8. Assurez-vous de cliquer sur le sauvegarder après avoir entré les valeurs appropriées pour la vidéo.
  9. Répétez la saisie des valeurs dans les champs personnalisés pour toutes vos vidéos d'épisodes.
  10. Connectez-vous à Brightcove Beacon.
  11. Clique sur le Outils option (clé icône) dans le menu du haut.

  12. Assurez-vous que le Ingestion l'option est sélectionnée dans le menu de gauche.

  13. Clique le Mettre à jour les vidéos Brightcove bouton pour ingérer les vidéos des épisodes nouvellement configurés.

  14. Pour rendre les vidéos immédiatement disponibles dans vos applications, effacez le cache de Brightcove Beacon en cliquant sur le bouton Cacher dans la navigation de gauche, puis cliquez sur le Purge du cache bouton.

    Pour plus de détails sur l'utilisation des options de Brightcove Beacon Tools, consultez le Utilisation des outils document.
