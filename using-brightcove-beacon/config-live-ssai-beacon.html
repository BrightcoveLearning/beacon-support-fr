Ce processus de configuration des annonces en direct n'est nécessaire que si la chaîne en direct a été activée avec SSAI. Si ce n'est pas le cas, ces étapes ne sont pas nécessaires.

Conditions préalables

  • Créez une chaîne en direct activée avec SSAI dans Video Cloud.
  • Créez et configurez le tag publicitaire dans Video Cloud.
  • Créez un canal dans Brightcove Beacon.
  • Créez la configuration de la publicité dans Brightcove Beacon.
  • Définissez les droits et la programmation dans le canal.

Configuration de l'annonce Video Cloud avec Brightcove Beacon

  1. Dans Video Cloud Studio, sélectionnez le HABITENT module.
  2. Créez ou ouvrez le canal Live SSAI que vous souhaitez utiliser avec Brightcove Beacon.
  3. Dans le Salle de contrôle , Cliquez sur Publier et intégrer du panneau de gauche.
  4. Dans le Options de lecteur en direct section, dans la liste déroulante, sélectionnez le Configuration de l'annonce vous voulez définir.
  5. Une fois sélectionné un nouveau Code d'intégration standard apparaîtra.
  6. Extraire le adConfigId en copiant une partie de l'URL attribuée à adConfigId, comme indiqué dans l'image.

Créer la publicité dans Brightcove Beacon

  1. Retournez sur la plate-forme Brightcove Beacon.
  2. Cliquez sur Publicité.
  3. Créez ou sélectionnez la configuration d'annonce que vous souhaitez définir.
  4. Pour une nouvelle publicité, donnez-lui un Nom.
  5. Assurez-vous que le Fournisseur est réglé sur Brightcove SSAI.
  6. Dans le ID de configuration en direct champ, collez le adConfigId URL que vous avez copiée dans la section précédente de ce document.
  7. Cliquez sur Créer une nouvelle publicité.

Définition des droits et planification dans Brightcove Beacon

  1. Dans la navigation supérieure, cliquez sur Canaux.
  2. Sélectionnez le canal dont vous souhaitez définir la configuration SSAI.
  3. Clique le Droits et planification languette.
  4. Dans Type de droits , sélectionnez Publicité - AVOD.
  5. Dans Configuration de la publicité , dans la liste déroulante, sélectionnez la configuration d'annonce que vous venez de créer.
  6. Cliquez sur Créer une nouvelle chaîne pour sauvegarder votre travail.
