Aperçu

Pour diffuser un événement en direct sur votre application, vous devez suivre une série d'étapes pour connecter votre compte Beacon Studio Live et Beacon Classic.

Une vue de haut niveau du processus est :

  1. Dans Beacon Classic, créez un événement et une liste de lecture.
  2. Dans Beacon Studio, accédez au module Live pour créer votre événement et obtenir un job ID.
  3. Dans Beacon Classic, enregistrez votre événement en direct à l'aide de l'ID de tâche de Beacon Studio.
  4. La vidéo en direct commencera à être diffusée dans l'application Beacon.

Dans la dernière section de ce document, vous verrez les étapes pour créer un type d'événement différent dans Live en utilisant un point d'entrée statique. Cela permet des événements en direct récurrents qui peuvent être activés et désactivés et ne doivent être configurés qu'une seule fois.

Le reste de ce document fournira des détails pour chacune des étapes ci-dessus.

Créer un événement en direct dans Beacon Classic

Suivez ces étapes pour créer un événement dans Beacon Classic.

Seuls les comptes activés avec Balise Classique Live aura le Listes de lecture et Événements disponible dans la section onglets. Veuillez contacter l'assistance Brightcove si vous ne les avez pas activés.
  1. Dans Beacon Classic, cliquez sur le Événements onglet pour démarrer la configuration d'un événement.
  2. Cliquer sur Ajouter un nouvel événement.
  3. Donnez à l'événement en direct un Nom Et un Date de début de l'événement.
  4. Dans le Données textuelles onglet, remplissez les informations de votre événement.
  5. Clique sur le Données non textuelles languette.
  6. Fournir une image pour le Affiche de portrait et Affiche de paysage correspondant aux exigences ci-dessous. Vous N'AVEZ PAS besoin de fournir des images pour La vignette et Large bannière.
    Image Conditions
    Affiche de portrait Image de l'affiche (2:3)
    Le minimum Largeur: 1000px Hauteur: 563px
    Maximum Largeur: 4000px Hauteur: 2250px
    Affiche de paysage Image paysage (16:9)
    Le minimum Largeur: 1000px Hauteur: 563px
    Maximum Largeur: 4000px Hauteur: 2250px
  7. Clique le Droits et planification onglet, comme indiqué ici :
  8. Ajoutez des valeurs pour les champs suivants :
    • Type de droits: Choisissez le type de droits entre Publicité, Abonnement ou Libérer.
    • Heure de début: Définissez la date de début de votre événement.
    • Heure de fin: Définissez la date de fin de votre événement.
    • Flux max.: Définissez le nombre maximal de flux autorisés pour afficher l'événement en direct. Le nombre par défaut est 0 (zéro).
    • Appareils à publier: Ajoutez les appareils sur lesquels vous souhaitez publier. Cliquez sur Sélectionner / Désélectionner tous les appareils pour voir la liste des appareils disponibles.
    • Emplacements autorisés: Sélectionnez les emplacements où vous souhaitez diffuser votre contenu, si aucune restriction entrez Monde.
    • Emplacements refusés: Sélectionnez les emplacements où vous ne souhaitez PAS diffuser votre contenu.
  9. Clique le Créer un nouvel événement bouton pour créer votre événement.

Ajout de l'événement Beacon Classic à une liste de lecture

Une fois que vous avez créé un événement, la tâche suivante consiste à ajouter l'événement à une liste de lecture. Les étapes suivantes vous guident tout au long de ce processus.

  1. Dans Beacon Classic, cliquez sur le bouton Listes de lecture languette.
  2. Clique sur le Ajouter une nouvelle liste de lecture bouton.
    Vous devez créer une liste de lecture dans Beacon Classic qui contiendra vos événements en direct. Ne pas utiliser une liste de lecture de Video Cloud. L'utilisation d'une liste de lecture Video Cloud peut écraser et supprimer vos enregistrements Beacon Classic Live.
  3. Dans Listes de lecture , dans les onglets de type de contenu, sélectionnez le Événements languette.
  4. Recherchez votre événement dans la liste des contenus et cliquez sur le bouton (+) pour l'ajouter à la liste de lecture de gauche. Vous pouvez positionner l'ordre des événements de la manière que vous souhaitez qu'ils soient affichés dans l'application.
  5. Ajustez les paramètres de votre liste de lecture (heure, appareils, emplacement, etc.) dans le Droits et planification onglet, comme vous l'avez fait avec votre événement.
  6. Clique sur le Enregistrer les modifications dans la liste de lecture bouton.
  7. Clique sur le Outils (clé) en haut à droite de l'écran. Videz ensuite le cache.

À ce stade, votre événement est créé en tant que conteneur dans l'application Brightcove Beacon comme indiqué dans l'image de gauche (les noms d'événements et les images sur l'image sont à titre d'exemple uniquement), cliquez sur votre événement et vous verrez quelque chose de similaire à l'image à droite. Pour réellement commencer à diffuser une vidéo en direct, vous devrez configurer votre transmission dans la plate-forme Beacon Studio Live.

Créer un événement en direct dans Beacon Studio

Suivez ces étapes pour créer l'événement en direct réel qui sera diffusé sur l'application Brightcove Beacon.

Pour plus d'informations sur tous les composants du module Live, veuillez visiter le Gestion de la documentation des événements en direct.
  1. Connectez-vous à Beacon Studio et cliquez sur Habitent dans la navigation principale.
  2. Du Gérer les événements page, cliquez sur Créer un évènement. Les Créer un événement en direct la page s'ouvrira.
  3. Saisissez les détails de l'événement.
    • Nom de l'événement - Nom de l'événement ; ce nom apparaîtra également dans le module Média si l'événement en direct est enregistré en tant qu'élément vidéo (255 caractères max.)
    • brève description - Description de l'événement (max 250 caractères)
    • Mots clés - Tags à associer à l'événement ; les tags seront attribués à la vidéo si l'événement en direct est enregistré en tant qu'élément vidéo (max 128 caractères et 1200 tags)
    • Pièce d'identité - Identifiant de référence pour l'événement ; doit être unique (max 150 caractères)
  4. Cliquez sur Autorisé si vous souhaitez permettre à l'événement en direct d'être diffusé sur les plateformes de médias sociaux, sinon cliquez sur Interdit. Ceci peut être fait une fois que l'événement a commencé à diffuser ou en utilisant le Module social.
  5. Sélectionner un Profil d'ingestion en direct. La liste déroulante répertorie tous les profils d'ingestion en direct. Le profil d'ingestion contrôlera la qualité et le nombre de rendus en direct créés. En cliquant sur le Gérer les profils lien ouvrira l'éditeur de profil d'ingestion dans un autre onglet du navigateur. Notez que vous devez être un administrateur de compte pour accéder à l'éditeur de profil d'ingestion. Clique le RENDUS lien pour afficher les rendus qui seront créés à l'aide du profil d'acquisition sélectionné.
    Les détails de débit et de facturation des trois niveaux sont les suivants :
    • Le niveau 1 permet jusqu'à 5 500 Kbps et est facturé à 1,0 fois les heures
    • Le niveau 2 permet jusqu'à 10 000 Kbps et est facturé 1,5 fois les heures
    • Le niveau 3 permet jusqu'à 20 000 Kbps et est facturé 2,0 fois les heures
  6. Sélectionner Convertir l'événement en élément vidéo une fois terminé pour enregistrer une version VOD de l'événement en direct sur votre compte Beacon Studio. Lorsque cette option est sélectionnée, le rendu de la plus haute qualité à partir du Profil d'ingestion en direct est utilisé comme source pour la vidéo VOD. Le profil d'ingestion par défaut du compte est utilisé lors de la création de rendus. Notez que la version VOD n'est pas disponible dès la fin de l'événement en direct, la vidéo sera disponible une fois le processus d'intégration terminé.
    Noter: Cette option n'est pas prise en charge lors de l'utilisation d'un événement récurrent.
  7. Cliquez sur OPTIONS AVANCÉES et afficher les options avancées. Dont un seul nécessitera votre attention.
  8. Cliquez dans le Activer les notifications d'état du flux cochez la case et ajoutez l'URL des notifications de Beacon Classic CMS (votre représentant de l'assistance Brightcove vous fournira cette URL).
    Les URL de notification configure des écouteurs pour déclencher d'autres actions dans l'événement. Pour plus d'informations sur la notification d'état et les changements d'état pour les événements VOD et Live, veuillez visiter le API en direct : Documentation des notifications .
  9. Cliquez sur Créer un évènement pour démarrer l'événement en direct. La page de la salle de contrôle s'ouvrira avec des détails sur l'événement en direct, y compris le Point de terminaison de streaming (URL RTMP) et Nom du flux qui sera utilisé par votre encodeur. Il faudra environ une minute pour traiter vos flux.
    Pour commencer à diffuser sur la plate-forme Live, vous aurez besoin d'un encodeur. Pour plus d'informations sur l'utilisation d'un encodeur, veuillez lire la documentation suivante :

    Pas à pas: Utilisation du Live Module avec Telestream Wirecast

    Pas à pas: Utilisation du Live Module avec le logiciel Open Broadcaster (OBS)

  10. Dans le Salle de contrôle , copiez le Identifiant de la tâche et le Identifiant de la vidéo du Evénement test panneau, vous aurez besoin des deux pour configurer l'événement en direct dans les paramètres de la plate-forme Beacon Classic.
  11. Ensuite, retournez à Beacon Classic et cliquez dans le Événement languette. Sélectionnez l'événement que vous y avez créé précédemment, et dans la Donnee de base paramètres coller le ID de tâche de votre événement Live.
  12. Allez au Ruisseaux onglet et créez un nouveau flux en cliquant dans le + Ajouter un nouveau flux bouton.
  13. Dans le Identifiant de la vidéo en streaming champ, collez le Identifiant de la vidéo à partir de l'événement Beacon Studio que vous avez créé précédemment. Remplissez également les informations suivantes sur le flux :
    • Sélectionnez le nombre d'appareils sur lesquels vous pouvez publier votre vidéo.
    • Met le Type de flux à Habitent dans le menu déroulant.
    • Et cliquez sur le Événement de mise à jour bouton avant de quitter la page.
  14. Clique sur le Outils (clé) en haut à droite de l'écran. Videz ensuite le cache.
  15. Ouvrez votre application Beacon et sélectionnez votre événement (les noms des événements et les images sur l'image sont à titre d'exemple uniquement).
  16. Clique sur le Jouer et vous devriez maintenant pouvoir voir votre événement en direct en streaming depuis Beacon Studio dans l'application Beacon Classic.

Une fois votre événement terminé, vous pouvez l'enregistrer en tout ou en partie sous forme de clip qui devient une ressource vidéo dans Beacon Studio ou un flux jouable qui peut être affiché dans Beacon Classic. Le processus est détaillé dans la section suivante de ce document.

Création d'un clip à partir de votre événement en direct

Vous devez créer un clip de votre événement Beacon Classic Live si vous souhaitez conserver la vidéo pour la diffusion ultérieure en tant que vidéo enregistrée en VOD.

Le module Live offre la possibilité de créer des clips vidéo à partir d'événements terminés. Des clips peuvent également être créés pendant qu'un événement en direct est en cours.

Pour une liste complète des fonctionnalités d'écrêtage, veuillez consulter le Création d'un clip vidéo à l'aide du module Live Documentation.

  1. Dans le module Live, accédez au Créer un clip onglet dans le panneau de gauche.
  2. À l'aide des boutons de découpage ou des touches de raccourci, définissez les points de début et de fin du clip. Vous pouvez aussi simplement cliquer sur le sauvegarder bouton pour garder la vidéo entière.
  3. Entrez les propriétés de la vidéo du clip.
    • Nom du clip - Nom du clip
    • brève description - Description du clip
    • Mots clés - Tags à associer au clip
    • Pièce d'identité - ID de référence à attribuer au clip
    • Supprimer le contenu de coupure publicitaire source - Cette option supprime les coupures publicitaires déclenchées dans le clip, en ne conservant que le contenu du programme
      Noter: La case à cocher Supprimer la coupure publicitaire source n'apparaîtra que lorsque le flux est activé SSAI.
  4. Cliquez sur Créer un clip. Le processus de transcodage du clip commencera. Les clips sont transcodés à l'aide du profil d'acquisition par défaut du compte.

Ajout de votre clip à Beacon Classic en tant que flux vidéo jouable

Ici, vous utiliserez l'ID de votre clip vidéo nouvellement créé et en ferez un flux pouvant être lu dans Beacon Classic.

  1. Toujours dans le module Live de Beacon Studio, copiez le numéro identifiant de votre clip créé.
  2. Dans la plate-forme Beacon Classic, accédez à la Événements onglet et sélectionnez le Ruisseaux menu. Vous verrez le formulaire suivant :
  3. Collez le identifiant vous venez de copier de la plate-forme Live de Studio dans le Identifiant de la vidéo en streaming champ.
  4. Du Type de flux changement de liste déroulante Habitent à Flux.
  5. Cliquez dans le Donnee de base onglet et modifiez le Statut de Terminé à Fini.
  6. Avant de partir, cliquez sur le Mettre à jour l'événement bouton en bas de la page. Cela rendra le Jouer bouton disponible dans l'application Beacon pour voir l'enregistrement de l'événement.

(Point d'entrée statique) SEP Événements en direct

Les SEP Live Events sont des événements récurrents que vous pouvez activer/désactiver lorsque votre événement recommence encore et encore comme pour un programme hebdomadaire par exemple. Cet événement peut être utilisé comme un pour tous ID de tâche en direct qui peut être transmis à plusieurs canaux dans l'application Beacon.

  • Notez que lorsque vous créez un événement récurrent, une vidéo est automatiquement créée pour celui-ci dans le module Média. Ne supprimez pas cette vidéo, car elle sera nécessaire à chaque fois que l'événement se reproduira.
  • L'événement récurrent doit être activé avant le démarrage de l'encodeur.

Suivez ces étapes pour créer un événement SEP (Point d'entrée statique) en direct dans Beacon Classic.

  1. Dans la plate-forme en direct, Créer un nouvel événement. Pour plus d'informations sur la création d'un événement, veuillez lire le Créer un nouvel événement partie de la Création et gestion d'événements en direct à l'aide du module Live document.
  2. Élargir la OPTIONS AVANCÉES , et activer le Créez-le en tant qu'événement récurrent à l'aide d'un point d'entrée statique (SEP) option. Pour plus d'informations sur le SEP, consultez le Travailler avec des événements récurrents partie de la Création et gestion d'événements en direct à l'aide du module Live document .
  3. Si vous avez déjà un événement SEP, cliquez sur le Etre prêt onglet, et vous pourrez activer/désactiver depuis la salle de contrôle en direct.
  4. Copiez votre ID de tâche en direct.
  5. Revenez sur la plateforme Beacon Classic et configurez votre événement en collant le ID de tâche de l'événement récurrent. Fournissez également un Nom Et un Date de début de l'événement.
  6. La date et l'heure de votre événement doivent être au format suivant.
  7. Cliquez dans le Événement de mise à jour bouton.
Beacon Classic CMS automatise le Statut transitions pour les événements en direct, mais dans les points d'entrée statiques, vous pouvez avoir plusieurs événements Beacon Classic associés au même SEP. S'il y a des problèmes avec l'automatisation en tant que retard et chevauchement, vous pouvez ajuster la transition CMS en modifiant manuellement le Statut. Par exemple, de Disponible à Fini dans le Statut champ.

Vous pouvez effectuer ce processus pour tous les événements en direct que vous souhaitez sur la plate-forme Beacon Classic. Tous seront diffusés à partir de la même transmission en direct, mais chacun avec la date et l'heure différentes que vous avez ajustées dans ses paramètres d'événement.
