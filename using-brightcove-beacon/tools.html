Utilisation des outils

Dans cette rubrique, vous apprendrez la fonctionnalité des options dans le Outils section de Brightcove Beacon.

Présentation des outils

Dans Brightcove Beacon le clé L'icône (étiquetée 1) représente un ensemble d'outils. Les options pour les outils sont Ingestion et Cacher (2). Ce document explore les deux options.

Ingestion

Dans le groupe Ingestion, vous disposez de trois outils :

  • Mettre à jour les vidéos Brightcove: Si vous avez ajouté des vidéos à votre catalogue Video Cloud ou modifié des vidéos existantes, ce bouton synchronise les vidéos nouvellement ajoutées ou mises à jour avec Brightcove Beacon.
  • Mettre à jour les listes de lecture Brightcove: Si vous avez ajouté des listes de lecture à votre catalogue Video Cloud ou modifié des listes de lecture existantes, ce bouton synchronise les listes de lecture nouvellement ajoutées ou mises à jour avec Brightcove Beacon.
  • XML EPG - Récupérer XML à partir du dossier: Lorsque votre EPG (Electronic Program Guide) doit être mis à jour, ce bouton récupère le fichier XML mis à jour. Un compte S3 vous sera fourni pour déposer le fichier.
Vous pouvez également travailler avec le personnel d'assistance Brightcove pour créer un calendrier pour effectuer automatiquement l'ingestion.

Mises en garde pour l'ingestion

Notez que toutes les données saisies sur une vidéo particulière dans Studio ne sont pas automatiquement ingérées dans Brightcove Beacon. Par exemple, les éléments suivants ne sont pas ingérés dans Brightcove Beacon à partir de Video Cloud :

  • Horaires de disponibilité: Les horaires définis dans le DISPONIBILITÉ section des propriétés de la vidéo ne sont pas ingérées dans Brightcove Beacon. Les dates le seront, mais pas les horaires.
  • Filtrage géographique: Informations de filtrage géographique définies dans le DISPONIBILITÉ section des propriétés de la vidéo ne sont pas ingérées dans Brightcove Beacon.
Les problèmes énumérés ci-dessus peuvent être résolus en créant un Emballer dans Brightcove Beacon, l'attribution de la vidéo à ce package lors de l'ingestion via des champs personnalisés. Voir le Création d'un forfait d'abonnement et Utiliser un forfait d'abonnement documents pour plus d'informations.

Notez également que tous les champs mentionnés dans Brightcove Beacon n'ont pas de champ personnalisé associé. Par exemple Pays de production et Sous-genre n'ont pas de champs personnalisés correspondants qui leur sont associés.

Cacher

Dans le groupe Cache, vous disposez de deux outils :

  • Purge du cache: Lorsque vous apportez des modifications dans Brightcove Beacon, cliquez sur ce bouton pour appliquer ces modifications à la configuration actuelle. Cette option récupère toutes les modifications SAUF les images dans la configuration actuelle.
  • Cache d'images: Si vous mettez à jour des images dans Brightcove Beacon, cliquez sur ce bouton pour appliquer les modifications apportées à l'image à la configuration actuelle.

Ce processus ne doit pas être effectué sans discernement, car le faire trop fréquemment en production a un effet négatif sur l'évolutivité et les performances.
