introduction

Vous voudrez sans aucun doute savoir ce que les téléspectateurs regardent dans vos applications Brightcove Beacon. Pour ce faire, vous utiliserez Google Analytics. Brightcove chargera un tableau de bord personnalisé et un rapport Google Data Studio personnalisé dans votre compte Google Analytics. Ces deux outils fournissent un excellent aperçu de la façon dont vos clients utilisent vos applications Brightcove Beacon.

En plus du tableau de bord et du rapport personnalisés, vous pouvez utiliser les outils d'analyse standard de Google Analytics pour afficher le comportement des clients. Vous pouvez utiliser les données d'analyse standard recueillies par Google, et en plus Brightcove ajoute d'autres entités à utiliser, détaillées plus loin dans ce document. Par exemple, Films les événements sont envoyés lors de la lecture d'un film, une mesure d'analyse par défaut de Google et le temps de visionnage sont calculés, une mesure ajoutée par Brightcove. Par conséquent, dans Google Analytics, vous pouvez voir les films "les plus regardés" à la fois par le nombre de fois où les téléspectateurs ont regardé le film et par le temps total de visionnage. Ces entités ajoutées par Brightcove sont largement utilisées dans le tableau de bord et le rapport personnalisés mentionnés dans le paragraphe précédent.

Affichage du tableau de bord

Pour afficher le tableau de bord Google Analytics installé par Brightcove et, plus important encore, les données affichées par le tableau de bord, procédez comme suit :

  1. Se connecter à Google Analytics.
  2. Cliquez sur la liste déroulante (surlignée en jaune dans la capture d'écran) pour voir tous vos comptes Google Analytics.
  3. Dans la liste déroulante à trois volets, sélectionnez :
    1. Votre compte Brightcove Beacon
    2. Votre application pour laquelle vous souhaitez des analyses
    3. Votre vue (vous n'aurez qu'une vue par application)
  4. Dans la navigation de gauche, cliquez sur le Personnalisation option, puis sur Tableaux de bord pour voir le tableau de bord personnalisé pour votre vue actuelle.
  5. Si le tableau de bord n'apparaît pas, vous devrez le sélectionner dans la liste :
  6. Votre tableau de bord apparaîtra sur le côté droit de l'écran. Un exemple de tableau de bord est présenté ici :

Affichage des dimensions et des statistiques

Si vous n'êtes pas familier avec les dimensions et les métriques d'analyse, voici une explication TRÈS simplifiée :
  • Les dimensions sont des attributs de vos données. Par exemple, la ville d'où proviennent les vues est une dimension. Une seule dimension peut contenir plusieurs valeurs, comme les noms de toutes les villes à partir desquelles vous avez des vues.
  • Les métriques sont des mesures quantitatives. Ainsi, le nombre de sessions d'une ville particulière serait une mesure.

Brightcove ajoute des fonctionnalités aux mesures par défaut de Google pour vos rapports d'analyse. Vous pouvez utiliser les dimensions et les métriques par défaut de Google Analytics, les blocs de construction des rapports d'analyse, ainsi que de nombreuses dimensions et une métrique ajoutées par Brightcove. Ces dimensions supplémentaires et la seule statistique sont utilisées dans le tableau de bord et le rapport Data Studio présentés dans les deux premières sections de ce document.

Notez que les dimensions et métriques ajoutées par Brightcove ne sont PAS les mêmes pour chaque type d'application Brightcove Beacon. Par exemple, les données mobiles iOS ont 16 dimensions personnalisées ajoutées, tandis que l'application du site Web a 20 dimensions personnalisées ajoutées. Pour afficher une liste des dimensions et des statistiques dans l'interface utilisateur de Google Analytics, procédez comme suit :

  1. Se connecter à Google Analytics.
  2. Cliquez sur la liste déroulante (surlignée en jaune dans la capture d'écran) pour voir tous vos comptes Google Analytics.
  3. Dans la liste déroulante à trois volets, sélectionnez :
    1. Votre compte Brightcove Beacon
    2. Votre application pour laquelle vous souhaitez des analyses
    3. Votre vue (vous n'aurez qu'une vue par application)
  4. Dans la navigation de gauche, cliquez sur le Administrateur option.
  5. Cliquez sur Définitions personnalisées.
  6. Lorsque vous cliquez Définitions personnalisées vous verrez des liens pour Dimensions personnalisées et Métriques personnalisées.
  7. Clique le Dimensions personnalisées lien pour voir les dimensions de cette application. Les dimensions changent par application.
  8. Clique le Métriques personnalisées lien pour voir la métrique unique, Temps de visionnage. Il s'agit de la seule métrique personnalisée pour les applications Brightcove Beacon.

Informations sur le public

Vous pouvez afficher des analyses sur votre audience en procédant comme suit :

  1. Choisissez votre application/vue en suivant les trois premières étapes de l'une des sections ci-dessus.
  2. Dans la navigation de gauche, cliquez d'abord sur Public alors Aperçu.

  3. Sur le côté droit de l'écran, vous voyez un résumé de votre audience à l'aide de différentes métriques comme Langue , Informations sur l'application , et Dispositifs. Vous pouvez cliquer sur une métrique spécifique pour plus de détails.

Informations sur les vues d'écran

Bien sûr, vous voudrez voir ce que font vos téléspectateurs dans vos applications. L'option pour cela est Comportement. Suivez ces étapes pour voir vues d'écran informations:

  1. Dans la navigation de gauche, cliquez sur Comportement → Contenu du site → Toutes les pages.

  2. Sur le côté droit de l'écran, vous voyez une liste des écrans les plus consultés.

Informations jouées et chronométrées

Toujours dans le Comportement option, vous pouvez voir le nombre de fois qu'un élément vidéo a été cliqué sur le bouton de lecture correspondant, ainsi que la durée pendant laquelle l'élément a été réellement regardé. Suivez ces étapes pour ces informations :

  1. Alors qu'il est encore dans le Comportement option, cliquez sur Événements → Top Événements.
  2. Dans le Catégorie d'événement liste, cliquez sur Lire des films
  3. Dans le Catégorie d'événement liste, cliquez sur Film.
  4. Dans le Action égale liste, cliquez sur Jouer.
  5. Notez que la liste indique le nombre de fois où le bouton de lecture a été cliqué pour chaque élément.
  6. Notez que vous pouvez voir des informations sur une ou plusieurs de vos dimensions personnalisées en ajoutant la colonne appropriée au tableau. Pour ce faire, cliquez sur la flèche vers le bas pour le Dimension secondaire liste déroulante, où vous pouvez voir le Dimensions personnalisées grouper.
  7. Vous pouvez maintenant ouvrir le Dimensions personnalisées groupe et sélectionnez une dimension.
  8. Voir l'exemple de tableau avec le Genre dimension personnalisée ajoutée.
  9. Notez le dans le Action d'événement liste que vous pouvez également voir Temps de visionnage , qui utilise bien sûr votre métrique personnalisée Temps de visionnage.
