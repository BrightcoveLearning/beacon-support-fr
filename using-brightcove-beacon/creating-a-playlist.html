Aperçu

Les listes de lecture sont une collection de vidéos et d'autres entités créées de l'une des deux manières suivantes pour Brightcove Beacon :

  1. Importé dans Brightcove Beacon depuis votre bibliothèque Video Cloud. Dans ce cas, les éléments de la liste de lecture seront UNIQUEMENT des vidéos.
  2. Créé dans Brightcove Beacon en utilisant le Listes de lecture languette. Lors de la création de la liste de lecture dans Brightcove Beacon, vous pouvez utiliser les vidéos importées de Video Cloud (films / épisodes), et en plus vous pouvez également ajouter des séries, des saisons, des chaînes et des événements en direct. Cette méthode est l'objet de ce document.
Notez ce qui suit :
  • Brightcove recommande de créer des listes de lecture contenant uniquement des vidéos dans Video Cloud.
  • Toutes les fonctionnalités présentées dans le Listes de lecture tab est utilisable pour le moment. Il est clairement indiqué dans ce document les éléments du formulaire que vous devez remplir.

Section Données de base

Pour créer une liste de lecture dans Brightcove Beacon, procédez comme suit :

  1. Connectez-vous à Brightcove Beacon.
  2. Dans la navigation supérieure, sélectionnez Listes de lecture.

  3. Clique le Ajouter une nouvelle liste de lecture bouton.

  4. Du Donnee de base onglet, vous allez définir le nom et le type d'affichage de la playlist, puis ajouter du contenu. Notez que la plupart des fonctionnalités de cette section ne sont pas utilisables pour le moment. Bien entendu, les étapes ci-dessous ne traiteront que des éléments utilisables.

  5. Dans le Nom champ, ajoutez un nom de système interne pour votre liste de lecture. C'est aussi le nom utilisé pour l'analyse. Notez que vous ajouterez le titre visible de la liste de lecture dans une étape ultérieure.
    Il est recommandé de désigner d'une manière ou d'une autre les listes de lecture créées dans Brightcove Beacon vs Video Cloud. Par exemple, placez simplement un astérisque (* ) devant le nom de la liste de lecture lorsque la liste de lecture est créée dans Brightcove Beacon. Bien sûr, il n'est pas nécessaire que ce soit un astérisque, mais choisissez une notation pour que, lorsque vient le temps de mettre à jour une liste de lecture, vous sachiez aller à Brightcove Beacon Listes de lecture onglet ou Video Cloud Studio pour apporter des modifications.
  6. Sélectionnez une mise en page parmi Afficher toute la disposition de la liste de lecture. Il s'agit de la mise en page utilisée lorsque les utilisateurs cliquent sur le Voir tout bouton dans les applications. Comme meilleure pratique, choisissez une grille quelconque.
  7. Pour commencer le processus d'ajout de contenu à votre liste de lecture, cliquez sur le bouton Les atouts languette. Vous voyez que vous avez le choix entre six types d'actifs.
    Vous êtes maintenant au point où vous voyez des options de liste de lecture, celles-ci étant Films , Séries , Saisons , Événements et Canaux , que vous ne pouvez pas choisir dans Video Cloud Studio. Ces types ne sont PAS accessibles dans Video Cloud Studio, vous devez donc créer la liste de lecture dans Brightcove Beacon.
  8. Pour sélectionner l'actif souhaité, cliquez d'abord sur le type d'actif, puis cliquez sur le à côté de l'actif réel.

Données textuelles

Les Données textuelles La section vous permet d'ajouter du texte affichable pour votre liste de lecture.

  1. Sélectionnez le Données textuelles languette.

  2. Ajoutez des valeurs pour les champs suivants :

    Champ Obligatoire Remarques
    Titre Oui Le titre affiché pour votre liste de lecture.
    La description N'est pas applicable Ce champ n'est pas utilisé actuellement.
    Sous-titre N'est pas applicable Ce champ n'est pas utilisé actuellement.

Données non textuelles

Les Données non textuelles l'onglet n'est pas fonctionnel pour le moment.

Mots clés

Les Mots clés l'onglet n'est pas fonctionnel pour le moment.

Droits et planification

Les Droits et planification La section est l'endroit où vous définissez le statut de publication et la disponibilité.

  1. Sélectionnez le Droits et planification languette.

  2. Assurez-vous que le Statut est réglé sur Publié pour vos listes de lecture.
  3. Pour le Détails de disponibilité , utilisez ce qui suit comme guide pour compléter les informations :
    Champ Obligatoire Remarques
    Type de droits Oui Sélectionner Libérer.
    Heure de début Oui Planifiez une heure de début pour votre liste de lecture.
    Heure de fin Oui Planifiez une heure de fin pour votre liste de lecture.
    Flux max. N / A Non fonctionnel pour le moment.
    Appareils à publier Oui Ajoutez les appareils sur lesquels vous souhaitez publier. Sélectionnez le Sélectionner/désélectionner tous les appareils pour voir la liste des appareils disponibles.
    Emplacements autorisés Oui

    Sélectionnez les emplacements où vous souhaitez diffuser votre contenu, si aucune restriction entrez Monde.

    Si vous ne spécifiez pas au moins 1 emplacement autorisé, ce message peut s'afficher dans votre application :
    Emplacements refusés Non Sélectionnez les emplacements où vous ne souhaitez PAS diffuser votre contenu.

    Pour ajouter ou modifier des emplacements géographiques, accédez à Paramètres puis Emplacements .
  4. Assurez-vous de cliquer sur le Créer une nouvelle liste de lecture bouton pour enregistrer votre travail.
