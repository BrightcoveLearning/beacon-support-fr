    Création d'une sélection

    Dans cette rubrique, vous apprendrez comment créer une playlist à l'aide de l'onglet Listes de lecture de Brightcove Beacon.

    Présentation

    Les playlists sont une collection de vidéos et d'autres entités créées de l'une des deux manières suivantes pour Brightcove Beacon :

    1. Importé dans Brightcove Beacon à partir de votre bibliothèque Video Cloud. Dans ce cas, les éléments de la liste de lecture ne seront que des vidéos.
    2. Créé dans Brightcove Beacon à l'aide de l'onglet Listes de lecture . Lors de la création de la playlist dans Brightcove Beacon, vous pouvez utiliser les vidéos importées depuis Video Cloud (films ou épisodes), et vous pouvez également ajouter des séries, des saisons, des chaînes et des événements en direct. Cette méthode est au centre de ce document.
    Toutes les fonctionnalités affichées dans l'onglet Listes de lecture ne sont pas utilisables pour le moment. Le présent document indique clairement quels éléments de formulaire vous devez remplir.

    Section Données de base

    Pour créer une playlist dans Brightcove Beacon, procédez comme suit :

    1. Connectez-vous à Brightcove Beacon.
    2. Dans la navigation supérieure, sélectionnez Listes de lecture.

    3. Cliquez sur le bouton Ajouter une nouvelle liste de lecture .

    4. Dans l'onglet Données de base , vous définissez le nom et le type d'affichage de la liste de lecture, puis ajoutez du contenu. Notez que la plupart des fonctionnalités de cette section ne sont pas utilisables pour le moment. Bien sûr, les étapes ci-dessous ne traitent que des articles utilisables.

    5. Dans le champ Nom , ajoutez un nom système interne pour votre liste de lecture. Il s'agit également du nom utilisé pour l'analyse. Notez que vous allez ajouter le titre visible de la liste de lecture dans une étape ultérieure.
      Il est recommandé de désigner d'une manière ou d'une autre les listes de lecture créées dans Brightcove Beacon contre Video Cloud. Par exemple, placez simplement un astérisque (*) devant le nom de la playlist lorsque la playlist est créée dans Brightcove Beacon. Bien sûr, il ne faut pas nécessairement être un astérisque, mais choisissez une notation de sorte que lorsque vient le temps de mettre à jour une playlist, vous savez aller à l'onglet Listes de lecture Brightcove Beacon ou Video Cloud Studio pour apporter des modifications.
    6. Sélectionnez un modèle dans Afficher toute la mise en page de liste de lecture. Il s'agit de la mise en page utilisée lorsque les utilisateurs cliquent sur le bouton Afficher tout dans les applications. En tant que meilleure pratique, choisissez une grille d'une sorte.
    7. Pour commencer le processus d'ajout de contenu à votre liste de lecture, cliquez sur l'onglet Ressources . Vous voyez que vous avez six types d'actifs à choisir.
      Vous êtes maintenant au point où vous voyez les options de playlist, les films, les séries, les saisons, les événements et les chaînes, que vous ne pouvez pas choisir dans Video Cloud Studio. Ces types ne sont PAS accessibles dans Video Cloud Studio. Vous devez donc créer la liste de lecture dans Brightcove Beacon.
    8. Pour sélectionner l'actif souhaité, cliquez d'abord sur le type de ressource, puis cliquez sur le côté de l'actif réel.

    Données textuelles

    La section Données textuelles vous permet d'ajouter du texte affichable à votre liste de lecture.

    1. Sélectionnez l'onglet Données textuelles .

    2. Ajoutez des valeurs pour les champs suivants :

      Champ Obligatoire Remarques
      Titre Oui Titre affiché pour votre liste de lecture.
      Description Sans objet Ce champ n'est pas utilisé actuellement.
      Sous-titre Sans objet Ce champ n'est pas utilisé actuellement.

    Données non textuelles

    L'onglet Données non textuelles n'est pas fonctionnel pour le moment.

    Balises

    L'onglet Balises n'est pas fonctionnel pour le moment.

    Droits et planification

    La section Droits et planification est l'endroit où vous définissez l'état et la disponibilité de la publication.

    1. Sélectionnez l'onglet Droits et planification .

    2. Assurez-vous que l' état est défini sur Publié pour vos sélections.
    3. Pour les détails de disponibilité, utilisez ce qui suit comme guide pour compléter les informations :
      Champ Obligatoire Remarques
      Type de droits Oui Sélectionnez Gratuit.
      Heure de début Oui Planifiez une heure de démarrage pour votre liste de lecture.
      Heure de fin Oui Planifiez une heure de fin pour votre playlist.
      Nombre maximal de flux S. O. Pas fonctionnel pour le moment.
      Périphériques à publier Oui Ajoutez les appareils sur lesquels vous souhaitez publier. Sélectionnez Sélectionner/Désélectionner tous les périphériques pour afficher la liste des périphériques disponibles.
      Emplacements autorisés Oui

      Sélectionnez les emplacements où vous souhaitez diffuser votre contenu, si aucune restriction n'est entrée dans World.

      Si vous ne spécifiez pas au moins 1 emplacement autorisé, vous pouvez voir ce message depuis votre application :
      Emplacements refusés Non Sélectionnez les emplacements où vous NE souhaitez PAS diffuser votre contenu.

      Pour ajouter ou modifier des emplacements géographiques, accédez à Paramètres , puis Emplacements.
    4. Veillez à cliquer sur le bouton Créer une nouvelle liste de lecture pour enregistrer votre travail.
