Création d'une nouvelle page

Dans cette rubrique, vous apprendrez à créer une nouvelle page Brightcove Beacon.

Aperçu

La page Brightcove Beacon est l'endroit où les utilisateurs sélectionnent et regardent votre contenu vidéo. Les carrousels affichent un groupe de vidéos ou de listes de lecture que vous avez définies dans Video Cloud Studio. À l'aide de la console d'administration Brightcove Beacon, vous pouvez créer une nouvelle page avec un carrousel de contenu.

Création d'une page

Pour créer une nouvelle page, procédez comme suit :

  1. Connectez-vous à Brightcove Beacon.

  2. Dans la navigation supérieure, sélectionnez Pages.

  3. Sélectionnez le Ajouter une nouvelle page bouton pour créer une nouvelle page.

  4. Pour le Donnee de base onglet, saisissez les valeurs de champ suivantes :

    • Titre du système - Nom de la page du système interne. Cela ne sera pas affiché aux utilisateurs, mais est utilisé pour générer une URL conviviale. Vous entrerez un titre visible dans une étape ultérieure.
    • Commander - Ordre des pages sur votre page d'accueil
    • Genre - Valeurs:
      • Maître - Page d'accueil
      • Enfant - Page subordonnée
    • Type de navigation - Sélectionnez parmi une liste de modèles de pages
    • Type de présentation - Valeurs:
      • icontexte - Afficher une icône avec du texte à l'intérieur
      • icône - Afficher une icône
      • texte - Afficher le texte
    • Mise en page de l'écran - Valeurs : (tous les types de navigation n'auront pas de disposition d'écran)
      • Disposition de l'écran intitulée - Mise en page avec titres
      • Disposition de l'écran sans titre - Mise en page sans titres
    • C'est chez moi ? - Sélectionnez s'il s'agit de la page d'accueil
  5. Ensuite, nous allons ajouter un bloc à la page. Les blocs sont des conteneurs pour vos listes de lecture. Certains types de navigation n'ont pas de blocs, tels que les types de connexion et de déconnexion. Nous avons sélectionné un Type de navigation avec une valeur de filtrer.

  6. Sélectionnez le Ajouter un bloc bouton. Lorsque vous sélectionnez le bouton, les types de blocs apparaîtront sur le côté droit.

  7. Choisissez le type de bloc que vous souhaitez. Dans ce cas, nous choisissons le liste-bloc. Sélectionnez le Ajouter un bloc bouton associé au bloc que vous souhaitez.

  8. Sélectionnez le Ajouter une mise en page bouton. Lorsque vous sélectionnez le bouton, les types de mise en page apparaîtront sur le côté droit.

  9. Choisissez le type de mise en page que vous souhaitez, qu'il s'agisse d'une présentation, d'un carrousel ou d'une grille. Dans cet exemple, nous sélectionnons le Étiquettes de paysage de carrousel dans disposition.

  10. Sélectionnez le Ajouter une liste de lecture bouton. Lorsque vous sélectionnez le bouton, les types de listes de lecture apparaîtront sur le côté droit.

  11. Choisissez une liste de lecture. Dans cet exemple, nous sélectionnons le Ajouter une liste de lecture bouton associé au Humour playlist.

    Vous créez des listes de lecture à l'aide de Video Cloud Studio et elles apparaîtront ici une fois que vous aurez synchronisé avec votre compte Video Cloud.

  12. Nous pouvons voir que le Humour playlist a été ajoutée à notre carrousel. Vous pouvez faire glisser et déposer pour réorganiser les articles.

  13. Pour le Données textuelles , Données non textuelles , Mots clés et Droits et planification onglets, voir le Paramètres de configuration communs document.

  14. Sélectionnez le Créer une nouvelle page bouton pour créer votre nouvelle page. Vous êtes prêt. Vous devriez voir le nouveau carrousel sur votre page Brightcove Beacon.
