Comprendre les pages non configurables

Il y a certaines pages que vous devez, devriez ou pouvez inclure que vous ne pouvez pas personnaliser dans Brightcove Beacon. Vous choisissez d'inclure la page ou non. Vous avez peut-être choisi une mise en page spécifique plutôt que d'autres pour ces pages lorsque vous avez travaillé sur le document Création de votre conception d'application - Options de mise en page . Ajouter une page non configurable à votre application

L'ajout d'une page non configurable à votre application est assez simple car il y a peu de configuration à faire. À un niveau élevé, le processus est :

  1. Dans Brightcove Beacon, créez une nouvelle page.
  2. Ajoutez des données de base, en choisissant la page que vous souhaitez dans le Type de navigation menu déroulant. Vous ne serez PAS autorisé à ajouter des blocs et/ou des mises en page.
  3. Fournissez le reste des données pour la page, comme indiqué dans le Paramètres de configuration de page communs document.

Par exemple, si vous souhaitez ajouter une page de recherche à votre application, vous devez créer une nouvelle page, dans le Donnee de base forme que vous choisiriez chercher- pour le Type de navigation , puis complétez le reste des données de base. Ce formulaire est montré ici :
