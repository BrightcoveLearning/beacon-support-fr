    Mise en œuvre de VOD SSAI

    Dans cette rubrique, vous apprendrez comment créer et gérer l'insertion d'annonces côté serveur (SSAI) avec Video On Demand (VOD) pour Brightcove Beacon.

    Présentation

    Les annonces côté serveur sont intégrées dans le flux vidéo de sorte qu'elles ne peuvent pas être bloquées par les bloqueurs d'annonces dans le navigateur. Il fonctionne à la fois avec du contenu protégé DRM et non protégé. Brightcove Beacon prend en charge la VOD avec SSAI.

    Voici les étapes à suivre pour ajouter des annonces côté serveur :

    1. Dans Video Cloud Studio, créez une configuration publicitaire
    2. Définir les valeurs de champ personnalisées pour votre vidéo
    3. Ajouter votre vidéo à une liste de lecture
    4. Dans Brightcove Beacon, importez vos vidéos et listes de lecture

    Meilleures pratiques

    Brightcove recommande de suivre ces meilleures pratiques lors de l'ajout d'annonces côté serveur à du contenu VOD :

    • Créez votre configuration publicitaire dans Video Cloud
    • Attribuer la configuration de l'annonce aux vidéos à l'aide des champs personnalisés de Video Cloud
    • Même si vous pouvez attribuer des configurations publicitaires à des vidéos dans Brightcove Beacon, il est recommandé de le faire dans Video Cloud
    • Utilisez uniquement la section Publicité de Brightcove Beacon pour les événements et les chaînes en direct

    Création d'une configuration d'annonce

    Vous allez créer ici une configuration d'annonce SSAI pour définir les annonces affichées dans votre actif VOD Brightcove Beacon.

    1. Connectez-vous à Video Cloud Studio.
    2. Dans l'en-tête de navigation, cliquez sur ADMIN et sélectionnez Paramètres publicitaires côté serveur. Une liste de toutes les configurations d'annonces s'affiche.
    3. Pour créer une nouvelle configuration d'annonce, cliquez sur le bouton Créer une configuration publicitaire .
    4. Dans la liste déroulante, sélectionnez VOD comme type.
    5. Vérifiez que l'écran Paramètres des annonces côté serveur s'voit bien.
    6. Donnez un nomà la configuration de votre annonce.
    7. Sélectionnez la réponse du serveur publicitaire:
      Réponse au serveur de publicité Détails
      VASTE 3.0 Modèle de diffusion d'annonces vidéo numériques
      VMAP DFP Playlist d'annonces multiples vidéo
      DFP (Règles publicitaires) Format propriétaire pour DFP/Google pour les éditeurs
      XML intelligent Format propriétaire pour FreeWheel

      Veuillez vérifier auprès de votre gestionnaire des opérations publicitaires pour connaître la valeur correcte à sélectionner.
    8. Entrez la balise publicitaire (URL du serveur).
    9. Cliquez sur Save.
    10. Copiez l'ID de votre configuration publicitaire, car vous en aurez besoin ultérieurement pour configurer votre ressource VOD.

    Définition de valeurs de champ personnalisées

    Dans cette section, vous apprendrez comment configurer votre ressource VOD avec les paramètres nécessaires pour afficher SSAI dans Brightcove Beacon.

    1. Dans Studio, accédez au module Media .
    2. Cliquez sur la vidéo à laquelle vous souhaitez ajouter SSAI.
    3. Faites défiler la page jusqu'à la section Champs personnalisés .
    4. Cliquez sur le bouton Modifier .
    5. Faites défiler la page vers le bas et localisez le champ Beacon.rights.0.adConfiguration , puis collez l'ID de votre configuration publicitaire précédemment créée.
    6. Continuez à faire défiler vers le bas et dans le champ beacon.rights.0.type écrire AVOD pour catégoriser cette VOD comme et la publicité VOD.
    7. Allez au bas de la section Champs personnalisés et cliquez sur le bouton Enregistrer .
      Par défaut, vous pouvez ajouter jusqu'à 3 configurations d'ID SSAI et de taper dans la section Champs personnalisés .
    8. Dans le menu supérieur, cliquez sur Retour aux vidéos.

    Ajout de vidéos à une liste de lecture

    Dans cette section, vous allez ajouter votre ressource VOD à une liste de lecture.

    1. Dans Studio, revenez au module Media .
    2. Cochez la case correspondant à votre vidéo.
    3. Cliquez sur le bouton Plus dans le menu principal.
    4. Dans le menu déroulant, sélectionnez Ajouter à la liste de lecture.
    5. Sélectionnez la liste de lecture à laquelle vous souhaitez ajouter votre vidéo.
    6. Cliquez sur le bouton Add (Ajouter).
    7. Une fois votre ressource VOD ajoutée à la liste de lecture, accédez à la plateforme Brightcove Beacon.

    Importation de vidéos et de listes de lecture

    Dans Brightcove Beacon, vous pouvez ingérer et importer des vidéos et des listes de lecture à partir de Video Cloud.

    1. Connectez-vous à votre compte Brightcove Beacon.
    2. Dans le menu supérieur, sélectionnez Outils, qui est l'icône de clé.
    3. Dans la navigation de gauche, sélectionnez Ingestion.
    4. Cliquez sur le bouton Mettre à jour les vidéos Brightcove et/ou sur le bouton Mettre à jour les listes de lecture Brightcove .
    5. Vos vidéos et listes de lecture de Video Cloud doivent désormais être dans Brightcove Beacon. Dans le menu supérieur, cliquez sur l'onglet Sélections .
    6. Sélectionnez votre playlist pour voir les vidéos qu'elle contient.
    7. Vérifiez que vous pouvez voir toutes les ressources VOD associées à cette liste de lecture dans la table des matières de gauche.
    8. Dans le menu supérieur Brightcove Beacon, cliquez dans l'onglet Films .
    9. Cliquez sur votre actif VOD.
    10. Cliquez sur l'onglet Droits et planification .
    11. Dans le menu Type de droits , vous devriez voir Publicité - AVOD sélectionné.
    12. Dans le menu Configuration de l'annonce , vous devriez voir votre identifiant de configuration d'annonce précédemment créé.
    13. Pour voir immédiatement vos vidéos dans votre application, vous devez effacer le cach. Dans le menu supérieur, cliquez sur le bouton Outils (clé), puis effacez le cache.
    14. Dans la navigation de gauche, sélectionnez Cache.
    15. Cliquez sur le bouton Purge du cache .
    16. Ouvrez votre application Brightcove Beacon et cliquez dans l'onglet Page sur laquelle votre playlist a été ajoutée. (Tous les noms d'événements et les images sur l'image ne sont qu'à titre d'exemple).
      Pour visualiser votre playlist dans l'application Brightcove Beacon, vous devez d'abord ajouter votre playlist à une page Brightcove Beacon. Pour en savoir plus sur la façon d'ajouter une liste de lecture à une Page, veuillez consulter la documentationsuivante.
    17. Cliquez sur le film que vous avez configuré avec des annonces côté serveur.
    18. Cliquez sur le bouton Lecture et vous devriez voir votre vidéo lire avec des annonces.
