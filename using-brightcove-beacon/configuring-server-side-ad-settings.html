Les configurations d'annonces définissent divers aspects de la lecture d'annonces, notamment les appels d'annonces, la synchronisation, les balises et d'autres options de configuration d'annonces. Les paramètres de publicité côté serveur définissent les publicités qui peuvent être demandées pendant les vidéos de vidéo à la demande (VOD) et pendant les événements en direct. Les publicités côté serveur tirent parti de la technologie d'insertion de publicité côté serveur (SSAI) de Brightcove et offrent ces fonctionnalités principales :

  • Étant donné que les publicités sont cousues dans le flux vidéo à la demande ou en direct côté serveur, elles ne sont pas affectées par les bloqueurs de publicités
  • Pour les événements en direct, les publicités sont insérées aux points de repère envoyés par votre encodeur ou vous pouvez créer un point de repère instantané à l'aide du Salle de contrôle du module en direct
  • Vous pouvez configurer des ardoises de remplissage pour remplir tout temps publicitaire inutilisé

Pour en savoir plus sur la publicité côté serveur, consultez Implémentation des publicités côté serveur avec Brightcove Player.

Noter:
  • Avant de pouvoir utiliser SSAI, votre compte Video Cloud doit être configuré pour la diffusion dynamique et activé pour SSAI. Contactez votre gestionnaire de compte pour plus de détails.
  • VMAP pour Brightcove Live SSAI n'est actuellement pas pris en charge.
  • Bien qu'il n'y ait pas de fonctionnalité de prélecture pour les annonces SSAI, nous déléguons la logique de position des annonces au fournisseur d'annonces. La livraison dynamique effectue tous les appels supplémentaires à l'avance. Si nous recevons une annonce du fournisseur d'annonces et qu'elle est transcodée dans notre système, nous l'insérons dans le flux.

La page Paramètres des publicités côté serveur est utilisée pour gérer les configurations des publicités en direct et VOD. Notez que pour accéder à la page Paramètres des annonces côté serveur, vous devez être un utilisateur avec le rôle Administrateur.

Pour accéder à la page Paramètres des annonces côté serveur :

  1. Connectez-vous à Studio.
  2. Dans l'en-tête de navigation, cliquez sur ADMINISTRER puis Paramètres d'annonce côté serveur.Une liste de toutes les configurations d'annonces sera affichée.

Pour créer une nouvelle configuration d'annonce, cliquez sur Créer une configuration d'annonce puis cliquez sur le type d'annonce spécifique à créer :

  • Vidéo à la demande (VOD)
  • Habitent

Création d'une configuration publicitaire de vidéo à la demande

Noter: Les configurations d'annonces côté serveur peuvent également être gérées à l'aide de l'API d'insertion d'annonces côté serveur. Pour plus d'informations, consultez le API de configuration d'annonces Video Cloud SSAI sujet.

Pour ajouter une configuration publicitaire de vidéo à la demande (VOD), procédez comme suit.

  1. Sur la page Paramètres des annonces côté serveur, cliquez sur Créer une configuration d'annonce > VOD.
  2. Entrez un Nom.
  3. Sélectionnez le Réponse du serveur publicitaire:
    • Règles relatives aux annonces DFP
    • DFP VMAP
    • XML intelligent
    • VASTE 3.0
  4. Entrer le Balise publicitaire.
  5. Vérifier Désactiver les balises côté serveur pour désactiver les balises côté serveur. Lorsqu'il est désactivé, SSAI ne déclenchera aucune balise côté serveur et inclura toutes les balises dans la sortie VMAP.
  6. Cliquez sur sauvegarder.

Pour modifier une configuration d'annonce, cliquez sur le nom de l'annonce, apportez les modifications, puis cliquez sur sauvegarder. Pour supprimer une configuration d'annonce, cochez la case en regard de l'annonce, puis cliquez sur Effacer. Pour dupliquer une configuration d'annonce, cochez la case en regard de l'annonce, puis cliquez sur Dupliquer.

Une fois les configurations d'annonces VOD créées, ouvrez le module Lecteurs et cliquez sur un lecteur pour afficher les propriétés. Quand le Activer côté serveur (SSAI) la propriété publicitaire est activée, une configuration publicitaire peut être sélectionnée. Pour plus d'informations sur la configuration de la publicité pour les lecteurs, consultez Configurer la publicité des joueurs à l'aide du module des joueurs (lien externe).

Création d'une configuration d'annonce en direct

Noter: Les configurations d'annonces en direct côté serveur peuvent également être gérées à l'aide de l'API Live. Pour plus d'informations, voir API en direct de Brightcove : Insertion d'annonces côté serveur (SSAI) .

Pour ajouter une configuration d'annonce en direct, procédez comme suit.

  1. Sur la page Paramètres des annonces côté serveur, cliquez sur Créer une configuration d'annonce > En direct.
  2. Entrez un Nom.
  3. Sélectionnez le Réponse du serveur publicitaire:
    • VASTE
    • DFP (Règles relatives aux annonces)
    • XML intelligent
  4. Entrer le Balise publicitaire.
  5. Cliquez sur sauvegarder.

Pour modifier une configuration d'annonce, cliquez sur le nom de l'annonce, apportez les modifications, puis cliquez sur sauvegarder. Pour supprimer une configuration d'annonce, cochez la case en regard de l'annonce, puis cliquez sur Effacer. Pour dupliquer une configuration d'annonce, cochez la case en regard de l'annonce, puis cliquez sur Dupliquer.

Une fois les configurations d'annonces en direct créées, lorsqu'un événement est créé avec le Activer l'insertion d'annonces côté serveur (SSAI) propriété activée, une configuration publicitaire peut être sélectionnée dans la salle de contrôle. Pour plus d'informations sur la création d'événements en direct, consultez Création et gestion d'événements en direct à l'aide du module Live.
