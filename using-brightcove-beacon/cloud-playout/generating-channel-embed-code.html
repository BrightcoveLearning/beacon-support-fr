Générer le code d'intégration pour une chaîne Cloud Playout

Pour générer le code d'intégration d'une chaîne Cloud Playout, vous utiliserez le module Live. Pour plus de détails sur l'utilisation du module Live, voir Création et gestion d'événements en direct à l'aide du module Live.

Deux types de code d'intégration sont disponibles :

  • Standard - code d'intégration iframe. Recommandé pour la plupart des cas. Ce type place le lecteur dans une balise <iframe>.
  • Avancée - Code d'intégration dans la page. Permet au lecteur d'exister directement dans la page HTML, pas dans un iframe. Cela offre aux développeurs les avantages de la facilité d'accès au lecteur et aux propriétés et événements associés.

Vous avez également la possibilité d'obtenir l'URL du lecteur publié. Ceci est utile pour afficher l'événement dans un navigateur.

Pour générer le code d'intégration d'un événement en direct, procédez comme suit :

  1. Ouvrez le module Live.
  2. Localisez l'événement en direct auquel la chaîne diffusera. En règle générale, l'événement se déroulera dans le Etre prêt onglet et avoir un nom de Canaliser à moins que vous n'ayez sélectionné un événement spécifique à diffuser lors de la création de la chaîne.
  3. Cliquez sur le nom de l'événement pour ouvrir le Salle de contrôle page.
  4. Cliquez sur Publier et intégrer > Configuration du lecteur dans la navigation de gauche.
    Noter: Si le canal est configuré pour activer les publicités côté serveur (SSAI), Brightcove Player version 6.18.3 et supérieure doit être utilisé. Seuls les joueurs de la version 6.18.3 et supérieure apparaîtront dans la liste déroulante.
  5. Clique le URL du lecteur pour prévisualiser l'événement dans un navigateur ou copier le code d'intégration approprié à utiliser.

Si le programme a été créé avec Insertion d'annonces option définie sur Au , les options de publication incluront un Sélectionnez la configuration de l'annonce liste déroulante où la configuration publicitaire à utiliser peut être sélectionnée.
