Formation à la demande : Affichage des données analytiques

Regarder la formation Affichage des données analytiques vous donne les compétences nécessaires pour utiliser Google Analytics pour voir ce que font les utilisateurs dans vos applications Brightcove Beacon. Vous apprenez à afficher les tableaux de bord/rapports fournis par Brightcove qui sont ajoutés à votre compte Google Analytics. De plus, vous obtiendrez une introduction sur la façon d'utiliser les dimensions et une métrique pour créer vos propres tableaux de bord.

Il existe un ordre logique dans ces vidéos, il est donc préférable de regarder les vidéos dans l'ordre de leur apparition ci-dessous.
