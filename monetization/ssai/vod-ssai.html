Aperçu

Les publicités côté serveur sont intégrées dans le flux vidéo afin qu'elles ne puissent pas être bloquées par les bloqueurs de publicités dans le navigateur. Il fonctionne avec le contenu protégé par DRM et non protégé. Brightcove Beacon prend en charge la VOD avec SSAI.

Voici les étapes à suivre pour ajouter des annonces côté serveur :

  1. Dans Video Cloud Studio, créer une configuration d'annonce
  2. Définir les valeurs des champs personnalisés pour ta vidéo
  3. Ajoutez votre vidéo à une liste de lecture
  4. Dans la balise de Brightcove, importer vos vidéos et playlists

Les meilleures pratiques

Brightcove recommande de suivre ces bonnes pratiques lors de l'ajout d'annonces côté serveur au contenu VOD :

  • Créez votre configuration publicitaire dans Video Cloud
  • Attribuez la configuration publicitaire aux vidéos à l'aide des champs personnalisés dans Video Cloud
  • Même si vous pouvez attribuer des configurations publicitaires aux vidéos dans Brightcove Beacon, il est recommandé de le faire dans Video Cloud
  • Utilisez uniquement le Publicité section dans Brightcove Beacon pour les événements en direct et les chaînes

Créer une configuration d'annonce

Ici, vous allez créer une configuration d'annonce SSAI pour définir les annonces affichées dans votre actif Brightcove Beacon VOD.

  1. Connectez-vous à Video Cloud Studio.
  2. Dans l'en-tête de navigation, cliquez sur ADMINISTRER et sélectionnez Paramètres d'annonce côté serveur. Une liste de toutes les configurations d'annonces sera affichée.
  3. Pour créer une nouvelle configuration d'annonce, cliquez sur le bouton Créer une configuration d'annonce bouton.
  4. Dans la liste déroulante, sélectionnez VOD comme le genre.
  5. Vérifiez que vous voyez le Paramètres d'annonce côté serveur former.
  6. Donnez à votre configuration d'annonce un Nom.
  7. Sélectionnez le Réponse du serveur publicitaire:
    Réponse du serveur publicitaire Des détails
    VASTE 3.0 Modèle de diffusion d'annonces vidéo numériques
    DFP VMAP Liste de lecture de plusieurs annonces vidéo
    DFP (Règles relatives aux annonces) Format propriétaire pour DFP/Google for Publishers
    XML intelligent Format propriétaire pour FreeWheel

    Veuillez vérifier auprès de votre responsable des opérations publicitaires la bonne valeur à sélectionner.
  8. Entrer le Balise publicitaire (URL du serveur).
  9. Cliquez sur sauvegarder.
  10. Copiez l'ID de votre configuration d'annonce, car vous en aurez besoin plus tard pour configurer votre élément VOD.

Définition des valeurs de champ personnalisées

Dans cette section, vous apprendrez à configurer votre ressource VOD avec les paramètres nécessaires pour afficher SSAI dans Brightcove Beacon.

  1. Dans Studio, accédez au Médias module.
  2. Cliquez sur la vidéo à laquelle vous souhaitez ajouter SSAI.
  3. Faites défiler jusqu'au Les champs personnalisés section.
  4. Clique sur le Éditer bouton.
  5. Faites défiler vers le bas et localisez le beacon.rights.0.adConfiguration champ, puis collez l'ID de votre configuration d'annonce précédemment créée.
  6. Continuez à faire défiler vers le bas et dans le beacon.rights.0.type écriture de champ AVOD pour catégoriser cette VOD en tant que VOD publicitaire.
  7. Allez au bas de la Les champs personnalisés rubrique et cliquez sur le sauvegarder bouton.
    Par défaut, vous pouvez ajouter jusqu'à 3 SSAI ID et saisir des configurations dans le Les champs personnalisés section.
  8. Dans le menu du haut, cliquez sur Retour aux vidéos.

Ajouter des vidéos à une liste de lecture

Dans cette section, vous allez ajouter votre actif VOD à une liste de lecture.

  1. Dans Studio, retournez au Médias module.
  2. Cochez la case de votre vidéo.
  3. Clique le Suite bouton du menu principal.
  4. Dans le menu déroulant, sélectionnez Ajouter à la playlist.
  5. Sélectionnez la liste de lecture à laquelle vous souhaitez ajouter votre vidéo.
  6. Clique le Ajouter bouton.
  7. Une fois votre actif VOD ajouté à la liste de lecture, accédez à la plate-forme Brightcove Beacon.

Importation de vidéos et de listes de lecture

Dans Brightcove Beacon, vous pouvez ingérer/importer des vidéos et des listes de lecture depuis Video Cloud.

  1. Connectez-vous à votre compte Brightcove Beacon.
  2. Dans le menu du haut, sélectionnez Outils , qui est l'icône de la clé.
  3. Dans la navigation de gauche, sélectionnez Ingestion.
  4. Clique le Mettre à jour les vidéos Brightcove bouton et/ou le Mettre à jour les listes de lecture Brightcove bouton.
  5. Vos vidéos et listes de lecture de Video Cloud devraient maintenant être dans Brightcove Beacon. Dans le menu du haut, cliquez sur le Listes de lecture languette.
  6. Sélectionnez votre playlist pour voir les vidéos qu'elle contient.
  7. Vérifiez que vous pouvez voir toutes les ressources VOD associées à cette liste de lecture dans la table des matières de gauche.
  8. Dans le menu supérieur de Brightcove Beacon, cliquez dans le Films languette.
  9. Cliquez sur votre actif VOD.
  10. Clique le Droits et planification languette.
  11. Dans le Type de droits menu, vous devriez voir Publicité - AVOD choisi.
  12. Dans le Configuration de la publicité menu, vous devriez voir votre identifiant de configuration d'annonce précédemment créé.
  13. Pour voir vos vidéos dans votre application immédiatement, vous devez vider le cache. Dans le menu du haut, cliquez sur le Outils (clé à molette), puis videz le cache.
  14. Dans la navigation de gauche, sélectionnez Cacher.
  15. Sélectionnez le Purge du cache bouton.
  16. Ouvrez votre application Brightcove Beacon et cliquez dans le Page onglet sur lequel votre playlist a été ajoutée. (Tous les noms d'événements et les images sur l'image sont à titre d'exemple uniquement).
    Pour visualiser votre liste de lecture dans l'application Brightcove Beacon, vous devez d'abord avoir ajouté votre liste de lecture à une page Brightcove Beacon. Pour en savoir plus sur la façon d'ajouter une liste de lecture à une page, veuillez visiter ce qui suit Documentation .
  17. Cliquez sur le film que vous avez configuré avec des publicités côté serveur.
  18. Clique le Jouer et vous devriez voir votre vidéo jouer avec des publicités.
