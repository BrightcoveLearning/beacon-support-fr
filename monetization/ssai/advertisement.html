Aperçu

Pour créer une configuration de publicité, vous devrez procéder comme suit :

  • Obtenez une URL de serveur de tags d'emplacement publicitaire auprès de votre fournisseur de services publicitaires. Cela sera utilisé dans Video Cloud Studio pour créer votre configuration publicitaire SSAI.
  • Dans Video Cloud Studio, créez votre configuration publicitaire SSAI. À la fin de la tâche, vous copierez l'ID du serveur publicitaire pour créer votre configuration de publicité dans Brightcove Beacon Classic.
  • Dans Brightcove Beacon Classic, créez votre configuration publicitaire, que vous pourrez ensuite appliquer à votre contenu vidéo.

Créer un ID de configuration d'annonce SSAI

Suivez ces étapes pour créer une configuration d'annonce SSAI :

  1. Dans Video Cloud Studio, accédez à Admin → Paramètres d'annonce côté serveur.
  2. Du Créer une configuration d'annonce liste déroulante, sélectionnez VOD
  3. Ajoutez un nom pour la configuration de l'annonce.
  4. Sélectionnez votre réponse publicitaire - VMAP ou VAST est recommandé. Si vous utilisez VAST, vous devrez ajouter des points de repère aux vidéos pour spécifier où les coupures publicitaires doivent apparaître.
  5. Collez votre tag d'emplacement publicitaire.
  6. Cliquez sur sauvegarder.
  7. Recherchez votre nouvelle configuration d'annonce dans le tableau et copiez votre ID de configuration d'annonce. Vous aurez besoin de cette valeur pour la configuration de la publicité Beacon.

Créer une configuration de publicité

Pour créer une configuration de publicité, procédez comme suit :

  1. Dans Brightcove Beacon Classic, dans la barre de navigation supérieure, sélectionnez Publicité .
  2. Cliquez sur Ajouter une nouvelle configuration.
  3. Nommez la configuration de l'annonce.
  4. Pour le Fournisseur sélectionner Brightcove SSAI
  5. Coller dans le ID de configuration d'annonce SSAI copié dans les étapes ci-dessus dans ID de configuration d'annonce VOD.
  6. Cliquez sur Créer une nouvelle publicité.
