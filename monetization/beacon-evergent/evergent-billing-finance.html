introduction

Pour accéder à l'historique des finances et de la facturation d'un utilisateur, vous devrez vous connecter au système Evergent CRM et localiser le compte sur lequel vous souhaitez travailler. Voir Localiser des comptes dans Evergent CRM pour les détails.

Les Facturation et finances L'onglet des informations de compte d'utilisateur est utilisé pour afficher l'historique de facturation et financier de vos téléspectateurs. Vous pouvez émettre des paiements, des ajustements et des remboursements à vos clients à partir de l'onglet Facturation et finances.

Les fonctionnalités qui peuvent être provisionnées sont les suivantes :

  • Solde courant
  • Factures
  • Paiements
  • Ajustements
  • Remboursements

Solde courant

Le solde courant est un solde du grand livre dynamique qui affiche toutes les transactions de crédit et de débit de ce compte d'utilisateur en fonction de la plage sélectionnée avec le dernier enregistrement.

Ajustements

Ce module est une transaction qui fournit les corrections de paiement d'une écriture de paiement en compte client. Pour effectuer les ajustements de paiement sur le compte, procédez comme suit :

  1. Clique le Solde courant lien hypertexte sous l'onglet Facturation et finances.
  2. Clique le Faire un ajustement bouton.
  3. Sélectionnez le Type de réglage dans la liste déroulante.
  4. Entrez le montant que vous souhaitez ajuster dans le Montant champ.
  5. Sélectionnez le motif des ajustements de paiement dans le Sélectionnez la raison la liste déroulante.
  6. Sélectionnez la facture dans le Facture remise liste déroulante pour laquelle vous souhaitez effectuer les ajustements.
  7. Sélectionnez le Date de début de la panne et Date de fin de panne du calendrier.
  8. Tapez les commentaires dans le Commenter champ.
  9. Clique le Terminé bouton. Les ajustements sont ajoutés dans la section Liste des ajustements.

Écrire

Le bouton Radiation dans le solde courant des comptes respectifs ne s'affichera que lorsque le solde dû pour le compte est positif, c'est-à-dire que le client doit payer le montant. En cliquant sur le bouton Radiation, si le client a payé le montant et ajouter un montant d'ajustement négatif pour rendre le solde dû à zéro.

Factures

La section factures affiche toutes les factures générales créées pour un compte. Il montre en détail pour chaque facture.

Paiements

Affichez l'historique des paiements du client existant, qui fournit un enregistrement des détails de la carte de crédit, l'ID du produit commandé, la date de réception du paiement, le montant, le statut, ainsi que la possibilité de rembourser le paiement au client.

Remboursement du montant

Pour rembourser le montant, procédez comme suit :

  1. Clique le Paiements lien hypertexte sous le Facturation et finances languette.
  2. Sur l'écran Historique des paiements, cliquez sur le bouton Remboursement bouton dans le Actions l'en-tête de colonne que vous souhaitez rembourser pour le compte.
  3. Une boîte de dialogue, "Es-tu sûr?" apparaît comme indiqué ci-dessous.
  4. Sélectionnez un motif de remboursement dans le Sélectionnez la raison la liste déroulante.
  5. Tapez les commentaires pour le remboursement du paiement dans le champ commentaires boîte.
  6. Clique le Oui, remboursement bouton pour effectuer un remboursement, ou bien cliquez sur Retourner pour rejeter les modifications.

Ajustements

Ce module est une transaction qui fournit les corrections de paiement d'une écriture de paiement en compte client.

Faire un ajustement de paiement

Pour effectuer des ajustements de paiement, procédez comme suit :

  1. Clique le Ajustements lien hypertexte sous le Facturation et finances languette.
  2. Clique le Faire un ajustement bouton pour ajuster les paiements.
  3. Les Ajustements l'écran s'affiche comme indiqué ci-dessous.
  4. Sélectionnez le Type de réglage dans la liste déroulante disponible.
  5. Entrez le montant que vous souhaitez ajuster dans le Montant champ.
  6. Sélectionner Raison et Donné sur facture dans la liste déroulante disponible.
  7. Sélectionner Date de début de la panne et Date de fin de panne.
  8. Entrez les commentaires pour les ajustements dans le Commenter boîte.
  9. Cliquez sur Terminé pour fournir le réglage ou bien cliquez Retourner.

Remboursements

Les remboursements sont calculés chaque fois qu'un service est supprimé ou déconnecté. A compter de la date de déconnexion/suppression, il remboursera le montant quelle que soit la durée de facturation.

Ajouter des remboursements

Pour ajouter des remboursements au compte utilisateur, procédez comme suit :

  1. Clique le Remboursements lien hypertexte sous le Facturation et finances languette.
  2. Cliquez sur l'icône Remboursements Ajouter () pour effectuer un remboursement à un client.
  3. Voici les types de remboursement dans la liste Type de transaction :
    • Facturé
    • Non facturé
    • Tous les paiements
    • Rétrofacturation

Remboursement sur les transactions facturées

En sélectionnant le Type de transaction facturée du Type d'opération zone de liste déroulante :

  1. Sélectionnez l'une des factures de la Sélectionnez la facture la liste déroulante.
  2. Sélectionnez l'un des produits de la Sélectionnez le produit commandé zone de liste déroulante.
  3. Entrez le prix à facturer dans le Prix facturé champ.
  4. Saisissez le montant de la taxe dans le Taxe facturée champ.
  5. Saisissez le montant du remboursement dans le Montant à rembourser champ.
  6. Sélectionnez le motif du remboursement du montant dans le Code de raison menu déroulant.
  7. Tapez les commentaires dans le commentaires boîte.
  8. Clique le Ajouter pour appliquer le remboursement au client, ou bien cliquez sur le Retourner bouton pour annuler.

Remboursement sur les transactions non facturées

En sélectionnant le Type de transaction non facturée du Type d'opération zone de liste déroulante :

  1. Sélectionnez l'un des Produits/Forfaits du Sélectionnez le produit commandé zone de liste déroulante.
  2. Entrez le prix à facturer dans le Prix facturé champ.
  3. Saisissez le montant de la taxe dans le Taxe facturée champ.
  4. Saisissez le montant du remboursement dans le Montant à rembourser champ.
  5. Par défaut, la case Ajustement est cochée, ce qui indique que l'ajustement du crédit sera ajouté au montant du remboursement qui sera émis, ce qui effacera l'affichage du solde courant à zéro.
  6. Sélectionnez le motif du remboursement du montant dans le Code de raison menu déroulant.
  7. Tapez les commentaires dans le commentaires boîte.
  8. Clique le Ajouter pour appliquer le remboursement au client, ou bien cliquez sur le Retourner bouton pour annuler.

Remboursement sur toutes les transactions de paiement

En sélectionnant le Tout type de transaction Paiements du Type d'opération zone de liste déroulante :

  1. Saisissez le montant de la taxe dans le Taxe facturée champ.
  2. Saisissez le montant du remboursement dans le Montant à rembourser champ.
  3. Entrez l'identifiant dans le Identifiant de rapprochement champ.
  4. Sélectionnez le motif du remboursement du montant dans le Code de raison menu déroulant.
  5. Tapez les commentaires dans le commentaires boîte.
  6. Clique le Ajouter pour appliquer le remboursement au client, ou bien cliquez sur le Retourner bouton pour annuler.

Remboursement sur les transactions de rétrofacturation

En sélectionnant le Type de transaction de rétrofacturation du Type d'opération zone de liste déroulante :

  1. Saisissez le montant de la taxe dans le Taxe facturée champ.
  2. Saisissez le montant du remboursement dans le Montant à rembourser champ.
  3. Entrez l'identifiant dans le Identifiant de rapprochement champ.
  4. Entrez l'identifiant dans le identifiant de transaction champ.
  5. Sélectionnez le motif du remboursement du montant dans le Code de raison menu déroulant.
  6. Tapez les commentaires dans le commentaires boîte.
  7. Clique le Ajouter pour appliquer le remboursement au client, ou bien cliquez sur le Retourner bouton pour annuler.

Crédits

Le crédit est ce que les entreprises fournissent généralement à un client lorsque le montant d'une facture ou d'un achat est contesté ou incorrect. Cela peut être dû à une erreur d'administration ou au fait que le client demande un remboursement total ou partiel.

Ajout de crédits d'élément de campagne

Pour ajouter le crédit d'élément de campagne au compte utilisateur, procédez comme suit :

  1. Sur le Crédits l'écran, cliquez sur le bouton Ajouter () pour ajouter un nouveau crédit d'article.
  2. Les Ajouter des crédits l'écran apparaît.
  3. Clique le Choisissez le type de crédit liste déroulante et sélectionnez le Crédit d'élément de ligne.
  4. Sélectionnez un type dans le Type d'opération la liste déroulante.
    • Facturé
    • Non facturé
    • Rétrofacturation

Seuls les crédits de rétrofacturation sont applicables à Beacon.

Rétrofacturation

Pour un crédit d'élément de ligne de rétrofacturation, procédez comme suit :

  1. Entrez le montant dans le Montant remboursé champ.
  2. Entrez le montant dans le Montant à créditer champ.
  3. Sélectionnez une raison dans le Code de raison la liste déroulante.
  4. Entrez les commentaires dans le commentaires boîte.
  5. Clique le Ajouter pour ajouter le crédit de service, ou bien cliquez sur le Retourner bouton pour annuler.
