introduction

Le CRM Evergent, intégré à Brightcove Beacon, vous permet de gérer vos comptes de téléspectateurs, vos abonnements et vos transactions. Ce guide couvre l'utilisation du CRM.

Connectez-vous à la gestion de compte

  1. Ouvrez un navigateur Web.
  2. Dans la barre d'adresse, saisissez l'URL de l'application fournie par Evergent.
  3. Saisissez une adresse e-mail/un nom d'utilisateur et un mot de passe valides.
  4. Clique le S'identifier bouton pour commencer votre session Evergent.
  5. Les utilisateurs ayant accès à plusieurs unités commerciales doivent sélectionner l'unité commerciale dans la liste déroulante disponible et cliquer sur Soumettre:
  6. Clique le Gestion de compte bouton image sur l'écran d'accueil du tableau de bord pour accéder à l'application de gestion de compte :
  7. Une boîte de dialogue modale contextuelle vous demandera de choisir parmi les options suivantes :
    • Créer un compte
    • Compte existant
    • Commandes incomplètes
  8. Sélectionner Compte existant

Gestion de compte

Le module de gestion de compte est utilisé pour conserver les détails des clients, les services, les commandes, la facturation et les finances, etc. pour établir des relations avec les clients actuels et potentiels d'Evergent.

Gestion des comptes existants

Une fois que vous êtes dans la section Comptes existants, vous pouvez rechercher le compte sur lequel vous souhaitez travailler.

  1. Sur le Recherche de compte écran, dans le RECHERCHER N'IMPORTE QUOI section, entrez les détails liés au compte pour rechercher le compte existant par Type de compte , Unité d'affaires , ou Numéro de compte:
  2. Pour voir plus d'options de recherche, activez le Champs de recherche avancée interrupteur à bascule:
  3. Les options de recherche avancées vous permettent de rechercher un compte en utilisant n'importe quelle combinaison de plusieurs champs de recherche :

    Cliquez sur l'un des clients' les détails du compte à partir des résultats de la recherche pour accéder à l'écran Récapitulatif du compte client pour afficher les informations client à 360 degrés.
