introduction

Pour gérer les détails du compte utilisateur, vous devrez vous connecter au système Evergent CRM et localiser le compte sur lequel vous souhaitez travailler. Voir Localiser des comptes dans Evergent CRM pour les détails.

Dans cette rubrique, nous couvrirons les détails du compte.

Gestion d'un compte

Les détails du compte peuvent être modifiés à partir de l'écran Récapitulatif du compte :

  1. Clique le Type de compte icône de modification () pour modifier le type de compte.
  2. Clique le Rôle du compte icône de modification () pour sélectionner les rôles de l'utilisateur (Exemple : Utilisateur Admin, Super Utilisateur, Utilisateur Employé et Utilisateur Test).
  3. Clique le Compte vérifié liste déroulante pour mettre à jour l'état de vérification de l'e-mail du client sur Oui/Non.
  4. Clique le Compte gratuit liste déroulante et sélectionnez Oui ou Non pour spécifier si le compte est gratuit ou non.
  5. Clique le Contrôle parental icône de modification () pour activer le contrôle parental permettant au client de restreindre la visualisation des chaînes ou des films sur l'écran vidéo d'un utilisateur. Une fois enregistré, l'utilisateur sera redirigé vers l'écran Modifier le contact principal où il lui sera demandé de fournir le niveau de contrôle parental et le code PIN de contrôle parental.

Remarques

  1. Pour voir plus d'options dans la section Détails du compte, cliquez sur l'icône Plus () au bas de la section Détails du compte.
  2. Les utilisateurs peuvent ajouter plus d'options à la section Détails du compte à partir de l'administrateur système. Les détails de compte suivants peuvent être ajoutés :
    • Méthode de vente
    • Utilisateur migré
    • Numéro d'identification fiscale
    • Type de client
    • Catégorie de migration
    • Rafraîchissez-vous
    • Mode de réclamation
    • ID client CP
    • ID client SP
    • Numéro d'enregistrement
    • Utilisateur migré
    • Segments

Facturation et finances

Notez que cette section traite des paramètres de compte pour la facturation et les finances. Pour les dossiers de facturation et de finances, voir Evergent : Facturation et finances.

La section Facturation et finances fournit les informations relatives aux détails de facturation du client, telles que le montant de la prochaine facture, la prochaine date de facturation, le dernier montant de la facture, le mode de facturation, le cycle de facturation, la date de livraison de la facture et le type de livraison de la facture. Il fournit également des détails financiers tels que le solde actuel, le solde du portefeuille, les appels du client jusqu'à la date, les points de fidélité, les points/montant d'encaissement, le nombre de compensations, le montant total de la compensation, les revenus jusqu'à la date et le numéro et la date de la facture.

Les éléments peuvent être modifiés en cliquant sur l'icône de modification ().
