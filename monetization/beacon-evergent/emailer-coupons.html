introduction

Le système de backoffice Evergent fournit des services pour vous aider à gérer votre entreprise. Dans cette rubrique, nous aborderons deux de ces services :

  • L'emailer automatisé
  • Le gestionnaire de coupons

E-mailer

L'emailer vous permet de créer des modèles et des calendriers pour divers e-mails générés par le système. Les sections qui suivent détailleront les opérations de l'emailer.

Types de messages

Evergent prend en charge les types de messages suivants :

Planification
Lorsque les tâches par lots sont planifiées dans l'administrateur système, un e-mail est envoyé à un administrateur.
Rappel d'expiration de carte de crédit
Lorsque la case "Rappel d'expiration de la carte de crédit" est cochée au niveau de l'entreprise, l'e-mail de rappel est envoyé aux clients lorsque leur carte de crédit est sur le point d'expirer (l'e-mail est généralement envoyé un mois avant l'expiration de la carte). Dans ce cas, le système utilisera le modèle de message d'e-mail de type « rappel d'expiration de carte de crédit ».
E-mails en masse
Si une entreprise souhaite planifier un e-mail promotionnel et spécifier une heure future d'exécution, le type d'e-mail en masse est utilisé. Ces types d'e-mails en masse peuvent être envoyés à un abonné d'unité commerciale, à un abonné d'une zone de marché spécifique ou à un service spécifique.
Transaction par carte de crédit
Lorsque la transaction de paiement par carte de crédit est effectuée par un abonné, les détails de la transaction seront envoyés par e-mail au client à l'aide de ce modèle d'e-mail.
Message d'événement
Les modèles d'e-mail de type message d'événement seront utilisés pour définir un modèle d'e-mail et les associer à la catégorie d'événement, ce qui permet de déclencher des notifications par e-mail aux abonnés une fois l'événement enregistré au niveau du compte.
Ad hoc
Le type de message ad hoc consiste à envoyer les messages individuels sans lien avec aucun mappage de catégorie d'événement.

E-mailers

L'onglet Emailers est utilisé pour ajouter/modifier les modèles d'emails et planifier les emails pour les Business Units.

Ajout d'un nouvel e-mail

Pour ajouter un nouvel e-mail, procédez comme suit :

  1. Sur le E-mailers l'écran d'une Business Unit, cliquez sur le bouton Ajouter () icône.
  2. Dans le Détails de base section, entrez les détails requis, comme suit :
    1. Sur le Titre champ, entrez le titre de l'e-mail.
    2. Sur le Sujet du courriel champ, saisissez l'objet de l'e-mail.
    3. Sélectionnez le type de message que vous définirez un e-mail à partir du Type de message la liste déroulante.
    4. Dans le Courriel de l'administrateur champ, entrez l'identifiant de l'e-mail à partir duquel vous souhaitez que l'e-mail soit envoyé.
    5. Dans le Nom de l'administrateur champ, entrez le nom de l'administrateur.
    6. Sélectionnez le Catégorie d'événement dans la liste déroulante fournie.
    7. Entrez l'identifiant e-mail des personnes auxquelles vous souhaitez que l'e-mail soit envoyé dans le champ CC et Cci des champs.
      Noter: Vous pouvez spécifier plusieurs ID d'e-mail dans les champs CC et BCC en séparant chaque ID d'e-mail par une virgule.
  3. Sélectionner Texte brut ou HTML pour envoyer l'e-mail dans le format sélectionné.
  4. Dans le Message électronique/lettre d'information , dans la zone de texte Corps de l'e-mail, saisissez le modèle de message que vous souhaitez afficher dans l'e-mail du client.
  5. Cliquez sur Aperçu à côté de la zone de texte pour afficher le message électronique au format Texte brut ou HTML.
  6. Clique le Créer bouton.

Mettre à jour un e-mailer

  1. Naviguez vers le E-mailer écran d'une Business Unit.
  2. Dans le Liste des modèles d'e-mails section, cliquez sur le modèle d'e-mail que vous souhaitez modifier.
  3. Les Mettre à jour l'expéditeur apparaît, apportez les modifications nécessaires dans les champs disponibles, puis cliquez sur Mettre à jour.
Notez que vous ne pouvez pas modifier les champs Type de message et Catégorie d'événement sur l'écran Mettre à jour l'e-mail.

Planifier des e-mails

Vous pouvez programmer les e-mails pour les envoyer plus tard. Cet écran permet de planifier les e-mails en masse sur une base horaire spécifiée ou quotidiennement pour le DMA ou le service.

Planification d'un e-mail

  1. Sur le E-mailers écran, sur le Modèles d'e-mails onglet, cliquez sur l'e-mail de type de message en tant qu'e-mails en masse.
  2. Clique le Planifier les e-mails languette. La liste des e-mails programmés apparaît.
  3. Pour planifier un e-mail, cliquez sur le bouton Ajouter () pour ouvrir la boîte de dialogue Planifier un e-mail.
  4. Saisissez l'identifiant de l'e-mail dans le champ Envoyer un courrier à champ.
  5. Clique le Temps d'exécution champ pour sélectionner la date et l'heure du programme.
  6. Pour programmer des e-mails quotidiennement à la date et à l'heure sélectionnées, sélectionnez le Horaire quotidien case à cocher.

Supprimer un e-mail programmé

  1. Clique le Modèles d'e-mails onglet sous le E-mailers onglet dans la barre de menu de gauche.
  2. Cliquez sur un e-mail en masse (type de message) que vous souhaitez supprimer.
  3. Les Mettre à jour l'expéditeur l'écran apparaît.
  4. Clique le Planifier les e-mails languette.
  5. Cliquez sur le bouton Supprimer () icône dans le Actions colonne pour supprimer l'e-mail programmé de la Liste des e-mails programmés.

Génération de coupons

L'autorisation de génération de coupons est basée sur l'autorisation de groupe BU de l'utilisateur. Les utilisateurs connectés disposant de cette autorisation peuvent générer et modifier les coupons.

Ajouter un nouveau coupon/bon

  1. Clique le Bons de réduction du menu de gauche.
  2. Sur le Bons de réduction l'écran d'une Business Unit, cliquez sur le Ajouter () icône. Les Bons de réduction l'écran par défaut apparaît.
  3. Cliquez sur Ajouter () pour générer le coupon.
  4. L'écran de génération de nouveau coupon/bon apparaît comme illustré ci-dessous.
  5. Saisissez les informations dans les champs comme suit :
    1. Préfixe du coupon: Préfixe du Coupon (ex : TestC, Hal, CD, vendredi, etc.)
    2. Nom du lot: Nom du Lot pour les Coupons (Ex : Coupon test, Noël, etc.)
    3. Détaillant: Des coupons peuvent être générés sur la sélection de détaillants comme (AT&T, Vodafone, Verizon, Idea, etc.)
    4. Date de début: La date à partir de laquelle le coupon est disponible pour l'utilisateur
    5. Date d'expiration: Les utilisateurs peuvent définir le jour d'expiration du coupon
    6. Demandeur de coupon: Indique le nom du demandeur qui a demandé les coupons
    7. Crédité en tant que recharge de portefeuille:
      • Pour générer le coupon pour le rechargement du portefeuille, cliquez sur l'interrupteur à bascule pour activer et saisir le montant
      • Pour générer le coupon en tant que bon, cliquez sur l'interrupteur à bascule pour le désactiver
    8. Type de remise: Définir le type de coupon - par exemple, gratuit, forfaitaire, forfaitaire, pourcentage, pourcentage de réduction et extension de validité
    9. Montant/Pourcentage: Pour le type sélectionné, le champ Montant/Pourcentage apparaît
    10. Période & Durée: La Période du Coupon appliquée peut être un nombre de jours ou un nombre de mois, selon la sélection respective de la période que les Durées du Coupon impliquent
    11. Faire un préfixe comme code de coupon: Lorsque vous cochez cette case, le préfixe sera enregistré en tant que code de coupon
    12. Nombre de coupons: L'utilisateur peut fournir le nombre de coupons générés pour le lot. Dans le cas du « Préfixe en tant que code de coupon », le nombre de coupons sera « 1 »
    13. Priorité du coupon: Utilisé pour définir l'affichage du coupon générique sur la base d'un ordre de priorité tel qu'il est généré dans l'entreprise et mappé à un type d'abonnement et d'offre
    14. Longueur du coupon: Longueur du code promo que vous souhaitez générer
    15. Rachats multiples: Si ce commutateur à bascule est sélectionné, l'utilisateur peut définir les remboursements maximum pour un coupon
    16. Maximum de rachats: Permet le nombre de rachats pour un coupon
    17. Rachats multiples par utilisateur unique : la case à cocher permet aux clients d'échanger un seul coupon sur leur compte pour le nombre maximum d'échanges par utilisateur unique
    18. Rachats multiples par utilisateur unique : la case à cocher permet aux clients d'échanger un seul coupon sur leur compte pour le nombre maximum d'échanges par utilisateur unique
    19. Sensible aux majuscules et minuscules: Cochez la case si vous souhaitez générer des codes promo en majuscules
    20. est cadeau: Si la case est cochée, le coupon peut être donné à d'autres
    21. Héritable: Si la case est cochée, le coupon peut être héritable
    22. Tous les éléments sont-ils obligatoires: Si cette case est cochée, alors tous les éléments sont obligatoires
    23. Ignorer le mode de paiement: Si la case est cochée, le mode de paiement sera ignoré après l'application du coupon
    24. Description du coupon: Entrez la description du coupon
    25. Informations sur la campagne: Entrez les détails de la campagne
    26. Message de réussite: Entrez le message de réussite qui doit être affiché à l'utilisateur.
    27. Message d'alerte: Entrez le message d'avertissement qui doit être affiché à l'utilisateur
    28. Produits applicables: L'utilisateur peut sélectionner le non. des produits applicables au Coupon (Ex : Coupon plat 30% OFF pour le produit Néon)
    29. Forfaits applicables: L'utilisateur peut sélectionner le non. des packages d'application aux Coupons
    30. Types d'offres: L'utilisateur peut sélectionner les types d'offres pour les Coupons
    31. Modes de paiement applicables: L'utilisateur peut sélectionner les modes de paiement applicables pour le coupon
  6. Clique le Créer un coupon bouton pour générer des coupons (le temps de génération des coupons dépend également du nombre de coupons générés par l'utilisateur).
  7. Une fois les coupons générés, Coupons générés avec succès message s'affiche :

Recherche de coupons

Recherchez les coupons en utilisant les différentes options. Pour rechercher les coupons, procédez comme suit :

  1. Sur le Bons de réduction l'écran, entrez la valeur du champ pour rechercher par nom de lot, préfixe, code de coupon, statut, date d'expiration du/au.
  2. Cliquez sur le bouton Rechercher. Les résultats de la recherche de coupons sont affichés, comme indiqué ci-dessous.
  3. Cliquez sur le commutateur à bascule Activer la recherche avancée pour activer la recherche avancée.
  4. Vous pouvez rechercher avec un lot de coupons les produits applicables et les modes de paiement applicables répertoriés dans les listes. Pour rechercher avec un lot de coupons, cliquez sur le bouton Recherche par lot de coupons.
  5. Les coupons recherchés sont affichés dans la liste des résultats de la recherche, comme illustré ci-dessous.
