Étapes pour créer un nouveau coupon

Pour créer de nouveaux coupons/bons dans Brightcove Beacon, procédez comme suit :

  1. Dans l'interface utilisateur de Brightcove Beacon, cliquez sur le bouton Commerce languette.
  2. Dans le menu de gauche, sélectionnez Coupons par bande.
  3. Clique le Ajouter un nouveau bon bouton.
  4. Dans le Identifiant champ, saisissez le code que les utilisateurs utiliseront.
  5. Pour le Nom champ, utilisez un nom convivial qui peut être affiché dans les factures.
  6. Pour le Durée champ, entrez la durée du coupon en mois.
  7. Dans le Type de remise vous avez la possibilité de spécifier la remise en devise ou en pourcentage. Par exemple, si la remise est de 3 mois gratuits, définissez la durée sur Répéter avec 3 mois et Type de remise comme 100%.
  8. Dans le Limites section, vous pouvez limiter le nombre d'échanges ou la date d'utilisation du coupon. Par exemple, supposons que le coupon ne peut être utilisé que jusqu'à la fin du mois d'août, ou que seules les 100 premières personnes recevront le coupon.
  9. Dans le Forfaits/Abonnement Associés section, spécifiez les abonnements qui peuvent être utilisés avec le coupon. Si vous ne sélectionnez pas d'abonnement, il s'appliquera à tous les abonnements aux packages.
Notez ce qui suit
  • Une fois le Coupon/Voucher créé, tout utilisateur enregistré peut l'appliquer lors de l'achat d'un abonnement.
