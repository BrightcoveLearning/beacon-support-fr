Création d'un forfait d'abonnement

Dans cette rubrique, vous apprendrez à configurer un forfait d'abonnement vidéo à l'aide de la Commerce onglet de Brightcove Beacon. Le forfait d'abonnement met en œuvre la monétisation SVOD (vidéo à la demande par abonnement). Pour savoir comment appliquer le forfait d'abonnement aux vidéos, consultez le Utiliser un forfait d'abonnement document.

Aperçu

Un forfait d'abonnement définit comment un ensemble particulier de vidéos peut être acheté par vos applications' les clients. Lors de la création du package, vous pouvez, par exemple, configurer les éléments suivants :

  • La fréquence de facturation
  • Définir si la publicité est prise en charge et la configuration publicitaire à utiliser
  • Méthodes de payement
  • Nombre maximum de flux pouvant être lus simultanément par un utilisateur avec le package
  • Dates de début et de fin d'achat du package
  • Appareils et géolocalisations autorisés pour visionner des vidéos dans le package

Étapes pour créer un package

Suivez les étapes suivantes pour créer un nouveau package :

  1. Dans Brightcove Beacon, sélectionnez le Commerce onglet, assurez-vous Paquets est sélectionné dans la navigation de gauche, puis cliquez sur Ajouter un nouveau paquet .
  2. Sur le Emballer (capture d'écran ci-dessous), complétez les étapes suivantes :
    • Nom : Le nom du package utilisé dans les champs personnalisés de Video Cloud pour une vidéo.
    • Fréquence de facturation : Choisissez la fréquence de facturation de ce forfait.
    • Période d'essai : La période d'essai de ce forfait. Entrer 0 sans période d'essai.
    • Statut : Choisissez si vous souhaitez que le package soit publié afin qu'il soit disponible pour utilisation. Notez que vous ne pourrez pas publier le package tant qu'au moins un mode de paiement ne sera pas sélectionné.
    • Publicité : Choisissez si vous souhaitez utiliser des publicités avec ce package.
    • Fournisseurs d'annonces : Si vous avez choisi d'utiliser des publicités à l'étape précédente, choisissez le fournisseur de publicité. Cette liste déroulante est remplie avec les configurations de publicité répertoriées dans le Publicité section de Brightcove Beacon.
    • Modes de paiement à utiliser : Sélectionnez les méthodes de paiement que vous souhaitez utiliser avec ce forfait. Notez que pour chaque méthode sélectionnée, une section s'ouvrira en dessous de la liste des méthodes de paiement, que vous devez remplir pour chaque méthode sélectionnée. La capture d'écran ci-dessous montre Rayure (en jaune) et Apple Store (en rouge) sont sélectionnés.
  3. Clique le Créer un package pour enregistrer la configuration actuelle de votre package.
  4. Sur le Données textuelles onglet, procédez comme suit :
    • Nom du pack : Nom affiché aux utilisateurs.
    • Gros titre : Titre affiché aux utilisateurs, par exemple Spécial vacances ! .
  5. Clique le Créer un package pour enregistrer la configuration actuelle de votre package.
  6. Sur le Données non textuelles onglet, fournissez une image à utiliser à côté du nom du package.
  7. Clique le Créer un package pour enregistrer la configuration actuelle de votre package.
  8. Les Teneur onglet n'est pas à utiliser pour le moment.
  9. Sur le Ruisseaux onglet, procédez comme suit :
    • Flux max. : Nombre maximal d'appareils connectés pouvant diffuser en continu en même temps.
    • Nombre maximal d'appareils : Nombre maximal d'appareils connectés pouvant avoir des connexions actives.
      Les deux paramètres ci-dessus sont également définis globalement, utilisez donc ces paramètres UNIQUEMENT SI vous souhaitez remplacer les paramètres globaux de ce package.
    • Flux avancés : Réglez ceci sur Oui si vous souhaitez personnaliser les droits définis par ce package. Les modifications disponibles sont illustrées dans la capture d'écran suivante :
  10. Clique le Ajouter un nouveau droit pour enregistrer vos droits personnalisés nouvellement configurés.
  11. Clique le Créer un package pour enregistrer la configuration actuelle de votre package.

C'est tout pour la création du package !

Créer un coupon Stripe

Des coupons peuvent être créés pour offrir une remise, soit une remise monétaire fixe, soit une remise en pourcentage, qui peut être appliquée lors de l'achat d'un forfait.

  1. Dans Brightcove Beacon, sélectionnez le Commerce onglet, puis cliquez sur Coupons par bande et enfin cliquez Ajouter un nouveau bon.
  2. Sur le Créer de nouveaux coupons formulaire (capture d'écran ci-dessous), remplissez les éléments suivants :
    • Identifiant : Code que les téléspectateurs saisiront dans l'application Brightcove Beacon pour utiliser le coupon
    • Nom : Un nom convivial pour le coupon qui peut être affiché dans les factures
    • Durée : Choisissez parmi Une fois que , Pour toujours ou Répéter;
      Si Répéter est sélectionné, puis définissez le nombre de mois pendant lesquels il peut être utilisé à plusieurs reprises.
    • Pour Type de remise sélectionnez l'un des éléments suivants :
      • Montant monétaire: Saisissez la valeur et la devise du coupon
      • Pourcentage: Entrez le pourcentage de réduction que le coupon portera

      Exemple: Si la remise est de 3 mois gratuits, définissez le Durée à Répéter avec 3 mois et définir le Type de remise avec Pourcentage de 100%

    • Date limite (UTC) : Sélectionnez la dernière date à laquelle le coupon peut être utilisé
    • Rédemption maximale : Saisissez le nombre maximum d'échanges pour le coupon ; par exemple, seuls 50 clients peuvent utiliser le coupon
    • Sélectionnez les forfaits : Sélectionnez les forfaits auxquels le coupon peut être appliqué ; si vous ne sélectionnez pas de forfait le coupon peut être appliqué à tous les forfaits
      Exemple: Le coupon n'est valable que si vous achetez l'abonnement annuel

  3. Assurez-vous de cliquer sur le sauvegarder bouton pour enregistrer votre nouveau coupon.
