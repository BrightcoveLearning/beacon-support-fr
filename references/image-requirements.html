Images du nuage vidéo

Chaque fois qu'une vidéo est ingérée dans Video Cloud, deux images sont automatiquement créées et affichées dans les propriétés de la vidéo. Les deux images créées sont un Affiche et un La vignette. Les images sont affichées dans le IMAGES rubrique dans Studio. Vous utiliserez ces images de téléchargement pour une utilisation dans Brightcove Beacon, mais devez modifier le profil d'acquisition par défaut pour une fonctionnalité appropriée. Les étapes indiquées dans la boîte d'alerte suivante vous montrent comment effectuer la modification. Si vous n'effectuez pas la modification, l'image de portrait que vous téléchargez continuera à être dimensionnée pour une miniature et semblera très médiocre dans votre application Brightcove Beacon.

Vous DEVEZ modifier votre profil d'ingestion par défaut. Suivez ces étapes:
  1. Dans Studio, allez à ADMIN → Ingérer des profils.
  2. Cliquez sur le titre de votre profil par défaut actuel qui est étiqueté avec un Téléchargement par défaut étiqueter.
  3. Clique le Dupliquer bouton.
  4. Localisez où le images sont définis. Le JSON exact peut être légèrement différent de ce qui est indiqué ci-dessous, selon le profil que vous avez dupliqué.
  5. Modifiez la largeur et la hauteur de l'affiche et des images miniatures comme indiqué :.

    Anciens paramètres

    {
  "media_type": "image",
  "format": "jpg",
  "label": "poster",
  "width": 1280,
  "height": 720,
  "id": "559a8b0ee4b07cbb40be7d22"
},
{
  "media_type": "image",
  "format": "jpg",
  "label": "thumbnail",
  "width": 160,
  "height": 90,
  "id": "559a8b0ee4b07cbb40be7d23"
}

    Nouveaux paramètres

    {
  "media_type": "image",
  "format": "jpg",
  "label": "poster",
  "width": 4000,
  "height": 2250,
  "id": "559a8b0ee4b07cbb40be7d22"
},
{
    "media_type": "image",
    "format": "jpg",
    "label": "thumbnail",
    "width": 3000,
    "height": 2000,
    "id": "559a8b0ee4b07cbb40be7d23"
}
  6. Cliquez sur sauvegarder.
  7. Cliquez ensuite sur le Faire de ceci mon profil par défaut relier.
  8. Assurez-vous que le profil est maintenant étiqueté comme votre valeur par défaut de téléchargement.
Vous êtes maintenant prêt à télécharger des images pour Brightcove Beacon à l'aide de Video Cloud Studio.

Utilisez l'interface Studio pour télécharger deux images. Vous choisirez une image qui représente votre vidéo et prendrez les captures d'écran appropriées qui répondent aux exigences indiquées ci-dessus. Vous téléchargerez ensuite les deux images dans Video Cloud.

Les Affiche et un La vignette sont des espaces réservés pour les images utilisées dans Brightcove Beacon. NE SUIVEZ PAS les recommandations pour les images telles qu'elles sont affichées dans Video Cloud Studio ! Utilisez plutôt ces directives :

Affiche La vignette
Image paysage (16:9) :
Le minimum Largeur: 1000px Hauteur: 563px
Maximum Largeur: 4000px Hauteur: 2250px
Image de l'affiche (2:3)
Le minimum Largeur: 1000px Hauteur: 1500px
Maximum Largeur: 2000px Hauteur: 3000px

L'image de gauche dans la capture d'écran est un exemple d'utilisation de l'image portrait. Si vous y voyez une image très pixelisée, vous saurez que le profil d'acquisition par défaut doit être mis à jour.

Travailler à Brightcove Beacon

Lors de la saisie des informations dans Brightcove Beacon, il vous sera demandé Données non textuelles. Cela signifie des images pour les actifs que vous créez actuellement. Ces images peuvent être décomposées en trois groupes, à savoir :

  • Séries/Saisons
  • Événements en direct
  • Logos des chaînes

Les détails pour les images des groupes suivent.

Séries/Saisons

Pour les séries/saisons, une page apparaîtra semblable à la suivante (sans le rouge OPTIONNEL texte sur le La vignette , pour laquelle vous n'avez pas besoin de fournir d'image) :

Lorsque vous rencontrez cet écran, fournissez un JPEG image pour le Affiche de portrait , Affiche de paysage et Large bannière correspondant aux exigences ci-dessous. Vous N'AVEZ PAS besoin de fournir une image pour La vignette. Si elle n'est pas fournie, l'image miniature sera générée à partir de l'affiche portrait.

Image Conditions
Affiche de portrait Image de l'affiche (2:3)
Le minimum Largeur: 1000px Hauteur: 1500px
Maximum Largeur: 2000px Hauteur: 3000px
Affiche de paysage Image paysage (16:9) :
Le minimum Largeur: 1000px Hauteur: 563px
Maximum Largeur: 4000px Hauteur: 2250px
Bannière large Image paysage (21:9) :
Le minimum Largeur: 1000px Hauteur: 428px
Maximum Largeur: 4000px Hauteur: 1714px

Événements en direct

Pour les événements en direct, une page semblable à la suivante s'affichera (sans le rouge OPTIONNEL texte sur le La vignette et Bannière large , pour lequel vous n'avez pas besoin de fournir d'images):

Lorsque vous rencontrez cet écran, fournissez un JPEG image pour le Affiche de portrait et Affiche de paysage correspondant aux exigences ci-dessous. Vous N'AVEZ PAS besoin de fournir des images pour La vignette et Large bannière. Si elle n'est pas fournie, l'image miniature sera générée à partir de l'affiche portrait et la bannière large à partir de l'affiche paysage.

Maximum Largeur: 2000px Hauteur: 3000px
Affiche de paysage Image paysage (16:9) :
Le minimum Largeur: 1000px Hauteur: 563px
Maximum Largeur: 4000px Hauteur: 2250px
