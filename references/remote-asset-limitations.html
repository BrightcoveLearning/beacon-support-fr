Limitations des actifs distants

Les limitations suivantes doivent être prises en compte lors de l'utilisation de vidéos d'éléments distants :

  • SSAI n'est pas pris en charge pour les actifs distants.
  • Les DRM sont pris en charge uniquement si vous pouvez personnaliser le lecteur pour communiquer avec le serveur de licences DRM. C'est possible avec le Brightcove Player via un plugin. Pour les autres lecteurs, vous devrez consulter la documentation du lecteur.
  • Toutes les vidéos ingérées dans Video Cloud peuvent être lues dans Brightcove Player. Pour les joueurs non Brightcove, que vous rencontrerez principalement dans Brightcove Beacon, vous devrez vérifier que le format vidéo est entièrement pris en charge par chaque plate-forme. En outre, il existe quelques fonctionnalités avancées de DASH et HLS que Brightcove ne prend pas encore en charge

Pour plus d'informations sur les vidéos d'actifs distants, consultez le Nuage vidéo document Travailler avec des vidéos de ressources distantes. Sachez que vous verrez cette vidéo dans un nouvel onglet.
