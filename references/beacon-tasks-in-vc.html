Résumé de la tâche

Voici la liste des tâches que vous, en tant que client Brightcove Beacon, devez/devez effectuer dans Video Cloud Studio :

  • Ajouter des éléments vidéo
  • Créer des champs personnalisés
  • Ajouter des images associées à l'élément vidéo
  • Ajouter des informations sur la gestion des droits
  • Ajouter des données d'épisode pour une série/saison à un élément vidéo
  • Créer des listes de lecture vidéo
  • Créer des événements Brightcove Live
  • Créer une configuration d'annonce SSAI

Chaque tâche sera expliquée plus loin dans ce document avec un lien vers la documentation où des instructions sont fournies sur la façon d'effectuer cette tâche particulière.

Ce document n'est PAS un manuel "comment faire" pour chaque tâche, mais chaque section vous liera au document approprié qui contient les étapes sur la façon d'accomplir la tâche.

Ajouter des éléments vidéo

Video Cloud est l'outil utilisé pour télécharger vos vidéos que vous utilisez dans Brightcove Beacon. Tout au long de ce document, vous verrez que Video Cloud Studio est un outil couramment utilisé lorsque vous travaillez avec votre application Brightcove Beacon.

Les ressources suivantes vous permettront d'intégrer vos vidéos dans Video Cloud :

  • Notions de base sur le cloud vidéo : Téléchargement de vidéos - Une vidéo de 1 minute 30 secondes et quatre étapes pour un démarrage rapide du téléchargement de contenu dans Video Cloud.
  • Téléchargement de vidéos à l'aide du module de téléchargement - Le document complet sur le téléchargement de vidéos avec Video Cloud.
  • Gestion des vidéos - La page de destination qui englobe l'ensemble du flux de travail de gestion vidéo, y compris les documents sur le téléchargement de vidéos, les listes de lecture, l'organisation de vidéos avec des dossiers, etc.

Créer des champs personnalisés

Video Cloud vous permet d'utiliser des champs personnalisés pour ajouter des métadonnées à vos vidéos. Une fois que vous avez des vidéos, les champs personnalisés jouent un rôle important dans la préparation de ces vidéos à utiliser dans Brightcove Beacon. Les champs personnalisés que vous ajoutez aux vidéos Video Cloud qui fonctionnent avec Brightcove Beacon se répartissent en trois catégories. Ces catégories, et une brève explication de chacune, sont :

  • Général - Des informations sur le contenu de la vidéo, notamment :
    • Genre
    • Description courte et longue
    • Année de production
    • Jeter
    • Écrivain
    • etc.
  • Gestion des droits - Prend en charge la monétisation et les restrictions d'accès. Par exemple, le type de publicité et les dates de disponibilité des vidéos.
  • Données sur les épisodes - Si la vidéo est un épisode d'une série, vous définissez la série, la saison et le numéro d'épisode.

Les Création de champs personnalisés Video Cloud document est uniquement dédié à vous aider à créer les champs personnalisés requis.

Des valeurs sont attribuées à ces champs personnalisés dans Video Cloud Studio.

Ajouter des informations sur la gestion des droits

Comme mentionné précédemment, les champs personnalisés sont utilisés pour ajouter des informations de gestion des droits qui contrôlent la monétisation et les restrictions d'accès. Par exemple, les champs

  • beacon.rights.indice.date de début
  • beacon.rights.indice.date de fin

définir quand un actif vidéo particulier est accessible.

De plus, les documents suivants montrent comment mettre en œuvre la monétisation via la publicité et les abonnements :

  • Mise en œuvre de la VOD SSAI ; voir le Définition des valeurs de champ personnalisées section du document pour fournir des valeurs aux champs personnalisés appropriés.
  • Utiliser un forfait d'abonnement

Les champs personnalisés de gestion des droits reçoivent des valeurs dans Video Cloud Studio.

Ajouter des données d'épisode

Si une vidéo particulière est un épisode d'une série, des champs personnalisés sont utilisés pour attribuer des données d'épisode qui associent la vidéo à la série, la saison et le numéro d'épisode corrects. Cet exemple montre les champs personnalisés utilisés et des valeurs spécifiques pour chaque champ.

Les Ingérer des épisodes partie de la Création de séries/saisons/épisodes document fournit des instructions spécifiques.

Créer des listes de lecture vidéo

Les listes de lecture sont fortement utilisées dans Brightcove Beacon pour décider du contenu vidéo à afficher sur une page d'application. Vous créez des listes de lecture dans Video Cloud Studio.

Les ressources suivantes vous permettent d'intégrer vos vidéos dans Video Cloud :

  • Notions de base sur le cloud vidéo : Création d'une liste de lecture manuelle
  • Notions de base sur le cloud vidéo : Créer une liste de lecture intelligente
  • Créer et gérer des listes de lecture
Les listes de lecture créées dans Video Cloud Studio peuvent contenir UNIQUEMENT des vidéos. Si vous souhaitez créer des listes de lecture contenant des séries, des événements en direct, etc., celles-ci doivent être créées dans Brightcove Beacon.

Créer des événements Brightcove Live

Si vous souhaitez lire des événements de diffusion en direct dans vos applications Brightcove Beacon, une partie du processus doit être effectuée dans Video Cloud Studio. Les Diffuser un événement en direct document vous guide tout au long du processus.

Créer une configuration d'annonce SSAI

L'insertion d'annonces côté serveur (SSAI) peut être utilisée dans Brightcove Beacon. Pour implémenter cela, vous devez créer une configuration publicitaire dans Video Cloud Studio. Les Mise en œuvre de la VOD SSAI document explique comment créer cette configuration publicitaire, ainsi que le processus de bout en bout de configuration VOD SSAI pour Brightcove Beacon.
