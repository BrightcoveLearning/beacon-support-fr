Aperçu

Un compte Stripe est nécessaire pour autoriser les paiements des clients pour les plateformes suivantes :

  • la toile
  • Téléviseur intelligent Samsung
  • Téléviseur intelligent Sony
  • Téléviseur intelligent Toshiba

Cela signifie que Stripe est nécessaire pour l'application Web et toutes les applications Smart TV autre que Android TV et Fire TV.

Les étapes suivantes donnent un aperçu du processus d'utilisation de Stripe avec Brightcove Beacon :

  1. Créez votre compte en fournissant des informations de base (email, nom, mot de passe).
  2. Activez votre compte en fournissant des informations sur votre entreprise.
  3. Récupérez vos clés API Stripe.
  4. Fournissez au personnel de Brightcove les clés API afin qu'elles puissent être ajoutées à Brightcove Beacon Experience.

Création et activation du compte Stripe

  1. Naviguez jusqu'au Page d'accueil de Stripe.
  2. Clique le COMMENCEZ MAINTENANT bouton.
  3. Remplissez vos informations de base pour créer le compte.
  4. Assurez-vous que le tableau de bord Stripe vous a été présenté.
  5. Clique le Activez votre compte Stripe option.
  6. Clique le Commencez maintenant bouton pour commencer le processus.
  7. Vous devrez vérifier votre adresse e-mail.
  8. Après avoir vérifié votre adresse e-mail, vous verrez le formulaire à remplir pour activer votre compte. Les sections principales du formulaire sont surlignées en jaune.
  9. Après avoir rempli le formulaire, cliquez sur le Présenter une demande bouton.

Obtenir vos clés API Stripe

La connexion entre votre compte Stripe et Brightcove Beacon se fait via les clés API Stripe. Dans la capture d'écran du tableau de bord ci-dessus, vous voyez un lien vers Obtenez vos clés API en direct. Notez que jusqu'à ce qu'un compte soit activé, vous ne pouvez obtenir que des clés API de test.

Dans le tableau de bord, cliquez sur le Obtenez vos clés API en direct et suivez les instructions pour les télécharger.

Connexion de Stripe à Brightcove Beacon

Une fois que vous avez vos clés API Stripe, vous les donnez au personnel de Brightcove qui configure votre Brightcove Beacon Experience. Ils associeront vos clés API Stripe à votre Brightcove Beacon Experience afin que, lorsque vous configurez Stripe pour qu'il soit utilisé dans vos applications, il soit associé à votre compte.
