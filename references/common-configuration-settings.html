Paramètres de configuration communs

Dans cette rubrique, vous découvrirez les paramètres de configuration courants utilisés lors de la création ou de la modification de votre application dans Brightcove Beacon.

Aperçu

Cette rubrique est une compilation des paramètres de configuration courants associés à votre application Brightcove Beacon. Vous trouverez ces paramètres dans d'autres documents relatifs à l'exécution de tâches spécifiques dans Brightcove Beacon, y compris les sections d'onglet suivantes :

  • Disposition
  • Pages
  • Canaux
  • Films
  • Listes de lecture
  • Programmes / Séries
  • Événements
  • Commerce / Package / Streams / Avancé

Droits et planification

Les Droits et planification Cette section vous permet de définir des autorisations de lecture et de définir des calendriers de publication pour votre application.

Voici un exemple d'ajout ou de modification de pages :

Ce tableau contient les champs que vous trouverez dans le Droits et planification languette:

Champ Obligatoire La description
Statut Oui Sélectionnez l'un des types suivants :
  • Publié
  • Inédit
Type de droits Oui Indiquez si votre chaîne accepte la publicité à partir des valeurs suivantes :
  • Libérer
  • Publicité - AVOD (Vidéo publicitaire à la demande)
  • Publicité - SVOD (Abonnement Vidéo à la Demande)
  • Hériter - Cela signifie que les droits seront hérités de la série à laquelle appartient cette saison.
Date de début des droits Oui Définissez la date et l'heure de début de votre application. (Aussi, Heure de début)
Date de fin des droits Oui Définissez la date et l'heure de fin de votre application. (Aussi, Heure de fin)
Appareils à publier Oui Ajoutez les appareils sur lesquels vous souhaitez publier. Sélectionnez le Sélectionner / Désélectionner tous les appareils pour voir la liste des appareils disponibles.
Emplacements autorisés Oui

Sélectionnez les emplacements géographiques où vous souhaitez autoriser les utilisateurs à afficher votre contenu. Cliquez à l'intérieur du Emplacements autorisés boîte pour voir une liste d'emplacements. Si aucune restriction, entrez Monde.

Si vous ne spécifiez pas au moins 1 emplacement autorisé, ce message peut s'afficher dans votre application :
Emplacements refusés Non Sélectionnez les emplacements géographiques où vous NE VOULEZ PAS autoriser les utilisateurs à afficher votre contenu. Cliquez à l'intérieur du Emplacements refusés boîte pour voir une liste d'emplacements.
Nombre maximal d'appareils Non Définissez le nombre maximum d'appareils connectés actifs pour un compte. Valeur entière.
Flux max. Non Définissez le nombre maximum de flux pouvant être regardés à partir du même compte. Valeur entière.

Exemple de réglage Flux max. à 4 et Nombre maximal d'appareils à 6: Cela signifie que 4 personnes peuvent diffuser (diffuser) de la vidéo, mais vous pouvez avoir 6 personnes connectées (appareils).

Données textuelles

Les Données textuelles La section vous permet d'ajouter du texte affichable. C'est le nom qui apparaît dans votre application.

Voici un exemple d'ajout ou de modification de pages :

Ce tableau contient les champs que vous trouverez dans le Données textuelles languette:

Champ Obligatoire Remarques
Titre Oui Le titre affiché pour votre application. Vous trouverez un champ de saisie pour chaque langue que vous avez définie.

On vous demandera généralement sur le Donnee de base onglet pour un Nom. Ce nom est pour les références internes, tandis que le Données textuelles onglet Titre est ce que les utilisateurs verront dans l'application.

Données non textuelles

Les Données non textuelles La section vous permet de télécharger des images d'arrière-plan et d'icônes pour votre application.

Voici un exemple d'ajout ou de modification de pages :

Ce tableau contient les champs que vous trouverez dans le Données non textuelles languette. Notez que vous pouvez télécharger des images pour chaque langue que vous avez définie.

Champ Obligatoire Remarques
Icône Oui L'image que vous voulez représenter votre page.
Icône de survol Oui Cette image s'affiche lorsque l'utilisateur survole l'image de l'icône.
Fond horizontal Oui L'image d'arrière-plan affichée lorsque l'appareil de l'utilisateur est en mode paysage.
Fond vertical Oui L'image d'arrière-plan affichée lorsque l'appareil de l'utilisateur est en mode portrait.

La taille de téléchargement maximale autorisée est de 500 Ko.
Pour plus de détails sur le téléchargement d'images dans Brightcove Beacon, consultez le Images et leurs spécifications document.
